## 🔬 第二步：深度复盘辅助

### Prompt A - 引导式年度复盘

```markdown
请引导我完成年度复盘。

逐步问我以下问题（每次一个）：
1. 一年前你对投资的认知是什么？
2. 现在你对投资的认知是什么？
3. 这一年最骄傲的投资决策是哪个？
4. 这一年最后悔的投资决策是哪个？
5. 你建立的投资系统运行得如何？
6. 你发现了自己什么新的弱点？
7. 你克服了什么之前的弱点？
8. 明年你最想改进什么？

请逐个问我，帮我深入思考。
```

### Prompt B - 成长客观评估

```markdown
请帮我客观评估我的成长：

我的年度数据：
[粘贴数据]

请评估：
1. 我的进步是显著的还是一般的？
2. 有没有自我欺骗的迹象？
3. 哪些方面成长最大？
4. 哪些方面还需要努力？
5. 给我一个综合成长评分（1-10）
```
