# EP37 - 回测思维与局限

> 📅 第37期 | 第三季度：建立系统与策略

---

## 📌 本期核心问题

**过去有效≠未来有效。回测的价值和陷阱是什么？**

---

## 💡 第一性原理拆解

### 回测的价值

| 价值 | 说明 |
|------|------|
| 验证想法 | 策略在历史上是否有效 |
| 发现问题 | 找出策略的薄弱环节 |
| 建立信心 | 知道策略的历史表现 |

### 回测的陷阱

| 陷阱 | 说明 |
|------|------|
| 过拟合 | 针对历史数据过度优化 |
| 未来信息 | 使用了当时不可能知道的数据 |
| 幸存者偏差 | 只看存活的公司 |
| 忽视成本 | 没有计入交易费用 |
| 样本不足 | 只测试了特定时期 |

### 正确的回测态度

> 回测是**必要条件**，不是**充分条件**。
> 过去有效是起点，不是终点。
