## 📚 模块一：概念深化

### Prompt 1.1 - 好规则的标准

```markdown
帮我理解什么是"好的入场规则"：

**三个标准**

1. **明确性**
   - 任何人看了都能判断是/否
   - 不含"可能"、"也许"、"感觉"
   - 有具体数字或条件

2. **可验证性**
   - 可以用数据验证条件是否满足
   - 不依赖主观判断
   - 可回测

3. **可执行性**
   - 条件发生时能及时发现
   - 有时间执行（不是超短线信号）
   - 不需要太多资源

**坏规则示例**
- "感觉便宜的时候买入"
- "趋势好的时候加仓"
- "基本面改善时买入"

**好规则示例**
- "PE低于历史20%分位时买入"
- "价格站上20日均线时买入"
- "连续3个季度净利润增长>10%时买入"

对比分析两类规则的差异。
```

### Prompt 1.2 - 入场规则的分类

```markdown
介绍不同类型的入场规则：

**基本面规则**
- 估值条件（PE/PB/PEG等）
- 质量条件（ROE/利润率等）
- 成长条件（增长率等）
- 示例：PE<15 且 ROE>15%

**技术面规则**
- 趋势条件（均线突破等）
- 动量条件（RSI/MACD等）
- 量价条件（放量突破等）
- 示例：价格突破20日均线且放量

**综合规则**
- 基本面+技术面
- 一级筛选+二级筛选
- 示例：先用基本面筛选股票池，再用技术面选择时机

给出每类规则的2-3个具体示例。
```

### Prompt 2.2 - 入场信号的有效性

```markdown
讨论入场信号的有效性：

**统计研究发现**
- 单一指标的预测能力有限
- 多指标组合是否更有效？
- 过拟合的风险

**常见入场信号的胜率**
- PE低分位买入的胜率
- 均线突破买入的胜率
- 这些胜率够吗？

**关键洞察**
- 入场只是系统的一部分
- 出场同样重要
- 期望值 = 胜率 × 盈亏比
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 设计个人入场规则

```markdown
帮我设计个人的入场规则：

请回答以下问题：
1. 你倾向基本面还是技术面？
2. 你的持股周期大概多长？
3. 你能获取什么数据？

根据你的回答，我会帮你：
1. 设计3条入场规则
2. 检验规则是否"明确、可验证、可执行"
3. 给出改进建议
```

### Prompt 3.2 - 规则验证练习

```markdown
验证你的入场规则的质量：

请写出你的3条入场规则：
1. ___
2. ___
3. ___

对每条规则，检验：
□ 是否明确？（能给"是/否"答案吗？）
□ 是否可验证？（用什么数据验证？）
□ 是否可执行？（能及时发现信号吗？）

如果有不通过项，我会帮你修改。
```
