# EP37 - 回测思维与局限：视频逐字稿

> 📺 时长：约12分钟

## 开场（0:00-1:00）

大家好，欢迎来到《通往自由之路》第37期。

今天我们学习一个重要但经常被误解的概念：**回测**。

你可能听过这样的话：
- "这个策略历史回测年化收益30%！"
- "过去10年都有效！"

这些听起来很诱人，但有一个核心问题：

> **过去有效 ≠ 未来有效**

今天，我们来理解回测的真正价值和它的陷阱。

---

## 第一部分：什么是回测（1:00-4:00）

### 回测的定义

```
回测 = 用历史数据模拟执行策略，看过去效果如何

例子：
策略："PE<15时买入，PE>25时卖出"
回测：用过去10年数据，模拟这样做会赚多少
```

### 回测的价值

| 价值 | 说明 |
|------|------|
| 验证想法 | 初步检验策略逻辑是否有效 |
| 发现问题 | 找出策略在特定环境下的弱点 |
| 建立信心 | 知道策略在历史上的表现范围 |
| 设定预期 | 了解可能的收益和回撤 |

### 回测的正确心态

```
回测能告诉你：
✓ 这个想法在历史上是否曾经有效
✓ 效果大概在什么范围
✓ 在什么环境下可能失效

回测不能告诉你：
✗ 未来一定有效
✗ 实际执行一定能获得同样结果
✗ 没有其他风险
```

---

## 第二部分：回测的陷阱（4:00-8:00）

### 陷阱1：过拟合（Overfitting）

```
什么是过拟合：
针对历史数据过度优化参数

例子：
测试发现"17日均线"效果最好
但这可能只是噪音
换个数据集就失效了

症状：
回测收益极高（年化50%+）
参数非常精确（如17.3日）
规则非常复杂
```

### 陷阱2：未来信息（Look-ahead Bias）

```
什么是未来信息：
使用了当时不可能知道的数据

例子：
"财报发布后买入"
但回测用的是修正后的财报数据
实际那天可能还不知道真实数据

症状：
用了调整后的数据
用了后来才知道的分类（如"行业龙头"）
```

### 陷阱3：幸存者偏差

```
什么是幸存者偏差：
只看存活下来的公司

例子：
回测说"买入低PE股票"效果好
但当时的低PE可能包含后来退市的公司
在实践中你可能买到就退市了

症状：
用当前的股票池回测历史
忽略了已退市、被收购的公司
```

### 陷阱4：忽视成本

```
忽视的成本：
- 交易佣金
- 买卖价差（滑点）
- 冲击成本（大额交易影响价格）
- 税费

影响：
频繁交易的策略，成本可能吃掉大部分收益
```

### 陷阱5：样本不足

```
问题：
只测试了特定历史时期

例子：
只在2010-2020这段牛市测试
遇到不同市场环境可能失效

解决：
尽可能长的回测期
至少包含一个完整周期（牛熊）
```

## 第三部分：如何正确使用回测（8:00-11:00）

### 基本原则

```
1. 回测是必要条件，不是充分条件
   通过回测 → 可以尝试
   未通过回测 → 基本不要用

2. 简单的策略比复杂的更可靠
   参数越少越好
   逻辑越清晰越好

3. 样本外测试
   一部分数据用于开发
   另一部分数据用于验证

4. 保守估计
   假设实际效果是回测的50-70%
```

### 回测检查清单

```
在相信回测结果前，检查：

□ 数据是否干净（无未来信息）？
□ 是否考虑了退市公司？
□ 是否扣除了交易成本？
□ 测试期间是否足够长（>10年）？
□ 是否覆盖不同市场环境？
□ 参数是否简单（不是过拟合）？
□ 逻辑是否说得通？
□ 其他研究者是否验证过？
```

### 个人投资者的简化回测

```
不需要复杂工具，Excel就够：

简单回测步骤：
1. 选一个简单策略（如定投指数）
2. 收集历史数据（如过去10年月度数据）
3. 模拟执行策略
4. 计算收益、回撤
5. 分析在不同年份的表现

目的：
不是追求精确
而是建立对策略的直觉理解
```

---

## 第四部分：正确的期望管理（11:00-11:30）

### 回测与实盘的差距

```
常见情况：
回测收益：15%
实盘收益：8%

原因：
- 执行偏差
- 滑点和成本
- 情绪影响
- 市场变化

建议：
预期实盘效果 = 回测效果 × 0.5-0.7
```

今天我们学习了回测思维与局限：

**回测的价值**：
初步验证想法，了解历史表现

**五大陷阱**：
过拟合、未来信息、幸存者偏差、忽视成本、样本不足

**正确态度**：
回测是必要条件，不是充分条件
