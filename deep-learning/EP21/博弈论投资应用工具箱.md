# EP21 - 博弈论投资应用工具箱

> 用博弈论思维分析投资决策

---

## 🎯 使用说明

本工具帮助你：
1. 识别投资中的博弈类型
2. 分析交易对手
3. 做出更理性的决策

---

## 📋 工具一：博弈类型判断器

### 判断你参与的是什么博弈

```
博弈类型快速判断
━━━━━━━━━━━━━━━━━━━━━━━━━━━

我正在考虑的投资：_______________

【零和博弈检查】
□ 我的盈利直接来自对手的亏损
□ 没有新价值被创造
□ 是衍生品（期货、期权、CFD）
□ 是短期交易（持有少于1周）

勾选3项以上 = 可能是零和博弈

【负和博弈检查】
□ 交易成本会侵蚀收益
□ 费用占预期收益的比例大于5%
□ 需要频繁交易才能执行策略
□ 平台/中介从中抽取费用

勾选3项以上 = 可能是负和博弈

【正和博弈检查】
□ 所投资的资产本身创造价值
□ 预计持有时间超过1年
□ 参与经济增长的收益分享
□ 有分红或其他持续回报

勾选3项以上 = 可能是正和博弈

━━━━━━━━━━━━━━━━━━━━━━━━━━━

我的判断：____博弈

如果是零和/负和：
我有什么独特优势能赢？
_________________________________
```

## 📋 工具三：囚徒困境识别器

### 识别可能的踩踏风险

```
囚徒困境风险评估
━━━━━━━━━━━━━━━━━━━━━━━━━━━

评估日期：____年____月____日
评估标的：_______________

【囚徒困境特征检查】

□ 持有者之间策略高度相关
  （如：都是追同一个热点进来的）

□ 一旦下跌，大家倾向同时卖出

□ 没有基本面锚定
  （没人知道"真实价值"是多少）

□ 持有者多为短期投机者

□ 流动性一般，大量卖出会砸盘

□ 近期有催化剂失效的风险

勾选数量：____/6

━━━━━━━━━━━━━━━━━━━━━━━━━━━

风险评估：
0-2项：低风险
3-4项：中风险
5-6项：高风险——警惕踩踏

我的应对策略：
□ 提前设置止损
□ 降低仓位
□ 做好最坏准备
□ 其他：_______________
```
