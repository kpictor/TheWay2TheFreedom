# EP09 - 投资目标的设定 | AI深度学习包

---

## 📚 模块一：概念深化

### Prompt 1.1 - SMART原则在投资中的应用

```markdown
请深入解释SMART原则如何应用于投资目标：

**S - Specific（具体）**
- "赚钱"不是目标，"3年后有50万"才是
- 请帮我把模糊目标转化为具体目标

**M - Measurable（可衡量）**
- 需要具体数字：金额、收益率、时间
- 如何设定可追踪的里程碑？

**A - Achievable（可实现）**
- 年化20%是合理目标吗？年化8%呢？
- 如何判断目标是否切合实际？

**R - Relevant（相关性）**
- 投资目标要与人生目标挂钩
- 为什么要这笔钱？对你意味着什么？

**T - Time-bound（有时限）**
- 1年、3年、10年目标有何不同？
- 如何设定合理的时间预期？
```

### Prompt 1.2 - 目标的层级结构

```markdown
帮我理解投资目标的层级结构：

```
人生目标
└── 财务目标
    ├── 短期（1-3年）：应急、购物、旅行
    ├── 中期（3-10年）：买房、创业、教育
    └── 长期（10年+）：财务自由、退休
```

对话引导：
1. 你的人生目标是什么？（不只是赚钱）
2. 实现这些目标需要多少钱？
3. 这些钱需要什么时候准备好？
4. 每个阶段的投资策略应该如何不同？
```

### Prompt 2.2 - 目标驱动投资法

```markdown
介绍"目标驱动投资"（Goal-Based Investing）方法论：

1. **核心理念**
   - 不是追求最高收益，而是实现特定目标
   - 不同目标用不同策略

2. **三桶理论**
   - 安全桶：保障确定性需求
   - 成长桶：追求合理增长
   - 梦想桶：追求高收益（可接受损失）

3. **如何将资金匹配到不同桶？**
   - 根据目标时间
   - 根据目标重要性
   - 根据风险承受

请帮我对自己的资金做分桶。
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 逆向目标推导

```markdown
进行逆向目标推导练习：

如果你想在10年后有100万用于[填入目的]：

问题序列：
1. 你现在有多少可投资？
2. 每月能追加投资多少？
3. 需要多少年化收益才能达成？
4. 这个收益率合理吗？需要承担什么风险？
5. 如果收益率不够，你选择：
   - 延长时间？
   - 增加投入？
   - 降低目标？
   - 承担更大风险？

帮我完成这个计算和选择。
```

### Prompt 3.2 - 目标冲突分析

```markdown
帮我分析可能存在的目标冲突：

常见冲突：
1. 短期要用钱 vs 长期想增值
2. 想要高收益 vs 害怕亏损
3. 多个目标竞争有限资金
4. 夫妻/家庭成员目标不一致

对话引导：
1. 你有没有这样的目标冲突？
2. 冲突发生时如何排优先级？
3. 有没有可能找到兼顾的方案？
```
