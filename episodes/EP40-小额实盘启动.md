## 📌 本期核心问题

**纸上得来终觉浅。如何用可承受的本金开始真正的实践？**

---

## 💡 第一性原理拆解

### 为什么需要实盘

| 模拟 | 实盘 |
|------|------|
| 没有真实情绪 | 真金白银的压力 |
| 执行完美 | 会犹豫、会冲动 |
| 验证策略逻辑 | 验证策略+验证自己 |

### 实盘启动原则

```
1. 使用你能100%承受亏损的资金
2. 金额足够让你在乎，但亏光不影响生活
3. 建议：总可投资资金的10-20%
4. 目标不是赚钱，是验证系统和认识自己
```

### 第一周任务

- [ ] 确定实盘资金数额
- [ ] 按系统规则执行第一笔交易
- [ ] 开始记录交易日志
- [ ] 每天记录自己的情绪状态
