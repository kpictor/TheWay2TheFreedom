# EP12 - 情绪日记初体验 | AI深度学习包

> 🎯 目标：开始记录投资情绪，建立自我观察的习惯

---

## 📚 模块一：概念深化

### Prompt 1.1 - 为什么记录情绪

```markdown
帮我理解记录投资情绪的重要性：

**记忆是不可靠的**
1. 我们会美化成功的决策
2. 我们会忘记失败时的情绪
3. 后见之明让我们无法真正学习

**情绪日记的作用**
1. 捕捉决策时的真实状态
2. 发现重复的情绪模式
3. 建立情绪-决策-结果的关联
4. 提供自我改进的客观依据

请用具体例子说明"不记录"会导致什么问题。
```

### Prompt 1.2 - 情绪量化方法

```markdown
帮我学会量化情绪：

**情绪维度**
1. 恐惧-贪婪轴（1-10）
2. 焦虑-平静轴（1-10）
3. 冲动-理性轴（1-10）
4. 自信-怀疑轴（1-10）

**身体信号**
- 心跳加速？
- 手心出汗？
- 反复查看账户？
- 失眠？

请设计一个快速情绪自检清单，我可以在每次交易前使用。
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 情绪研究证据

```markdown
介绍关于投资情绪的学术研究：

1. **情绪与市场择时**
   - 研究发现高情绪时买入的回报如何？
   - 低情绪时买入呢？

2. **情绪传染**
   - 市场恐慌如何通过社交网络传播？
   - 媒体情绪如何影响个人决策？

3. **情绪与认知能力**
   - 压力下的决策质量下降多少？
   - 科学依据是什么？

这些研究给我什么启示？
```

### Prompt 2.2 - 专业交易员的情绪管理

```markdown
介绍专业交易员如何管理情绪：

1. **交易前仪式**
   - 情绪检查
   - 身体状态确认
   - 规则复习

2. **交易中纪律**
   - 强制休息机制
   - 最大亏损熔断
   - 情绪升级预警

3. **交易后复盘**
   - 情绪记录
   - 决策质量评估
   - 规则遵守检查

我可以借鉴什么？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 情绪模式识别

```markdown
帮我识别自己的情绪模式：

请和我进行以下对话：
1. 回忆一次冲动买入，当时什么情绪？
2. 回忆一次恐慌卖出，当时什么情绪？
3. 回忆一次后悔没买，当时什么情绪？
4. 回忆一次后悔没卖，当时什么情绪？

分析：
- 你的情绪模式是什么？
- 什么情况触发这些情绪？
- 情绪如何影响你的决策质量？
```

### Prompt 3.2 - 情绪交易复盘

```markdown
用情绪视角复盘一次交易：

请描述你最近的一次交易，我会帮你分析：

信息收集：
1. 交易时间和价格
2. 买入/卖出原因
3. 交易前你的情绪状态（1-10）
4. 交易后你的情绪状态（1-10）
5. 现在回看这笔交易的质量

分析：
- 情绪是否影响了这个决策？
- 如果当时情绪平静，你会做同样的决定吗？
```

```markdown

要求：
```
