## 开场（0:00-1:30）

大家好，欢迎来到《通往自由之路》第31期。

今天我们学习投资系统的另一半：**出场规则**。

有一句话说得好：
> "会买的是徒弟，会卖的是师傅。"

很多人花大量时间研究何时买入，却很少思考何时卖出。结果是：
- 赚了的不知道什么时候走，涨上去又跌回来
- 亏了的不愿意割肉，小亏变大亏
- 持有了很久不涨不跌，浪费时间成本

**好的出场规则，能帮你锁定收益、控制损失、释放资金。**

今天，我们就来设计完整的出场规则。

---

## 第一部分：三类出场规则（1:30-5:00）

### 止损规则：保护本金

**核心目标**：当判断错误时，限制亏损幅度

```
止损的威力：

无止损：
买入 → 下跌20% → 下跌40% → 下跌60% → 几乎回不来

有止损（10%）：
买入 → 下跌10% → 止损出场 → 保留90%本金 → 重新出发
```

**常见止损方法**：

| 方法 | 设置方式 | 优点 | 缺点 |
|------|---------|------|------|
| 固定比例 | 下跌X%止损 | 简单清晰 | 可能被震出 |
| 移动止损 | 从高点回撤X% | 保护利润 | 需要追踪 |
| 技术止损 | 跌破支撑位 | 符合市场结构 | 需要技术分析 |
| ATR止损 | N倍ATR | 适应波动 | 计算复杂 |

### 止盈规则：锁定收益

**核心目标**：将浮盈变成实际收益

```
常见止盈策略：

目标止盈：
"盈利达到30%卖出"

分批止盈：
"盈利20%卖1/3，盈利50%再卖1/3，剩余用移动止损"

估值止盈：
"PE超过历史均值的1.5倍时考虑卖出"
```

### 时间止损：释放资金

**核心目标**：避免长期占用资金却无收益

```
时间止损示例：

"持有6个月，既没涨也没跌超过5%，重新评估是否继续持有"

逻辑：
资金有成本（机会成本）
长时间不涨 = 可能逻辑有问题
不如释放资金寻找更好机会
```

## 第二部分：止损的心理突破（5:00-8:00）

### 止损为什么这么难

**心理障碍1：损失厌恶**
```
从EP16回顾：
亏100元的痛苦 ≈ 赚200-300元的快乐

结果：
不愿意承认亏损 → 持有等回本 → 越亏越多
```

**心理障碍2：沉没成本**
```
"我已经亏了这么多了，不能就这样卖了"

正确思维：
过去亏多少与未来是否会涨无关
应该问：如果现在没有持仓，我会买入吗？
```

**心理障碍3：自我认同**
```
"承认止损 = 承认我错了 = 我不够聪明"

正确思维：
止损 = 我按规则操作 = 我是合格的投资者
错误不可怕，不改正才可怕
```

### 重新定义止损

```
旧定义：止损 = 认输 = 失败
新定义：止损 = 执行计划 = 专业

止损意味着：
✓ 我有规则
✓ 我遵守规则
✓ 我保护了本金
✓ 我保留了战斗力
```

## 第三部分：各类止损方法详解（8:00-12:00）

### 固定比例止损

**设置方法**：
```
买入价 × (1 - 止损比例) = 止损价

例：
买入价 = 100元
止损比例 = 10%
止损价 = 100 × 0.9 = 90元
```

**如何选择比例**：

| 波动类型 | 建议止损 | 适用 |
|---------|---------|------|
| 低波动 | 5-8% | 大盘股、ETF |
| 中波动 | 8-15% | 普通个股 |
| 高波动 | 15-25% | 小盘股、成长股 |

### 移动止损

**设置方法**：
```
持有期间最高价 × (1 - 移动止损比例) = 止损价

例：
买入100，涨到150
移动止损比例 = 20%
止损价 = 150 × 0.8 = 120元

即使触发止损，仍然盈利20%
```

**优点**：保护利润，让利润奔跑

### ATR止损

**ATR（平均真实波幅）**：衡量资产近期波动幅度

```
止损价 = 当前价 - N × ATR

例：
当前价 = 100
ATR = 5
N = 2
止损价 = 100 - 2×5 = 90

好处：自动适应波动大小
波动大 → ATR大 → 止损距离大
波动小 → ATR小 → 止损距离小
```

### 技术止损

```
常见技术止损位：

1. 关键支撑位下方
   跌破前期低点止损

2. 均线下方
   跌破20日均线止损

3. 趋势线下方
   跌破上升趋势线止损
```

### 条件止损

```
当买入逻辑不再成立时止损

例如：
买入原因："公司即将推出新产品"
止损条件："新产品推出失败或延期"

原因消失 = 卖出理由出现
```

## 第四部分：止盈策略详解（12:00-14:00）

### 目标止盈

```
设定明确盈利目标

例：
"盈利30%全部卖出"
"盈利50%卖出一半"
```

### 分批止盈

```
分批锁定收益

例如三批止盈：
盈利20%：卖出30%仓位
盈利50%：卖出30%仓位
盈利100%或触发移动止损：卖出剩余

好处：
既锁定部分利润
又保留上涨空间
```

### 估值止盈

```
基于估值而非价格卖出

卖出条件：
- PE超过历史90%分位
- 股息率低于1%
- 估值明显透支未来
```

### 不卖的情况

```
长期持有策略：

如果持有优质公司，可能不主动卖出
除非：
- 基本面发生根本变化
- 估值极端泡沫化
- 有更好的机会
```
