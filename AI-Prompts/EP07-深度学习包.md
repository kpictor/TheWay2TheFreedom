## 📚 模块一：概念深化

### Prompt 1.1 - 资产负债表思维

```markdown
你是一位既懂财务又懂投资的导师。

请帮我理解"用资产负债表思维看个人财务"：

1. **什么是资产**：区分"真资产"和"伪资产"（如贬值的车）
2. **什么是负债**：显性负债vs隐性负债（如未来承诺）
3. **净资产计算**：为什么净资产才是真实财富
4. **流动性视角**：资产的变现能力差异
5. **时间维度**：资产负债的动态变化

请用具体数字例子说明每个概念。
```

### Prompt 1.2 - 可投资资金计算

```markdown
帮我理解"可投资资金"的真正含义：

引导我思考：
1. 总资产中有多少是"锁定"的？（房产、保险等）
2. 需要留多少紧急备用金？（6个月支出？12个月？）
3. 未来确定的大额支出有哪些？
4. 什么钱是"真正可以亏掉也不影响生活"的？
5. 我的可投资资金应该是多少？

请用问答方式帮我梳理清楚。
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 财务状况诊断

```markdown
请对我进行一次财务状况诊断，通过问答引导我：

问题序列：
1. 你清楚自己有多少存款吗？精确到万。
2. 你有几项负债？各是多少？
3. 你每月必须支出多少？可压缩多少？
4. 如果失业，你可以维持多久？
5. 你现在投资的钱，亏50%会影响生活吗？

根据我的回答，给我一个财务健康评分（1-10）。
```

### Prompt 3.2 - 财务决策模拟

```markdown
和我进行财务决策模拟：

场景：你有100万可支配资产，考虑以下决策
- A: 全部买房首付
- B: 全部投资股市
- C: 50万首付 + 50万投资
- D: 留20万备用 + 40万首付 + 40万投资

请引导我思考：
1. 每个选择的利弊是什么？
2. 在什么情况下每个选择最优？
3. 我的实际情况应该选择哪个？
4. 有没有更好的方案？
```
