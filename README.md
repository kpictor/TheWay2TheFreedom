## 📌 核心原则

1. **第一性原理** - 剥离表象，回归本质
2. **极度求真** - 直面错误，记录分析
3. **费曼学习** - 学习 → 教授 → 识别盲区 → 优化
4. **系统进化** - 原则化决策，持续迭代
