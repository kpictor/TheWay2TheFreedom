# EP35 - AI在投资中的应用 | AI深度学习包

> 🎯 目标：明确AI的能力边界，学会正确使用AI辅助投资决策

---

## 🔍 本期特色学习路径

**本期最佳学习方式：边界认知+实际操作**
- 关键是知道AI能做什么、不能做什么
- 通过实际Prompt练习，而非理论学习

---

## 🚫 第一步：认清边界

### Prompt A - AI能力边界对话

```markdown
请和我进行一次"能力边界"对话。

对于以下请求，请诚实告诉我你能不能做到，为什么：

1. "预测明天A股涨还是跌"
2. "告诉我现在应该买什么股票"
3. "帮我分析这家公司的基本面"
4. "帮我理解市场结构"
5. "帮我检验我的投资逻辑有没有漏洞"
6. "帮我生成投资研究报告的摘要"
7. "帮我识别我刚才决策中的认知偏差"

对每个请求，说明：
- 能/不能/部分能
- 原因是什么
- 正确的使用方式是什么
```

---

## 🛠️ 第二步：实用Prompt练习

### Prompt B - 公司研究助手

```markdown
我想研究[填入公司名称]这家公司。

请帮我：
1. 梳理公司的主营业务和商业模式
2. 列出主要的竞争对手
3. 指出这家公司可能的优势和劣势
4. 提出5个我应该深入研究的问题
5. 指出这个分析可能的局限性

注意：你能提供的是框架和问题，不是买卖建议。
```

### Prompt C - 研报解读助手

```markdown
我刚读了一份关于[行业/公司]的研报，核心观点是：
[粘贴研报要点]

请帮我：
1. 用更简单的语言解释核心逻辑
2. 指出这个逻辑链条中可能的薄弱环节
3. 提出反方可能的论点
4. 这份研报可能的利益冲突是什么？
5. 我应该自己验证什么？
```

### Prompt D - 决策挑战者

```markdown
我正在考虑[买入/卖出/持有] [标的名称]，理由是：
[描述你的理由]

请扮演"魔鬼代言人"：
1. 给出至少3个反对这个决策的理由
2. 指出我的逻辑中可能的漏洞
3. 用最坏情况测试我的决策
4. 问我3个尖锐的问题
5. 最后，让我自己做决定，不要给建议
```

### Prompt E - 信息过滤助手

```markdown
我看到一条财经新闻/消息：
[粘贴新闻内容]

请帮我分析：
1. 这是新信息还是已知信息？
2. 消息来源可靠吗？
3. 是否可能是有意释放的？
4. 已经反映在价格中了吗？
5. 我需要立即行动吗？
```

## ⚠️ 第三步：使用红线

### AI使用禁区清单

❌ **绝不能做的事**
1. 让AI直接告诉你买什么
2. 信任AI的价格预测
3. 用AI代替自己的决策
4. 假设AI有你不知道的信息
5. 不验证就接受AI的分析

✅ **应该做的事**
1. 让AI帮你整理信息
2. 让AI挑战你的观点
3. 让AI提出你没想到的问题
4. 让AI解释复杂概念
5. 自己做最终决策

## 🎬 第四步：费曼输出

### Prompt F - 视频脚本

```markdown
我要录制3分钟视频，解释"如何正确使用AI辅助投资"。

要点：
1. 开场：AI不是预言家，也不是专家
2. 用具体例子展示AI能做什么
3. 用具体例子展示AI不能做什么
4. 展示几个我实际使用的Prompt
5. 强调：AI是工具，你是决策者
```
