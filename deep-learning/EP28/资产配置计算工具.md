# EP28 - 资产配置计算工具

> 设计和计算你的个人资产配置方案

---

## 🎯 使用说明

本工具帮助你：
1. 评估风险偏好确定股债比例
2. 设计个人资产配置方案
3. 跟踪和再平衡

---

## 📋 工具一：风险偏好与配置对照

### 根据风险偏好确定基础比例

```
风险偏好配置对照表
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第一步：确认你的风险偏好】

回顾EP08风险测评，或回答以下问题：

如果你的投资一个月内下跌20%，你会：
○ A. 恐慌，想全部卖出
○ B. 焦虑，但能坚持
○ C. 担忧，可能减仓一些
○ D. 平静，甚至考虑加仓

根据答案：
A → 保守型
B → 稳健偏保守
C → 稳健型
D → 积极型

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第二步：确定基础股债比例】

         │ 股票 │ 债券+现金 │ 预期年化 │ 最大回撤
━━━━━━━━━┿━━━━━┿━━━━━━━━━━━┿━━━━━━━━━┿━━━━━━━━
保守型    │ 20-30%│ 70-80%   │ 4-6%    │ 10-15%
稳健保守  │ 30-40%│ 60-70%   │ 5-7%    │ 15-20%
稳健型    │ 50-60%│ 40-50%   │ 6-8%    │ 20-30%
积极型    │ 70-80%│ 20-30%   │ 7-9%    │ 30-40%
激进型    │ 90-100│ 0-10%   │ 8-10%   │ 40-50%

我选择：_______型
股票目标比例：____%
债券+现金目标比例：____%
```
