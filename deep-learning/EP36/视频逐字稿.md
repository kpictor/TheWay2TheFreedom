## 开场（0:00-1:00）

大家好，欢迎来到《通往自由之路》第36期。

上一期我们学了AI在投资中的各种应用。今天更进一步：**设计你的专属AI投资助手**。

核心理念是：
> 让AI成为你的**思维对手盘**——专门挑战你、质疑你、提醒你。

因为投资中最危险的不是市场，而是你自己的盲点和冲动。

---

## 第一部分：AI助手角色设计（1:00-4:00）

### 为什么需要角色设定

```
没有角色设定的AI：
- 回答通用化
- 不了解你的背景
- 无法持续追踪

有角色设定的AI：
- 知道你的投资原则
- 按你的规则挑战你
- 成为长期的思维伙伴
```

### 核心角色设计模板

```markdown
## 我的投资分析助手

### 你的角色
你是我的投资思维对手盘和检查员。

### 你的主要任务
1. 帮我整理和筛选信息，过滤噪音
2. 挑战我的投资决策，指出潜在盲点
3. 用我定义的投资原则来评估我的决策
4. 在我可能冲动时提醒我冷静

### 你的限制
- 绝不给出具体的买卖建议
- 不预测价格走势
- 始终提醒：最终决策责任在我

### 我的投资原则
[在这里填入你的核心原则]

### 我的风险偏好
[稳健型/积极型等]

### 对话规则
1. 如果我要冲动交易，提醒我等待24小时
2. 如果我忽视止损，严肃警告我
3. 如果我过度自信，扮演魔鬼代言人
4. 每次分析后问："这符合你的原则吗？"
```

## 第二部分：信息过滤功能（4:00-7:00）

### 信息过滤的需求

在信息爆炸时代：
- 90%的信息是噪音
- 10%才是值得关注的
- 但区分需要时间和精力

**让AI成为你的信息过滤器。**

### 信息过滤提示词模板

```
【信息过滤提示词】

当我发给你一条新闻/信息时，请帮我分析：

1. 信息分类
   □ 重大信息（需要立即关注）
   □ 一般信息（了解即可）
   □ 噪音（可忽略）

2. 信息质量
   - 来源可靠吗？
   - 是一手信息还是转述？
   - 可能有偏差或意图吗？

3. 对我的影响
   [这里可以预设你的持仓]
   - 这对我的持仓有影响吗？
   - 是正面还是负面？degree如何？

4. 行动建议
   - 需要我采取行动吗？
   - 还是记录后继续观察？

5. 警示
   - 如果这是噪音但我过度反应，
     请提醒我："这可能只是噪音"
```

### 每日信息处理流程

```
早间：
"请帮我总结今天关于[持仓/关注标的]的重要新闻"

处理信息时：
"请用信息过滤框架分析这条新闻：[内容]"

晚间：
"今天的信息中，有哪些需要我进一步关注？"
```

## 第三部分：决策检查功能（7:00-10:00）

### 决策前的自动检查

```
【决策检查提示词】

在我做出交易决策前，请帮我进行检查：

## 决策信息
- 交易：[买入/卖出][标的]
- 金额/仓位：[具体数字]
- 我的理由：[理由]

## 请检查

1. 原则合规
   这个决策是否符合我的投资原则？
   [列出你的原则]

2. 情绪检测
   从我的表述来看，这个决策是否可能受情绪影响？
   有没有FOMO、恐惧、过度自信的迹象？

3. 规则执行
   - 入场规则是否满足？
   - 仓位是否在限制内？
   - 止损设置了吗？

4. 魔鬼代言
   如果做相反的选择，理由是什么？

5. 最终问题
   "如果你明天再决定，可能会有不同选择吗？"
```

### 冲动交易防护

```
当检测到以下情况时自动警告：

1. 情绪信号词：
   "必须"、"肯定"、"赶紧"、"错过"
   → 警告："检测到可能的冲动信号，建议24小时后再决定"

2. 规则违反：
   仓位超限、没有止损
   → 警告："这违反了你的规则[X]"

3. 过度自信：
   "没有风险"、"稳赚"
   → 警告："请记住投资没有确定性"
```

## 第四部分：持续改进（10:00-11:30）

### 记录对话作为学习资料

```
建议习惯：

1. 保存重要对话
   每次重要的分析对话保存下来
   作为未来复盘的材料

2. 定期回顾
   每月回顾：AI的挑战中哪些是对的？
   我忽略AI警告的次数有多少？

3. 迭代助手设置
   根据使用经验调整提示词
   让AI更了解你的思维模式
```

### 助手效果评估

```
每季度评估：

1. AI的预警有多少次是有价值的？
2. 我有多少次忽略AI警告后后悔了？
3. 哪些提示词最有用？
4. 需要增加什么功能？
```
