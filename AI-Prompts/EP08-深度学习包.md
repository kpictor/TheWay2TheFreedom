# EP08 - 风险容忍度自测 | AI深度学习包

> 🎯 目标：科学评估自己的风险承受能力，区分主观意愿和客观能力

---

## 📚 模块一：概念深化

### Prompt 1.1 - 风险容忍度的双维度

```markdown
帮我理解风险容忍度的两个维度：

**客观风险承受能力**（你能承受多少）
- 由财务状况决定
- 由投资期限决定
- 由收入稳定性决定
- 可以精确计算

**主观风险偏好**（你愿意承受多少）
- 由性格决定
- 由过去经历决定
- 由情绪反应决定
- 需要测试发现

请解释：
1. 为什么这两个维度经常不一致？
2. 当它们冲突时，应该听从哪个？
3. 如何让它们趋于一致？
```

### Prompt 1.2 - 情绪测试深度解读

```markdown
请为我设计一个情绪化的风险测试（不是问卷，而是情境模拟）：

场景1：
"你投入了一年收入，一周后账户显示-20%。描述你的第一反应。"

场景2：
"你的股票涨了50%，但你没有卖。第二天暴跌20%，现在只有+30%。你的想法是？"

场景3：
"连续3个月你的收益率都低于银行理财。你会怎么做？"

根据我的回答，分析我真实的风险偏好（不是我以为的）。
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 历史回撤压力测试

```markdown
用历史数据进行压力测试：

假设你按不同比例配置股票：

| 股票比例 | 2008年最大回撤 | 2015年A股回撤 | 2020年3月回撤 |
|----------|----------------|----------------|----------------|
| 100% | -50% | -45% | -35% |
| 70% | -35% | -32% | -25% |
| 50% | -25% | -23% | -18% |
| 30% | -15% | -14% | -10% |

问我：
1. 你能接受哪个级别的回撤？
2. 当真正发生时，你确定不会恐慌卖出吗？
3. 这个回撤发生在你的钱上是多少金额？
4. 这些钱对你意味着什么？（几个月工资？）

帮我做诚实的自我评估。
```

### Prompt 3.2 - 目标与风险的平衡

```markdown
帮我思考目标与风险的关系：

对话流程：
1. 你的投资目标是什么？（10年翻倍？年化8%？）
2. 实现这个目标需要承担多大风险？
3. 你能承受的风险能实现这个目标吗？
4. 如果不能，你选择：调低目标 还是 接受更大风险？
5. 有没有第三条路？

帮我找到平衡点。
```
