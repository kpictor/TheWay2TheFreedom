## EP07 - 认知自己的财务状况

### 核心AI提示词

```markdown
请扮演一位理财规划师，帮我评估个人财务状况。

根据我提供的信息：
1. 计算我的净资产
2. 分析我的资产负债结构是否健康
3. 评估我真正可用于投资的资金
4. 给我资产配置的建议
5. 指出需要改进的地方

我会提供我的财务数据：[开始对话]
```

## EP08 - 风险容忍度自测

### 核心AI提示词

```markdown
请为我进行一个交互式的风险偏好测试。

规则：
1. 用情景问题而非抽象问题
2. 根据我的回答追问
3. 区分我"以为的"和"真实的"风险偏好
4. 最后给出评估和建议

开始测试：[等待第一个问题]
```

## EP09 - 投资目标的设定

### 核心AI提示词

```markdown
帮我设定SMART投资目标。

对话流程：
1. 问我为什么要投资（目的）
2. 问我的时间期限
3. 问我需要多少钱
4. 检验目标是否合理
5. 帮我重新表述为SMART格式
6. 指出潜在的问题和风险
```

## EP10 - 时间的复利效应

### 核心AI提示词

```markdown
请用复利模拟器帮我理解时间的力量：

1. 根据我提供的：初始金额、年化收益、投资年限
2. 计算最终金额
3. 分解前10年和后10年的差异
4. 展示不同收益率的差距
5. 告诉我"开始投资"比"完美时机"更重要

请给我具体的数字震撼。
```

## EP11 - 成本的隐形杀手

### 核心AI提示词

```markdown
请帮我计算投资成本的真实侵蚀：

根据我提供的：
- 交易频率
- 佣金率
- 基金管理费
- 投资年限

计算：
1. 累计支付的费用总额
2. 这些费用如果用于投资能增值多少
3. 真正的收益率vs名义收益率
4. 给我降低成本的具体建议
```
