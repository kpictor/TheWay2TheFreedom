## 📚 模块一：概念深化

### Prompt 1.1 - 什么是因子

```markdown
帮我理解"因子"的概念：

**定义**
因子是能够解释股票收益差异的特征/属性

**经过学术验证的经典因子**

| 因子 | 定义 | 逻辑 |
|------|------|------|
| **价值** | 低PE/PB | 低估值长期回归 |
| **动量** | 过去涨得好 | 趋势延续 |
| **质量** | 高ROE/低负债 | 好公司长期胜出 |
| **规模** | 小市值 | 小公司增长空间大 |
| **低波动** | 波动性小 | 风险收益悖论 |
| **红利** | 高股息率 | 稳定现金流 |

**因子的超额收益来源**
1. 风险溢价：承担特定风险获得补偿
2. 行为偏差：市场错误定价
3. 结构因素：机构约束

每个因子的超额收益来源是什么？
```

### Prompt 1.2 - 因子的使用方式

```markdown
介绍因子投资的不同方式：

**单因子策略**
- 只用一个因子筛选
- 如纯价值策略
- 风险：因子失效时期

**多因子策略**
- 综合多个因子打分
- 分散单一因子风险
- 如何组合因子？

**因子轮动**
- 根据市场环境切换因子
- 哪个环境用什么因子？
- 难点：判断环境

**Smart Beta ETF**
- 通过ETF获得因子敞口
- 低成本简单操作
- 常见的因子ETF
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 因子筛选实践

```markdown
帮我用因子方法筛选股票：

请选择2-3个因子，设定具体条件：

例如：
- 价值因子：PE < 行业中位数
- 质量因子：ROE > 15%
- 动量因子：近6个月涨幅 > 0

我会帮你：
1. 评估条件的合理性
2. 讨论不同因子的权衡
3. 给出筛选建议

你想用什么因子组合？
```

### Prompt 3.2 - 因子失效风险

```markdown
讨论因子投资的风险：

**因子可能失效**
- 价值股持续跑输成长股（2010-2020年美股）
- 动量崩溃（2009年）
- 小盘股长期不涨（A股2017-2020）

**如何应对？**
1. 多因子分散
2. 长期坚持
3. 接受均值回归需要时间

**思考**
- 如果一个因子1年没跑赢，你会放弃吗？
- 如果3年呢？5年呢？
- 你的信念从何而来？
```
