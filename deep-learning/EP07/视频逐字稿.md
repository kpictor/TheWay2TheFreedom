# EP07 - 视频逐字稿

> 《认知自己的财务状况》教学视频脚本

---

## 🎬 开场（0:00-0:30）

**【画面】** 各种投资广告闪过："年化10%"、"稳健增值"、"财富自由"

**【旁白】**
> 每天都有人告诉你该买什么、投资什么。
>
> 但在此之前，有一个更基础的问题：你有多少钱可以投资？

**【画面】** 切换到一个人面对账本思考

**【旁白】**
> 今天，我们从最基础的开始：认知自己的财务状况。

---

## 📍 第一部分：像企业一样管理财务（0:30-3:00）

**【画面】** 企业财务报表三张表

**【旁白】**
> 每家公司都有三张核心财务报表。
>
> 你的个人财务也应该有。

**【动画】**
```
资产负债表：你有什么？欠什么？
利润表：    你赚多少？花多少？
现金流量表：钱从哪来？到哪去？
```

### 资产负债表

**【动画】**

```
┌─────────────────────────────────────────┐
│           你的资产负债表                 │
├─────────────────────────────────────────┤
│ 资产（你有什么）   │ 负债（你欠什么）   │
├─────────────────────────────────────────┤
│ 现金存款           │ 房贷               │
│ 股票基金           │ 车贷               │
│ 房产               │ 消费贷             │
│ 其他               │ 其他               │
├─────────────────────────────────────────┤
│        净资产 = 资产 - 负债              │
└─────────────────────────────────────────┘
```

**【旁白】**
> 做个简单的练习：花5分钟，把你所有的资产和负债列出来。
>
> 很多人第一次做这个练习时，会发现：
>
> "原来我没有自己以为的那么有钱"
>
> 或者：
>
> "原来我欠了这么多"

---

## 📍 第二部分：可投资金额的真相（3:00-5:30）

**【画面】** 计算器和账单

**【旁白】**
> 知道自己有多少钱，还不够。
>
> 关键问题是：有多少钱可以真正用来投资？

**【动画】**
```
你以为的可投资金额：
= 银行卡余额 + 理财产品

真正的可投资金额：
= 流动资产
  - 6个月紧急备用金
  - 未来1年确定支出
  - 已承诺的支出
```

**【举例】**

```
小王的情况：
  存款：50万
  
他以为50万都能投资。

但实际上：
  - 紧急备用金（6个月生活费）：6万
  - 明年要交的房租押金：3万
  - 计划明年换车：15万
  - 可能的医疗支出：2万
  
真正可投资：50 - 6 - 3 - 15 - 2 = 24万

差了一半！
```

---

## 📍 第三部分：紧急备用金（5:30-7:30）

**【画面】** 消防员的形象

**【旁白】**
> 为什么一定要留紧急备用金？
>
> 因为生活不会按计划进行。

**【场景动画】**

```
场景A：有紧急备用金的人
  → 突然失业
  → 使用备用金维持生活
  → 从容找工作
  → 投资组合不受影响

场景B：没有紧急备用金的人
  → 突然失业
  → 被迫卖出基金（可能正好亏损）
  → 焦虑中找工作
  → 财务和情绪双重打击
```

**【旁白】**
> 紧急备用金不是浪费，是保护。
>
> 它保护你不用在最糟糕的时候被迫卖出投资。

**【建议】**
```
紧急备用金标准：
- 单身：3-6个月必要支出
- 有家庭：6-12个月必要支出
- 收入不稳定：12个月或更多
```

---

## 📍 第四部分：财务健康诊断（7:30-9:30）

**【画面】** 体检报告

**【旁白】**
> 就像身体需要体检，财务也需要诊断。
>
> 几个关键指标：

**【动画表格】**

```
1. 储蓄率 = 每月储蓄 ÷ 每月收入
   健康标准：> 20%

2. 负债率 = 总负债 ÷ 总资产
   健康标准：< 50%

3. 偿债比 = 每月还款 ÷ 每月收入
   健康标准：< 40%

4. 紧急储备 = 流动资产 ÷ 每月必要支出
   健康标准：> 6个月
```

**【旁白】**
> 如果你的指标不健康，不要着急投资。
>
> 先把财务基础打好。
>
> 地基不稳，楼盖得再高也会塌。

---

## 📍 总结（9:30-10:00）

**【画面】** 整理好的账本

**【旁白】**
> 投资的第一步，不是选股票、选基金。
>
> 而是清楚地知道自己有多少筹码。

**【三个行动】**
```
今天就做：
1. 列出所有资产和负债
2. 计算真正可投资的金额
3. 给自己做一次财务体检
```

**【结束语】**
> 下一期，我们继续认识自己：
>
> 你的风险容忍度是多少？能承受多大的亏损？
