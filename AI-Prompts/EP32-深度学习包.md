## 📚 模块一：概念深化

### Prompt 1.1 - 风险预算的思维转变

```markdown
帮我理解"风险预算"的思维方式：

**传统思维**
"我想在这只股票上赚多少" → 决定买多少

**风险预算思维**
"我能承受这只股票亏多少" → 决定买多少

**为什么要转变？**
1. 思考收益是贪婪驱动
2. 思考风险是理性驱动
3. 风险可以预设，收益无法控制

**核心公式**
仓位 = 可承受亏损金额 / 止损幅度

示例：
- 账户10万
- 单笔最大风险1%（1000元）
- 止损设在买入价-10%
- 仓位 = 1000 / 10% = 10000元（10%仓位）

用你自己的数字练习计算。
```

### Prompt 1.2 - 风险预算的层次

```markdown
介绍风险预算的不同层次：

**第一层：单笔风险**
- 一笔交易最多亏账户的X%
- 建议范围：0.5%-2%
- 这个数字如何确定？

**第二层：持仓风险**
- 单只股票的全部风险
- 考虑加仓后的累计风险

**第三层：组合风险**
- 所有持仓的总风险
- 相关性的影响
- 极端情况下的最大亏损

**第四层：机会成本风险**
- 现金比例的考量
- 未来机会的保留

如何平衡这四层风险？
```
