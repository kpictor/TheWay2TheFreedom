# EP06 - 历史案例研究

> 基金行业的激励扭曲案例

## 📜 案例1：中国公募基金的"规模诅咒"

### 背景

**现象**：
> "明星基金-规模暴涨-业绩下滑"的怪圈

### 典型时间线

```
2019年：某消费主题基金
        规模：50亿  收益：+80%
              │
              ▼
2020年Q1：媒体追捧，投资者涌入
          规模：200亿  收益：+30%
              │
              ▼
2020年Q4：狂热期
          规模：600亿  收益：+50%
              │
              ▼
2021年：高位接盘者入场后
        规模：800亿  收益：-15%
              │
              ▼
2022年：
        规模：400亿（赎回）  收益：-25%

结果：
- 早期投资者（50亿时）：赚翻
- 高位入场者（800亿时）：巨亏
- 基金公司：管理费收入创历史新高
```

### 激励分析

| 参与者 | 激励 | 行为 | 结果 |
|--------|------|------|------|
| 基金公司 | 扩大AUM | 牛市末期大力营销 | 收取大量管理费 |
| 基金经理 | 排名+规模奖金 | 配合营销 | 管理难度增加 |
| 销售渠道 | 销售佣金 | 推荐"明星基金" | 赚取尾随佣金 |
| 投资者 | 追逐收益 | 高位买入 | 亏损 |

### 核心教训

> 基金规模是业绩的敌人。当所有人都在谈论一只基金时，往往是最危险的时刻。

## 📜 案例2：LTCM的激励失衡

### 背景

**长期资本管理公司（1994-1998）**：
- 创始人：诺贝尔经济学奖得主
- 策略：高杠杆套利
- 业绩：年化40%+

### 激励结构

```
合伙人投入：自有资金（利益绑定）
杠杆：25-30倍（放大收益和风险）
业绩费：25%
        │
        ▼
      激励扭曲
        │
        ├── 成功：巨额回报
        │
        └── 失败：最多亏光本金
                  ↓
            但"最多亏光"的假设是错的
            因为超高杠杆可能亏超过本金
```

### 崩盘过程

```
1997-1998：
├── 亚洲金融危机
├── 俄罗斯债务违约
├── 市场恐慌
        ↓
模型假设失效：
├── "价差会收敛" → 价差反而扩大
├── "流动性充裕" → 流动性枯竭
        ↓
1998年9月：
├── 资产46亿→只剩4亿
├── 杠杆敞口1250亿
├── 濒临破产
        ↓
美联储协调14家银行注资36亿救援
```

### 激励分析

| 激励缺陷 | 表现 | 后果 |
|----------|------|------|
| 过度自信 | 两位诺奖得主 | 忽视尾部风险 |
| 杠杆诱惑 | 25倍放大收益 | 也放大损失 |
| 短期激励 | 年化40%吸引投资 | 忽视长期风险 |
| 利益绑定不足 | 合伙人资金<总资产 | 出问题时不够痛 |

### 核心教训

> 即使是最聪明的人，在错误的激励结构下，也会做出灾难性决策。

## 📜 案例3：2008年投行激励危机

### 背景

**次贷危机中的华尔街**：
- 承销有毒资产（MBS/CDO）
- 同时对赌这些资产会跌

### 激励结构

```
投行员工的薪酬：
├── 高额年终奖：当年业绩决定
├── 股票期权：延迟但可抛售
└── 解雇风险：业绩不佳走人
        │
        ▼
    激励时间错配
        │
        ├── 短期：交易量越大奖金越高
        │
        └── 长期风险：反正到时候我已经离职了
```

### 经典案例

**高盛的"做多做空"两手**：

```
2006-2007：
├── 向客户销售次贷产品（赚销售费）
├── 同时做空这些产品（赚下跌收益）
        ↓
客户亏损，高盛两边赚钱
        ↓
2010年：SEC起诉高盛欺诈
        支付5.5亿美元和解
```

### 激励分析

| 利益冲突 | 激励扭曲 | 系统性后果 |
|----------|----------|------------|
| 销售佣金 | 推销有毒产品 | 风险扩散到全系统 |
| 年终奖文化 | 只看短期 | 埋下长期炸弹 |
| 公司vs客户 | 客户不是第一 | 信任崩塌 |

### 核心教训

> 当一个人的奖金取决于短期业绩，而风险会在长期爆发时，他会选择做短期的事。

## 📜 案例4：中国私募"跑路"事件

### 背景

**2015-2018年私募行业乱象**：
- 大量私募"画饼"吸金
- 承诺高收益保本
- 最终人去楼空

### 典型模式

```
第一阶段：包装
├── 高端写字楼
├── 名人背书
├── 虚假业绩展示
        ↓
第二阶段：吸金
├── 承诺10-20%年化
├── 保本承诺
├── 老客户介绍新客户
        ↓
第三阶段：挪用
├── 实际投资很少
├── 庞氏结构：新钱付老利
├── 大量资金被转移
        ↓
第四阶段：崩盘
├── 新资金不够付利息
├── 跑路或被查
└── 投资者血本无归
```

### 激励分析

| 问题 | 根源 | 教训 |
|------|------|------|
| 缺乏监管 | 私募门槛低 | 看资质很重要 |
| 信息不透明 | 没有公开披露 | 要求定期报告 |
| 托管缺失 | 资金和投资不分离 | 必须有托管 |
| 承诺保本 | 违反投资常识 | 高收益必有高风险 |

### 核心教训

> 当激励结构有漏洞时，坏人会钻进来。保护自己的第一步是理解对方的激励。

## 💡 历史规律总结

### 四个案例的共性

| 共性 | 表现 | 投资者应对 |
|------|------|------------|
| **利益错配** | 管理人激励与投资者不一致 | 分析激励结构 |
| **短期主义** | 考核周期太短 | 关注长期记录 |
| **规模诅咒** | 大了之后业绩下滑 | 警惕明星基金 |
| **信息不对称** | 投资者难以监督 | 选择透明产品 |

### 保护自己的原则

1. **问"他怎么赚钱"**：理解激励结构
2. **看费率**：确定损失要最小化
3. **查历史**：长期业绩比短期更重要
4. **看规模**：太大的基金要警惕
5. **验证托管**：资金和管理要分离
