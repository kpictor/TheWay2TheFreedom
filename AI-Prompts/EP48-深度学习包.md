## 📝 第一步：原则来源回顾

### 我的原则从哪里来？

```markdown
原则的五个来源：

1. **理论学习**（Q1-Q3学到的）
   - 关于市场的认知
   - 关于风险的认知

2. **自我发现**（Q2情绪和偏差）
   - 我的心理弱点
   - 我的行为模式

3. **实践教训**（Q4实盘）
   - 成功经历
   - 失败教训

   - Dalio、巴菲特等的原则
   - 经过验证选择性采纳

5. **错误分析**（EP46）

```

### Prompt A - 回顾52期核心洞察

```markdown
请帮我回顾52期学习中最重要的洞察：

对于每个季度，帮我回答：
- Q1（认识自我与市场）：最重要的3个认知是？
- Q2（理解博弈与行为）：最重要的3个认知是？

这些将成为我原则的来源。
```

## 🔬 第三步：原则质量检验

### Prompt B - 原则质量检验

```markdown
请检验我的原则清单质量：

[粘贴你的原则清单]

请评估每条原则：
1. 是否来自真实经验？
2. 是否足够具体可执行？
3. 是否与其他原则矛盾？
4. 是否是你真正相信的？

给出整体评分和改进建议。
```

### 好原则的标准

```markdown
**好原则的5个标准**

✅ 来自真实经验（不是抄来的）
✅ 可直接指导行动（不是空话）
✅ 你真正相信（不是应该相信）
✅ 经过检验（用过且有效）
✅ 简洁明确（能记住）

**坏原则的特征**

❌ "要理性投资"（太空泛）
❌ "低买高卖"（废话）
❌ 从书上抄来但没验证
❌ 互相矛盾
```
