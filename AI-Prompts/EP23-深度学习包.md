# EP23 - 市场周期的心理阶段 | AI深度学习包

> 🎯 目标：掌握市场情绪周期的9个阶段，学会定位当前位置

---

## 📚 模块一：概念深化

### Prompt 1.1 - 情绪周期的9个阶段

```markdown
帮我深入理解市场情绪周期的完整循环：

**上升阶段**
1. **怀疑** - "这次反弹能持续吗？"
2. **希望** - "也许真的在复苏"
3. **乐观** - "经济确实在好转"
4. **兴奋** - "这是大牛市！"
5. **狂热** - "这次不一样！买什么都涨！"

**下跌阶段**
6. **否认** - "只是调整，很快会回来"
7. **恐惧** - "糟糕，可能亏钱了"
8. **绝望** - "一切都完了，永远不会回来"
9. **回到怀疑** - "市场开始稳定了吗？"

对每个阶段：
- 投资者的典型言论
- 媒体标题特征
- 市场数据信号
- 正确的应对策略
```

### Prompt 1.2 - 识别当前阶段的方法

```markdown
教我如何判断市场处于情绪周期的哪个阶段：

**定性指标**
1. 媒体情绪（标题分析）
2. 周围人的言论
3. 社交媒体热度
4. 投资者调查

**定量指标**
1. 估值水平（PE、PB）
2. 成交量变化
3. 融资余额
4. 新基金发行量
5. 开户数

**综合判断框架**
- 如何权衡这些指标？
- 什么时候指标会给出矛盾信号？
- 如何处理不确定性？
```
