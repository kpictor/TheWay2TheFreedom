## ⚠️ 第一步：认识连续亏损的必然性

### 数学现实

```markdown
请帮我理解连续亏损的概率：

假设我的策略胜率是55%，计算：
- 连续亏损2次的概率？
- 连续亏损3次的概率？
- 连续亏损5次的概率？
- 100次交易中，出现连续3次亏损的概率？

结论：即使策略长期有效，连续亏损也是___
```

### 心理阶段

```markdown
解释亏损时的心理五阶段：

1. **否认**："只是运气不好，会反弹"
   - 危险行为：___
   
2. **愤怒**："市场不公平！有人操控！"
   - 危险行为：___
   
3. **讨价还价**："再给我一次机会"
   - 危险行为：___
   
4. **沮丧**："我不适合投资"
   - 危险行为：___
   
5. **接受**："这是正常的，继续执行系统"
   - 正确行为：___

帮我识别自己目前在哪个阶段。
```

## 🛡️ 第二步：设计应对机制

### 个人熔断机制设计

```markdown
帮我设计个人的"亏损熔断机制"：

**触发条件**（达到任一条件即触发）
- 单日亏损超过账户的___%
- 单周亏损超过账户的___%
- 连续亏损___次
- 情绪分低于___分

**触发后动作**
1. 立即___
2. 暂停交易___天
3. 做___事情冷静
4. 满足___条件后才能恢复交易

**复盘要求**
- 暂停期间必须完成___分析
```

### Prompt A - 情景演练

```markdown
请和我进行一次"连续亏损"情景演练：

假设场景：我连续亏损4次，账户回撤15%

请问我：
1. 你现在什么感觉？
2. 你想做什么？
3. 根据你的规则，你应该做什么？
4. 如果感觉和规则冲突，你听谁的？
5. 你的熔断机制触发了吗？

帮我在安全环境下体验和准备。
```

---

## 🧘 第三步：冷静恢复工具

### Prompt B - 亏损时的自我对话

```markdown
我刚经历了连续亏损，请帮我进行理性自我对话：

当前状态：
- 连续亏损___次
- 账户回撤___%
- 情绪状态（1-10）：___

请帮我分析：
1. 这是系统正常波动还是系统失效？
2. 我的决策质量如何？（规则执行得好吗？）
3. 亏损是运气还是错误？
4. 我现在应该做什么？
5. 给我三个理由说明为什么不应该恐慌
```

### 冷静清单

```markdown
当情绪失控时，按顺序做：

□ 关闭交易软件
□ 离开电脑前
□ 深呼吸10次
□ 走动10分钟
□ 喝一杯水
□ 问自己：24小时后这还重要吗？
□ 只有冷静后才能做决策
```
