## 🎯 第一步：进阶使用模式

### 模式A - 红队挑战

```markdown
我正在考虑投资[标的]，我的分析如下：
[粘贴你的分析]

请扮演红队，从以下角度挑战我：

1. **空头视角**：列出5个做空这个投资的理由
2. **魔鬼代言人**：假设我的逻辑完全错误，为什么？
3. **最坏情况**：如果所有假设都不成立，结果是什么？
4. **盲点检查**：我可能忽略了什么？
5. **心理陷阱**：我是否被某种偏差影响？

请毫不留情地挑战我。
```

### 模式B - 决策评估

```markdown
用我的投资原则评估这个决策：

我的原则清单：
[粘贴原则清单]

我正在考虑的决策：
[描述决策]

请逐条对照：
□ 原则1：符合/违反/无关
□ 原则2：符合/违反/无关
...

综合评估：这个决策与我的原则一致吗？
```

### 模式C - 事后复盘

```markdown
帮我复盘这笔交易：

交易记录：
[粘贴交易日志]

请分析：
1. 决策质量（不看结果）：好/中/差
2. 规则执行情况：是否严格执行
3. 情绪影响：有多大
4. 可以学到什么
5. 如果重来会怎么做
```

## 🔧 第二步：优化你的AI使用

### Prompt A - AI使用效果评估

```markdown
我使用AI辅助投资已经___周了。

请帮我评估使用效果：
1. AI最帮助我的3件事是什么？
2. AI最没用的3件事是什么？
3. 我有没有过度依赖AI？
4. 我有没有忽略AI有用的功能？
5. 如何优化我的使用方式？
```

### Prompt B - 个人AI使用原则

```markdown
帮我制定个人的AI使用原则：

基于我的使用经验，我需要：
- 什么情况下一定要问AI？
- 什么情况下不能只听AI？
- 如何避免过度依赖？
- 如何最大化AI的价值？

请帮我生成3-5条个人AI使用原则。
```
