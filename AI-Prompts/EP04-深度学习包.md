# EP04 - 谁在和你博弈 | AI深度学习包

> 🎯 目标：建立对手盘思维，理解市场各参与者的动机和策略

---

## 📚 模块一：概念深化

### Prompt 1.1 - 参与者深度画像

```markdown
请帮我建立各类市场参与者的深度画像：

**散户**：
- 典型行为模式是什么？
- 信息来源主要是什么？
- 最常犯的错误？

**公募基金**：
- 他们的真实KPI是什么？
- 排名压力如何影响他们的行为？
- 季末效应是什么？

**私募/对冲基金**：
- 不同策略类型有什么区别？
- 他们的信息优势来源？

**做市商/高频交易**：
- 他们如何赚钱？
- 对普通投资者有什么影响？

**主权基金/养老金**：
- 投资期限和约束是什么？
- 行为有什么特点？
```

### Prompt 1.2 - 利益链条分析

```markdown
帮我分析围绕股票交易的利益链条：

1. **券商**：希望你怎么做？为什么？
2. **分析师**：他们的激励是什么？
3. **财经媒体**：什么内容最吸引流量？
4. **上市公司IR**：他们的目标是什么？
5. **基金代销**：什么时候推销最积极？

理解这条利益链后，我应该如何调整信息获取方式？
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 机构交易策略

```markdown
介绍几种常见的机构交易策略，让我理解"专业玩家"在做什么：

1. **价值投资**：机构如何做？和散户有什么不同？
2. **动量策略**：机构如何识别和利用动量？
3. **事件驱动**：并购套利、财报博弈如何操作？
4. **宏观对冲**：如何根据宏观判断配置资产？
5. **统计套利**：量化如何发现价差？

对于每种策略，说明散户能否模仿，风险是什么。
```

### Prompt 2.2 - 市场微观结构

```markdown
帮我理解市场微观结构：

1. **订单簿**：买一卖一意味着什么？
2. **大宗交易**：为什么折价？信息含义？
3. **融资融券**：多空信息如何解读？
4. **北向资金**：为什么被关注？
5. **龙虎榜**：机构席位意味着什么？

这些微观数据如何帮助我识别"对手在做什么"？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 对手模拟器

```markdown
我要买入/卖出 [具体股票]，价格 [具体价格]。

请扮演5种可能的对手盘，分别告诉我：

1. **对手A - 私募基金**：为什么愿意和你交易？
2. **对手B - 公司内部人**：可能知道什么？
3. **对手C - 做市商**：纯粹的流动性提供？
4. **对手D - 另一个散户**：和你一样的处境？
5. **对手E - 量化基金**：算法判断是什么？

告诉我哪种情况对我最有利，哪种最不利。
```

### Prompt 3.2 - 信博弈场景

```markdown
和我玩一个投资博弈游戏：

设定：
- 有一只股票，真实价值100元
- 你知道一个消息，但我不知道
- 当前股价95元

作为AI（扮演机构），你会：
1. 如何决定是否出手？
2. 如何控制建仓节奏不被发现？
3. 如何利用散户的反应？
4. 什么时候开始出货？

教我理解"大资金"的行为逻辑。
```

## ✅ 模块五：掌握度验证

### Prompt 5.1 - 交易复盘

```markdown
请我描述最近的一次交易，帮我分析对手盘：

交易详情：[请填写]
- 买入/卖出
- 标的
- 价格
- 时间
- 你当时的理由

我会分析：
1. 对手最可能是谁？
2. 对手的逻辑可能是什么？
3. 这笔交易中谁更可能是对的？
4. 你从这个分析中学到什么？
```
