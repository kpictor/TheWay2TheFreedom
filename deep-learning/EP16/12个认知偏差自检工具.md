# EP16 - 12个认知偏差自检工具

> 识别并对抗你大脑的"出厂设置"

---

## 🎯 使用说明

这份工具帮助你：
1. 深入理解每个认知偏差
2. 识别自己最常犯的偏差
3. 为每个偏差建立"反偏差"机制

---

## 📋 偏差1：确认偏差

### 定义
只关注支持自己观点的信息，忽视反面证据。

### 投资中的表现
- 买入后只看利好消息
- 在喜欢的股票论坛找"同温层"
- 对质疑的声音感到不耐烦

### 自检问题
```
最近一次买入决策回顾：
我看了多少份看多的分析：_____份
我看了多少份看空的分析：_____份
比例是否严重失衡？□是 □否
```

### 反偏差策略
**刻意寻找反面观点**：每次投资决策前，必须找到3条反对理由

```
投资标的：_____________
支持理由：
1. _____________
2. _____________
3. _____________

反对理由（强制自己找）：
1. _____________
2. _____________
3. _____________
```

## 📋 偏差2：锚定效应

### 定义
过度依赖第一个接收到的信息（锚点）做判断。

### 投资中的表现
- 用买入价作为心理锚点
- 不愿在"亏损"状态下卖出
- 被52周高点/低点影响判断

### 自检问题
```
我是否有以下想法：
□ "等回本就卖"
□ "它曾经涨到XX，应该还能回去"
□ "比我买入价跌了，不能卖"

如果有，说明你被锚定了。
```

### 反偏差策略
**重新锚定问题**：

```
忘记你的买入价，问自己：
"如果我现在一股都没有，以当前价格，我会买吗？"

□ 会 → 可以继续持有
□ 不会 → 应该考虑卖出

买入价不应该影响卖出决策！
```

## 📋 偏差3：损失厌恶

### 定义
亏损的痛苦约为同等金额盈利快乐的2-2.5倍。

### 投资中的表现
- 小赚就跑（怕变成亏损）
- 亏损死扛（希望回本）
- 导致"截断利润，让亏损奔跑"

### 自检问题
```
我的交易记录分析：
平均盈利交易持有时间：_____天
平均亏损交易持有时间：_____天

如果亏损持有时间 > 盈利持有时间，你可能有损失厌恶
```

### 反偏差策略
**预设止损和止盈**：

```
买入前就决定：
止损位：_____（亏损_____%卖出）
止盈位：_____（盈利_____%开始考虑卖出）

严格执行，不因"感觉"改变
```

## 📋 偏差4：过度自信

### 定义
高估自己的判断能力和信息优势。

### 投资中的表现
- 认为自己能战胜市场
- 频繁交易
- 仓位过重

### 自检问题
```
诚实回答：
你认为自己的投资能力在所有投资者中排前：_____%

（研究显示：90%的人认为自己高于平均水平，这在数学上不可能）
```

### 反偏差策略
**记录并验证预测**：

```
预测记录：
日期：_______ 我预测：_________________________
实际结果：_________________________
准确吗？□是 □否

累计正确率：_____/_____=_____%

低于50%说明你可能过度自信
```

## 📋 偏差5：后见之明

### 定义
事后觉得一切都是"显而易见"的，高估自己事前的预测能力。

### 投资中的表现
- "我早就知道会跌"（但当时没卖）
- 美化自己的投资记录
- 不愿意承认是运气

### 自检问题
```
回忆一次"预测正确"的经历：
事件：_____________
你说"我早就知道"，但当时：
□ 你实际采取了行动吗？
□ 你有书面记录吗？
□ 你当时真的有把握吗？
```

### 反偏差策略
**事前书面记录**：

```
日期：_______
我预测：_________________________
预测依据：_________________________
采取的行动：_________________________
置信度：_____%

（事后对照，避免记忆美化）
```

## 📋 偏差6：从众效应

### 定义
跟随多数人的行为，认为多数人是对的。

### 自检问题
```
最近一次买入：
□ 因为自己分析决定买入
□ 因为看到很多人在买
□ 因为朋友/大V推荐
□ 因为新闻说在涨

如果选后三项，可能是从众
```

### 反偏差策略
```
在做决策前问：
"如果没有别人的意见，仅凭我自己的分析，我会做同样的决策吗？"
```
