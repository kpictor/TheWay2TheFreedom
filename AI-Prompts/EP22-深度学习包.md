## 📚 模块一：概念深化

### Prompt 1.1 - 反身性的本质

```markdown
帮我深入理解索罗斯的反身性理论：

**传统观点**
- 市场反映基本面
- 价格趋向"真实价值"
- 投资者是被动观察者

**反身性观点**
- 市场参与者的看法影响现实
- 价格可以改变基本面
- 观察者也是参与者

**核心insight**
1. 认知函数：现实→认知（有局限）
2. 参与函数：认知→现实（有影响）
3. 两者相互作用形成循环

用具体例子说明这个循环如何运作。
```

### Prompt 1.2 - 正反馈与负反馈循环

```markdown
解析反身性中的反馈循环：

**正反馈循环（自我强化）**
- 股价上涨 → 更多人买入 → 股价继续涨
- 何时开始？何时加速？何时崩溃？

**实例分析**
- 科技泡沫：叙事→股价→更多投资→验证叙事
- 银行挤兑：恐惧→取款→银行困难→验证恐惧
- 房价泡沫：上涨预期→购买→价格上涨→验证预期

**负反馈循环（自我修正）**
- 什么情况下市场自我修正？
- 为什么负反馈不如正反馈引人注目？

为什么理解这个至关重要？
```
