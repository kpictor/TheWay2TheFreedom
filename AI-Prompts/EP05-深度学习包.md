# EP05 - 信息不对称的真相 | AI深度学习包

---

## 📚 模块一：概念深化

### Prompt 1.1 - 信息传播动力学

```markdown
帮我理解金融信息的传播机制：

1. **信息产生**：什么算"新信息"？谁最先知道？
2. **信息传播**：通过什么渠道传播？速度如何？
3. **信息解读**：同样信息为什么有不同解读？
4. **信息反映**：价格如何"消化"信息？
5. **信息过时**：什么时候信息失去价值？

请用一个具体的利好消息（如业绩超预期）走完这个流程。
```

### Prompt 1.2 - 有效市场假说批判

```markdown
和我讨论有效市场假说（EMH）：

1. EMH的三种形式是什么？
2. 有哪些证据支持EMH？
3. 有哪些证据反对EMH？
4. 行为金融学如何挑战EMH？
5. 对于普通投资者，EMH意味着什么？

最后，告诉我：市场到底有效还是无效？如何利用这个结论？
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 内幕交易案例

```markdown
请介绍几个著名的内幕交易案例：

1. **案例背景**：谁，什么公司，什么消息
2. **如何获取内幕**：信息如何泄露
3. **如何被发现**：监管如何识别异常
4. **法律后果**：处罚是什么
5. **市场启示**：散户能从中学到什么

这些案例如何证明"当你知道消息时已经太晚了"？
```

### Prompt 2.2 - 信息优势来源

```markdown
深入分析机构的信息优势来源：

1. **调研优势**：上市公司调研如何进行？
2. **数据优势**：另类数据（卫星、消费）如何使用？
3. **分析优势**：专业研究团队的价值
4. **速度优势**：新闻解读和交易速度
5. **关系优势**：产业链信息如何获取

对于每种优势，散户有没有可能弥补？如何弥补？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 消息逆向解读

```markdown
训练我的"消息逆向解读"能力。

给我5条常见的财经新闻类型，教我如何逆向思考：

对于每种新闻，回答：
1. 表面含义是什么？
2. 谁希望你看到这条消息？
3. 发布的时机有什么讲究？
4. 市场可能已经提前反映了吗？
5. 相反的解读是什么？
```

### Prompt 3.2 - 迟到新闻识别

```markdown
帮我建立"识别迟到新闻"的框架：

判断标准：
1. 价格是否已经反映？
2. 消息首次出现的时间？
3. 消息来源的级别？
4. 是否有机构早已行动的迹象？

给我练习题：分析以下消息是否"迟到"
[请提供几条最近的财经新闻让我分析]
```
