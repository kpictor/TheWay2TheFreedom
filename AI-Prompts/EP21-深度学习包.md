## 📚 模块一：概念深化

### Prompt 1.1 - 博弈论核心概念

```markdown
帮我系统理解博弈论在投资中的应用：

**博弈类型**
1. **零和博弈**：你赚我赔
   - 期货市场是零和吗？
   - 股票市场是零和吗？

2. **负和博弈**：加上成本后总体损失
   - 高频交易如何把游戏变成负和？
   - 散户在负和博弈中的胜率？

3. **正和博弈**：可以共赢
   - 长期持股是正和吗？
   - 如何把投资变成正和博弈？

**关键概念**
- 纳什均衡：没有人有动力单方面改变策略
- 信息博弈：谁知道更多？
- 重复博弈：长期合作vs短期欺骗
```

### Prompt 1.2 - 囚徒困境与市场

```markdown
用囚徒困境解释市场现象：

**经典囚徒困境**
- 两人各自理性选择
- 却得到集体非理性结果

**市场中的囚徒困境**
1. 恐慌性抛售
   - 每个人理性逃跑
   - 集体踩踏损害所有人

2. 追涨
   - 每个人理性跟风
   - 泡沫对所有人危险

3. 信息战
   - 每个人理性隐瞒
   - 市场效率下降

如何打破囚徒困境？
```
