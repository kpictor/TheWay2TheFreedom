# EP09 - 投资目标的设定

> 📅 第9期 | 第一季度：认识自我与市场

---

## 📌 本期核心问题

**"赚钱"不是目标——具体赚多少、多久赚到、承担什么风险？**

---

## 💡 第一性原理拆解

### SMART投资目标

| 原则 | 错误示范 | 正确示范 |
|------|----------|----------|
| **S**pecific 具体 | 我要赚钱 | 我要实现年化8%收益 |
| **M**easurable 可衡量 | 尽量多赚 | 3年后资产达到50万 |
| **A**chievable 可实现 | 一年翻倍 | 年化10%（长期可实现） |
| **R**elevant 相关性 | 跑赢隔壁老王 | 跑赢通胀+2% |
| **T**ime-bound 有时限 | 早晚会赚 | 3年/5年/10年目标 |

### 投资目标的三个层次

```
Layer 1: 生存目标
         → 不亏钱，至少跑赢通胀
         
Layer 2: 财务目标
         → 买房首付、子女教育、退休储备
         
Layer 3: 财务自由
         → 被动收入覆盖日常支出
```

## 🔍 Dalio原则应用

### 原则：目标先于方法

> "先搞清楚你想要什么，再研究如何得到它。"

**投资目标设定模板**：

```
我的投资目标：

短期（1年）：
- 金额目标：_______
- 收益目标：_______
- 最大可承受亏损：_______

中期（3年）：
- 金额目标：_______
- 这笔钱的用途：_______

长期（10年）：
- 期望达到的财务状态：_______
- 为什么这个目标对我重要：_______
```
