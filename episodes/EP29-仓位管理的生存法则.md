## 💡 第一性原理拆解

### 仓位管理的核心原则

> **先求不死，再求赢利。**

### 凯利公式简化版

```
最优仓位比例 = 胜率 - (败率/盈亏比)

例如：
胜率 = 55%
盈亏比 = 2（赚2赔1）
最优仓位 = 0.55 - (0.45/2) = 32.5%
```

### 仓位管理的基本规则

| 规则 | 内容 |
|------|------|
| 单笔上限 | 单笔投资不超过总资产的X% |
| 行业上限 | 单一行业不超过总资产的Y% |
| 最大回撤 | 设定最大可承受亏损 |
| 保留现金 | 始终保留一定现金比例 |
