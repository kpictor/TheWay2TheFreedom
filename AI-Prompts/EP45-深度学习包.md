## 🛡️ 第二步：黑天鹅预案设计

### 个人危机应对预案

```markdown
# 市场异常事件预案

## 触发条件
- 单日跌幅超过___%
- 单周跌幅超过___%
- 出现熔断/停市
- 重大地缘政治事件

## 第一反应（10分钟内）
□ 深呼吸，打开这份预案
□ 不做任何交易决策
□ 评估：这是短期冲击还是根本变化

## 评估清单
□ 我的持仓受实质影响吗？
□ 我的止损规则触发了吗？
□ 我的仓位在可承受范围内吗？
□ 我有足够的现金吗？

## 24小时内
□ 不追涨杀跌
□ 检查所有持仓的止损设置
□ 评估是否需要调整仓位
□ 记录自己的情绪状态

## 一周内
□ 复盘事件对持仓的影响
□ 评估是否有抄底机会
□ 更新投资手册（如需要）

## 禁止动作
□ 不恐慌全部清仓
□ 不加杠杆抄底
□ 不听信"内幕消息"
```

### Prompt B - 预案演练

```markdown
和我进行一次危机演练：

假设场景：明天早上开盘，你发现市场暴跌10%，你的持仓缩水15%。

请问我：
1. 你的第一反应是什么？
2. 根据你的预案，你应该做什么？
3. 你会检查哪些东西？
4. 你可能会犯什么错误？
5. 什么情况下你应该卖出？什么情况下应该持有？

帮我模拟一次正确的应对流程。
```

## 🧠 第三步：正确的思维框架

### Prompt C - 危机思维训练

```markdown
帮我建立面对市场危机的正确思维框架：

**问题1**：暴跌后应该立即行动吗？
**问题2**："这次不一样"的说法可信吗？
**问题3**：什么时候恐慌是正确的？

**问题5**：黑天鹅来临时，最重要的是什么？

请给我详细的思考框架。
```
