## 📚 模块一：概念深化

### Prompt 1.1 - 复利的数学本质

```markdown
请用数学帮我深入理解复利：

**基础公式解读**
FV = PV × (1 + r)^n

1. 为什么是指数增长而非线性增长？
2. n（时间）的影响力为什么大于r（收益率）？
3. 请用图表展示10年vs20年vs30年的差异

**复利三变量分析**
- 本金PV：增加50%的效果
- 收益率r：提高2%的效果
- 时间n：早10年开始的效果

哪个变量对普通人最可控？
```

### Prompt 1.2 - 复利的反直觉之处

```markdown
帮我理解复利中反直觉的地方：

**巴菲特财富时间线之谜**
- 30岁：100万
- 50岁：3.76亿
- 90岁：800亿
- 为什么99%的财富在50岁后获得？

**复利的"沉默期"**
- 前期增长缓慢，让人失去耐心
- 如何在"沉默期"保持信心？

**频繁交易对复利的破坏**
- 每次交易打断复利
- 税费如何侵蚀复利？
- 模拟：频繁交易vs持有不动的差异
```
