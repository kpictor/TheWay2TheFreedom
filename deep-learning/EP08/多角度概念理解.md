# EP08 - 多角度概念理解

> 风险容忍度的多维度解析

---

## 🎯 金融学视角

### 风险的正式定义

**学术定义**：
> 风险 = 未来结果的不确定性

**量化指标**：
| 指标 | 含义 | 适用场景 |
|------|------|----------|
| **标准差** | 收益率的波动幅度 | 整体波动 |
| **最大回撤** | 从最高点的最大跌幅 | 极端损失 |
| **VaR** | 一定概率下的最大损失 | 风险控制 |
| **夏普比率** | 风险调整后收益 | 比较基金 |

### 风险与收益的权衡

```
        高收益
          ▲
          │    股票（高风险高收益）
          │   /
          │  /  平衡型
          │ /
          │/  债券
          ├────────────────────▶ 高风险
        货币基金
       (低风险低收益)
```

**核心关系**：
> 长期来看，风险和收益正相关。
> 天下没有免费的午餐。

## 🧠 心理学视角

### 损失厌恶

**卡尼曼的前景理论**：

```
损失的痛苦 ≈ 2倍 × 同等收益的快乐

例如：
亏1000元的痛苦 = 赚2000元的快乐
```

**投资中的表现**：
- 赚钱时急于落袋为安
- 亏钱时死扛不愿止损
- 小利润满足，大损失麻木

### 过度自信

```
调查结果：
├── 90%的司机认为自己驾驶技术高于平均
├── 80%的投资者认为自己选股能力高于平均
└── 数学上只有50%的人能高于平均

你真的知道自己的风险承受能力吗？
```

### 情绪周期

```
市场上涨时：
"我风险承受能力很高" → 追涨

市场下跌时：
"我受不了了" → 割肉

实际行为 ≠ 测评时的回答
```

## 📊 行为金融学视角

### 客观能力 vs 主观意愿

| 维度 | 含义 | 决定因素 |
|------|------|----------|
| **客观能力** | 财务上能承受多少损失 | 收入、储蓄、负债、时间 |
| **主观意愿** | 心理上能承受多少波动 | 性格、经历、知识 |

**实际风险容忍度 = 两者的较小值**

**例子**：
```
小李：年收入50万，存款200万，无负债
      → 客观能力：高

但他：
      夜里股票跌2%就睡不着觉
      → 主观意愿：低

结论：他的实际风险容忍度 = 低
```

### 风险感知的扭曲

| 偏差 | 表现 | 后果 |
|------|------|------|
| 近因效应 | 牛市高估能力，熊市低估 | 高买低卖 |
| 控制错觉 | 以为自己能控制风险 | 过度冒险 |
| 频率偏差 | 频繁看账户增加痛苦 | 不必要的交易 |
| 框架效应 | 涨10%和跌10%感受不同 | 非理性决策 |

## ⏰ 生命周期视角

### 年龄与风险能力

```
风险承受能力
      ▲
      │       年轻时
      │      /
      │     /
      │    /
      │   /
      │  /     退休时
      │ /
      └──────────────────────▶ 年龄
        25   35   45   55   65
```

**为什么年轻人能承受更多风险**：
1. **时间长**：有时间从亏损中恢复
2. **人力资本大**：未来收入是安全垫
3. **责任轻**：通常无家庭负担
4. **可学习**：亏损是学费

**传统法则**：
> 股票占比 = 100 - 年龄
> （已经过时，现代建议更激进些）

## 🏛️ 投资组合理论视角

### 马科维茨的有效前沿

```
收益
  ▲       有效前沿
  │      ●●●●●●●●●
  │     ●         激进
  │    ●
  │   ●   稳健
  │  ●
  │ ●  保守
  │●
  └───────────────────────▶ 风险

在有效前沿上，没有免费的午餐：
更高收益必须接受更高风险
```

### 分散化的作用

```
分散化能做到：
├── 降低个股风险
├── 降低行业风险
└── 不能降低市场整体风险

即使完美分散，市场大跌时也会亏
所以：你必须知道自己能承受多少市场风险
```

## 💡 整合理解

### 核心原则

1. **测评不等于真实**：平时的测评和下跌时的反应可能完全不同
2. **客观主观取最小**：财务能承受≠心理能承受
3. **时间改变一切**：年龄、经历会改变风险偏好
4. **分散不是万能**：只能降低部分风险
5. **知道自己不行也是能力**：选择保守没什么丢人的

### 实用框架

```
确定风险容忍度四步法：

1. 客观评估：
   - 紧急备用金够吗？
   - 这笔钱多久不用？
   - 亏损50%会影响生活吗？

2. 主观测试：
   - 最近一次大跌你是什么反应？
   - 你能多久不看账户？
   - 亏多少会睡不着觉？

3. 取最小值：选择两者较低的

4. 预留buffer：再保守10-20%
```
