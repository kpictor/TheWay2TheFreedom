# EP19 - 美联储的心理游戏 | AI深度学习包

> 🎯 目标：学会解读央行政策信号，理解预期管理的威力

---

## 📚 模块一：概念深化

### Prompt 1.1 - 央行的双重工具

```markdown
帮我理解央行影响市场的两类工具：

**实际行动**
- 调整利率
- 量化宽松/紧缩
- 准备金率
- 这些行动如何传导到市场？

**预期管理**
- 前瞻指引
- 措辞变化
- 官员讲话
- 为什么"说"有时比"做"更有效？

**两者的关系**
- 什么时候用行动？什么时候用预期管理？
- 预期管理失效时会发生什么？
- 历史上有哪些"说话就够了"的例子？
```

### Prompt 1.2 - 美联储措辞解读

```markdown
教我解读美联储声明的措辞：

**关键词汇表**
- "耐心" (patient) → ?
- "密切关注" (closely monitoring) → ?
- "取决于数据" (data-dependent) → ?
- "适当" (appropriate) → ?
- "坚定承诺" (firmly committed) → ?
- "暂时性" (transitory) → ?

**措辞变化的意义**
- 从"耐心"到"密切关注"意味着什么？
- 删除某个词比添加某个词更重要？
- 如何追踪措辞变化？

**实操建议**
- 我应该关注FOMC的哪些内容？
- 如何建立自己的"央行观察"系统？
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 央行政策工具详解

```markdown
详细介绍央行的各种政策工具：

**常规工具**
- 联邦基金利率：如何操作？如何传导？
- 贴现率：什么情况下使用？

**非常规工具**
- QE（量化宽松）：具体怎么操作？买什么？
- 扭曲操作：目的是什么？
- 负利率：日本和欧洲的经验
- 收益率曲线控制：如何运作？

每种工具对不同资产的影响是什么？
```

### Prompt 2.2 - 历史政策转向案例

```markdown
分析历史上重要的美联储政策转向：

**2008-2009年**：零利率+QE的开始
- 市场反应
- 教训

**2013年**："Taper Tantrum"（缩减恐慌）
- 伯南克说了什么
- 为什么市场暴跌
- 后来如何修复

**2018年**：鲍威尔的"Powell Pivot"
- 发生了什么
- 为什么转向
- 对投资者的启示

**2021-2022年**：从"暂时性通胀"到激进加息
- 预期管理失误
- 市场冲击

从这些案例中我能学到什么？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 央行声明解读练习

```markdown
请给我一份最近的美联储/央行声明，让我练习解读：

分析框架：
1. 与上一次声明相比，有哪些措辞变化？
2. 这些变化暗示什么政策方向？
3. 市场预期是什么？实际声明符合预期吗？
4. 如果有超预期的部分，是鸽派还是鹰派？
5. 这对不同资产意味着什么？

我会尝试解读，请给我反馈。
```

### Prompt 3.2 - 政策模拟游戏

```markdown
和我玩一个"你是美联储主席"的模拟游戏：

设定：
- 当前通胀：X%
- 失业率：Y%
- 经济增长：Z%
- 市场预期：加息/降息/维持

问我：
1. 你会做出什么政策决定？
2. 你会如何措辞？
3. 你预期市场如何反应？
4. 有什么风险？

帮我理解央行决策的复杂性。
```
