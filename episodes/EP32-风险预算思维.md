---

## 💡 第一性原理拆解

### 风险预算的核心思想

```
传统思维：我想赚多少 → 决定买入多少
风险思维：我能亏多少 → 决定买入多少
```

### 风险预算计算

```
单笔最大风险 = 总资产 × 单笔风险比例

例如：
总资产 = 100万
单笔风险 = 1%
单笔最大亏损 = 1万

如果止损位是10%：
最大仓位 = 1万 / 10% = 10万
```
