## 🛠️ 第二步：错误分析实践

### Prompt A - 引导式错误分析

```markdown
请引导我进行一次深度错误分析。

我会描述一次投资错误，请逐步问我：
1. 发生了什么？（事实）
2. 你当时在想什么？（心理）
3. 为什么做这个决定？（第一个Why）
4. 为什么会这样想？（第二个Why）
5. ...继续追问直到根因

每次只问一个问题，等我回答后再问下一个。
最后帮我总结根因和教训。
```

### Prompt B - 教训转化为原则

```markdown
我从错误中得到了以下教训：
[粘贴教训]

请帮我：
1. 把这个教训这个表达为一条清晰的原则
2. 检验原则是否可操作
3. 建议放入原则清单的哪个位置
4. 如何验证这条原则有效

输出可以直接使用的原则表述。
```

