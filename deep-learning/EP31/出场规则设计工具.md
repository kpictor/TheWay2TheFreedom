# EP31 - 出场规则设计工具

> 设计止损、止盈和卖出决策流程

---

## 🎯 使用说明

本工具帮助你：
1. 设计完整的出场规则
2. 计算止损止盈价格
3. 建立卖出决策流程

## 📋 工具二：止损价计算器

### 各类止损方法计算

```
止损价计算器
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【固定比例止损】

买入价：___________元
止损比例：____%

止损价 = 买入价 × (1 - 止损比例)
       = _______ × (1 - ____%)
       = ___________元

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【移动止损】

买入价：___________元
持有期间最高价：___________元
移动止损比例：____%

止损价 = 最高价 × (1 - 移动止损比例)
       = _______ × (1 - ____%)
       = ___________元

如果触发：
盈亏 = 止损价 - 买入价
     = _______ - _______
     = ___________元（盈利/亏损____%）

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【ATR止损】

当前价格：___________元
ATR（平均真实波幅）：___________
N倍数：____

止损价 = 当前价 - N × ATR
       = _______ - ____ × _______
       = ___________元

止损距离占价格比例：____%
```

## 📋 工具五：止损执行心理准备

### 突破止损心理障碍

```
止损心理准备清单
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【重新定义止损】

旧定义：止损 = 认输 = 失败
新定义：止损 = 执行规则 = 专业

我承诺的新信念：
"止损意味着我在保护本金"
"止损是专业投资者的标志"
"不止损才是真正的失败"

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【止损前的自问】

当止损价接近时问自己：

"如果我现在没有持仓，
 会在当前价格买入吗？"

○如果答案是"会"→ 可以考虑继续持有
○如果答案是"不会"→ 应该执行止损

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【执行止损时的提醒】

当我犹豫是否止损时，念出：

"规则在情绪冷静时制定，
 现在应该遵守规则。"

"保留90%的本金，
 比可能的回本更重要。"

"这只是一次普通的规则执行，
 不是对我能力的否定。"

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【止损后的反思】

执行止损后填写：

这次止损是规则执行，我做得对 □确认

需要反思的是买入决策，而非止损决策

买入决策的反思：
_________________________________
```
