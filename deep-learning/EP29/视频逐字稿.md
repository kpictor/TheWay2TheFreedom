## 第一部分：仓位管理的本质（1:30-5:00）

### 投资与赌博的数学相似性

假设你有一个正期望值的游戏：
- 50%概率赚20%
- 50%概率亏10%

**问题**：你应该下注多少？

```
情况1：每次全仓（100%）
赢一次：100 → 120
输一次：120 → 108
赢一次：108 → 129.6
输一次：129.6 → 116.6
继续下去...波动巨大，可能破产

情况2：每次半仓（50%）
波动更小，更平稳增长
长期来看效果更好！
```

**关键洞见**：
> 即使是正期望值的游戏，如果下注太大也会破产。

### 生存第一原则

```
巴菲特的两条规则：
规则1：永远不要亏钱
规则2：永远不要忘记规则1

翻译成仓位管理语言：
规则1：控制单笔风险，确保能活下来
规则2：永远不要忘记规则1
```

### 为什么仓位比选股更重要

**一个反直觉的事实**：

```
投资者A：选股准确率60%，每次下注50%
投资者B：选股准确率55%，每次下注20%

谁的长期表现更好？

答案：很可能是B
因为A会在那40%的失败中亏损更多
波动更大，情绪影响更大，更难坚持
```

## 第二部分：凯利公式（5:00-9:00）

### 凯利公式介绍

1956年，贝尔实验室的约翰·凯利发表论文，推导出最优下注比例公式。

**凯利公式**：
```
f* = p - q/b

其中：
f* = 最优下注比例
p = 获胜概率
q = 失败概率（1-p）
b = 盈亏比（赚时收益/亏时损失）
```

### 计算示例

**例1**：
- 胜率 = 55%
- 盈亏比 = 2（赚2亏1）

```
f* = 0.55 - 0.45/2
f* = 0.55 - 0.225
f* = 0.325 = 32.5%

建议仓位：32.5%
```

**例2**：
- 胜率 = 60%
- 盈亏比 = 1.5

```
f* = 0.60 - 0.40/1.5
f* = 0.60 - 0.267
f* = 0.333 = 33.3%

建议仓位：33.3%
```

**例3**：
- 胜率 = 50%
- 盈亏比 = 1

```
f* = 0.50 - 0.50/1
f* = 0.50 - 0.50
f* = 0

不值得下注！
```

### 凯利公式的应用注意

**实际应用通常使用半凯利或更保守**

| 使用比例 | 波动性 | 收益 | 适合 |
|---------|--------|------|------|
| 全凯利 | 很高 | 最大化 | 高风险偏好 |
| 半凯利 | 中等 | 次优 | 大多数人 |
| 1/4凯利 | 较低 | 稳健 | 保守型 |

**原因**：
- 你估计的胜率可能过于乐观
- 盈亏比可能被高估
- 实际中有黑天鹅
- 心理上难以承受大波动

## 第三部分：实用仓位规则（9:00-12:30）

### 规则1：单笔风险上限

**2%规则**：单笔交易的最大亏损不超过账户的2%

```
例子：
账户：100万
单笔最大亏损：2万

如果止损设在10%：
最大仓位 = 2万 ÷ 10% = 20万
即最多只能买20%仓位
```

**计算公式**：
```
仓位上限 = (账户 × 风险比例) ÷ 止损幅度

例：
仓位上限 = (100万 × 2%) ÷ 10%
         = 2万 ÷ 10%
         = 20万
```

### 规则2：单只股票上限

```
保守型：单只不超过5%
稳健型：单只不超过10%
积极型：单只不超过20%

无论多么看好，不单押一只
```

### 规则3：行业/主题上限

```
单一行业不超过30%
防止行业性系统风险
```

### 规则4：保留现金

```
始终保留一定现金：
- 市场高估时：30%以上现金
- 正常时期：10-20%现金
- 市场低估时：可降至5%

现金的作用：
1. 应对机会
2. 应对紧急需求
3. 心理安全感
```

### 规则5：分批建仓

```
不要一次性买入全部仓位

分批方式示例：
第1批：目标仓位的30%
确认后加第2批：再加30%
再确认后第3批：最后40%

好处：
- 平均成本
- 降低择时错误影响
- 更好的心理控制
```

**核心原则**：
- 先求不死，再求赢利
- 仓位比选股更重要

**实用规则**：
1. 单笔风险不超过2%
