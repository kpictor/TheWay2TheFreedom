## 📚 模块一：概念深化

### Prompt 1.1 - 两个大脑系统

```markdown
帮我深入理解大脑的两个决策系统：

**系统1（快思考/杏仁核主导）**
- 演化目的是什么？
- 在什么情况下激活？
- 反应速度和特点？
- 在投资中如何表现？

**系统2（慢思考/前额叶主导）**
- 功能是什么？
- 需要什么资源？
- 什么情况下会"离线"？
- 在投资中如何表现？

**两系统冲突**
- 为什么系统1常常胜出？
- 如何让系统2更常主导？
```

### Prompt 1.2 - 杏仁核劫持详解

```markdown
解释"杏仁核劫持"现象：

**生理过程**
1. 感知到威胁（账户大跌）
2. 杏仁核激活
3. 肾上腺素、皮质醇释放
4. 前额叶功能受抑制
5. 本能反应：战斗或逃跑

**在投资中的表现**
- 恐慌性抛售
- 无法进行理性分析
- 事后后悔但当时无法控制

**识别信号**
- 心跳加速
- 出汗
- 视野狭窄
- 只能想到"逃跑"

请教我如何在被"劫持"时恢复理性。
```
