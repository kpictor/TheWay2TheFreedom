# EP15 - 对抗本能的实用技巧

> 让理性在关键时刻战胜情绪的操作手册

---

## 🎯 核心原则

> "你无法消除情绪，但可以设计系统来绕过情绪决策。"

---

## 📋 技巧1：24小时冷静规则

### 原理
杏仁核劫持通常在30分钟到几小时内消退。强制等待给前额叶时间"重新上线"。

### 执行方法

```
触发条件：
当以下情况发生时，启动24小时冷静期：
□ 单次交易金额超过账户 10%
□ 账户一天波动超过 5%
□ 你感到"必须马上行动"
□ 你感到极度恐惧或兴奋

执行步骤：
1. 记录下冲动（想做什么，为什么）
2. 设定24小时后的提醒
3. 关闭交易软件
4. 做其他事情转移注意力
5. 24小时后重新评估

评估问题：
□ 我还想做同样的事吗？
□ 冷静下来后，理由还成立吗？
□ 这符合我的投资规则吗？
```

## 📋 技巧2：预设规则法

### 原理
在冷静时制定规则，情绪时按规则执行，不做任何决策。

### 构建你的规则系统

**止损规则**（预设，非临场决定）
```
股票止损规则：
单只股票亏损达到 ____% 时，必须止损
条件：_________________________________

仓位止损规则：
总账户回撤达到 ____% 时，清仓休息
恢复条件：___________________________
```

**买入规则**（预设触发条件）
```
允许买入的条件（必须全部满足）：
□ 符合我的选股标准
□ 价格在我设定的买入区间
□ 仓位不会超过单只 _____%
□ 情绪检查通过（平静状态）
```

**关键**：规则写下来，打印出来，放在交易时能看到的地方。

## 📋 技巧3：身体觉察法

### 原理
情绪有身体先兆。学会识别身体信号，在情绪爆发前介入。

### 身体信号清单

```
恐惧的身体信号：
☐ 心跳加速
☐ 手心出汗
☐ 胃部紧缩
☐ 肩膀紧张
☐ 呼吸变浅变快
☐ 想上厕所

贪婪的身体信号：
☐ 坐立不安
☐ 频繁查看手机
☐ 呼吸急促
☐ 身体前倾
☐ 手指不自觉点击
☐ 说话语速变快
```

**当你注意到这些信号时**：
1. 暂停（不要继续当前动作）
2. 深呼吸3次（激活副交感神经）
3. 离开屏幕5分钟
4. 问自己：我现在是系统1还是系统2在主导？

## 📋 技巧4：第三人称思考

### 原理
把自己当作别人来观察，可以激活更理性的脑区。

### 操作方法

当你想做一笔交易时，问自己：

```
"如果我的朋友在这种情况下想这样做，我会给他什么建议？"

场景填写：
我的朋友遇到了这种情况：_____________________
他想做这个决定：_____________________
作为旁观者，我会说：_____________________

这个建议适用于我自己吗？□是 □否
```

## 📋 技巧5：环境设计

### 原理
不依赖意志力，通过改变环境来减少情绪触发。

### 环境检查清单

```
减少恐惧触发：
□ 关闭实时行情推送
□ 不看股评/论坛炒股区
□ 设置账户查看频率（每周1次）
□ 大跌日不打开交易软件

减少贪婪触发：
□ 取消关注"炒股大V"
□ 不参与股票群讨论
□ 屏蔽朋友圈晒收益
□ 不看"暴涨"新闻

物理隔离：
□ 交易密码设置复杂（增加操作摩擦）
□ 手机上删除交易APP（需要时重新下载）
□ 把可投资资金放在需要T+1才能用的账户
```

## 📋 技巧6：情绪急救包

### 当情绪失控时的紧急处理

```
【紧急情况处理流程】

Step 1: 识别
"现在我是情绪状态吗？"
检查身体信号 → 如果有2个以上信号，进入Step 2

Step 2: 暂停
立即停止当前操作
关闭交易软件
离开电脑/放下手机

Step 3: 平复
深呼吸10次（吸4秒-屏2秒-呼6秒）
喝杯水
走动5分钟

Step 4: 记录
写下：我刚才想做什么？为什么？
写下：我的身体信号是什么？
写下：冷静后我怎么看这件事？

Step 5: 等待
设定24小时后的提醒
在此之前不做任何交易决策
```
