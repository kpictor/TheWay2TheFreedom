## 📚 模块一：概念深化

### Prompt 1.1 - 经济机器模型拆解

```markdown
请帮我深入理解Ray Dalio的"经济机器"模型：

1. **生产力增长**：什么驱动生产力？为什么它是长期稳定的？
2. **短期债务周期**：5-8年周期的驱动力是什么？
3. **长期债务周期**：75-100年周期的特征是什么？
4. **三者叠加**：如何形成我们看到的经济波动？

请用简单的比喻（比如波浪叠加）帮我理解。
```

### Prompt 1.2 - 货币与价格的关系

```markdown
帮我理解"资产价格 = 货币量 ÷ 资产量"这个公式：

1. 为什么央行印钱会推高股价？
2. "通胀是隐形税"怎么理解？
3. 日本央行大量印钱为什么没有通胀？
4. 中国M2增长和房价的关系如何解读？
5. 美联储缩表对股市意味着什么？

请用实际数据和案例说明。
```
