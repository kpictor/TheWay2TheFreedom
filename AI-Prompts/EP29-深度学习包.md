### Prompt 1.1 - 仓位管理的核心原则

```markdown
帮我理解仓位管理的核心智慧：

**核心原则**
> "先求不死，再求胜利"

**为什么仓位比选股更重要？**
1. 选错股，仓位小 → 损失可控
2. 选对股，仓位大 → 收益可观
3. 选错股，仓位大 → 可能致命

**反直觉的真相**
- 即使胜率60%，连亏5次的概率约1%
- 如果每次全仓，连亏5次就清零
- 合理仓位是生存的前提

给出具体数字例子说明仓位的重要性。
```

### Prompt 1.2 - 凯利公式简化版

```markdown
教我使用凯利公式：

**完整公式**
f* = (bp - q) / b
- f*: 最优仓位比例
- b: 盈亏比（赢多少/输多少）
- p: 胜率
- q: 败率（1-p）

**简化版**
f* = 胜率 - (败率/盈亏比)

**计算示例**
假设：胜率55%，盈亏比2（赚2赔1）
f* = 0.55 - (0.45/2) = 0.55 - 0.225 = 32.5%

**实践调整**
- 凯利公式结果通常过于激进
- 实际使用半凯利或更保守
- 为什么要保守？

帮我用自己的数据计算。
```
