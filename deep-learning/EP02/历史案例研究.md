# EP02 - 历史案例研究

> 3个"大多数人亏钱"的历史案例

---

## 📈 案例1：2000年互联网泡沫

### 背景

**时间**：1995-2000年

**狂热表现**：
- 纳斯达克从1000点涨到5000点
- 任何带".com"的公司都受追捧
- 市盈率动辄100倍以上
- "新经济"论调盛行

### 大多数人的行为

| 阶段 | 大多数人的行为 | 结果 |
|------|----------------|------|
| 1997-1998 | 观望，觉得太贵 | 错过第一波 |
| 1999 | 看到别人赚钱，开始跟风 | 高位入场 |
| 2000.3 | 顶部全仓，信心满满 | 最危险时刻 |
| 2000-2002 | 越跌越抄，深套 | 亏损70-90% |

### 数据说明

- 纳斯达克峰值5048点（2000.3.10）
- 底部1114点（2002.10）
- 跌幅78%
- 直到2015年才创新高（15年）

### 教训

1. **群体狂热时保持清醒**
2. **新概念不等于新价值**
3. **估值终究会回归**
4. **追高的代价是时间和金钱**

## 📉 案例2：2015年A股股灾

### 背景

**时间**：2014.7 - 2015.6

**狂热表现**：
- 上证从2000点涨到5178点
- "国家牛市"论调盛行
- 杠杆资金大量入市
- 全民炒股，开户创纪录

### 大多数人的行为

| 阶段 | 大多数人的行为 | 心理 |
|------|----------------|------|
| 2014下半年 | 观望不信 | "反弹而已" |
| 2015.1-3 | 开始入场 | "可能是牛市" |
| 2015.4-5 | 加杠杆满仓 | "只会涨不会跌" |
| 2015.6 | 顶部追高 | "4000点是起点" |
| 2015.6-8 | 恐慌抛售 | "完了完了" |

### 关键数据

- 最高点5178点（2015.6.12）
- 18个交易日跌去32%
- 配资爆仓连环踩踏
- 千股跌停反复出现

### 散户亏损结构

| 入场时点 | 占比 | 平均亏损 |
|----------|------|----------|
| 4000点以上 | 40% | 50-80% |
| 3000-4000 | 30% | 20-50% |
| 3000以下 | 30% | 0-20% |

### 教训

1. **杠杆是双刃剑**
2. **政策/媒体不是投资依据**
3. **群体狂热是危险信号**
4. **仓位管理比选股重要**

## 💥 案例3：比特币2017-2018周期

### 背景

**时间**：2017-2018

**狂热表现**：
- 比特币从$1000涨到$20000
- ICO遍地开花
- "区块链革命"论调
- 普通人通过各种渠道入场

### 大多数人的行为

| 阶段 | 大多数人的行为 | 典型心理 |
|------|----------------|----------|
| 2017.1-6 | 不了解，观望 | "骗局吧" |
| 2017.9-11 | 开始关注 | "好像真能赚钱" |
| 2017.12 | 顶部入场 | "怕错过" |
| 2018.1-3 | 补仓抄底 | "跌这么多该涨了" |
| 2018.12 | 割肉离场 | "再也不碰了" |

### 关键数据

- 最高点约$19,800（2017.12.17）
- 最低点约$3,200（2018.12）
- 跌幅84%
- 大量ICO代币归零

### 散户行为模式

```
价格曲线：  /\
           /  \
          /    \
         /      \_____

散户情绪：
         恐惧 → 犹豫 → FOMO → 狂热 → 恐惧 → 绝望
         │      │       │       │       │
         不买   小买    重仓    顶部    割肉
                              入场    出场
```

### 教训

1. **FOMO（错失恐惧）是大敌**
2. **新资产类别波动更大**
3. **群体狂热是卖出信号**
4. **不懂的东西不要重仓**

## 💡 三个案例的共性

| 特征 | 互联网泡沫 | A股股灾 | 比特币周期 |
|------|------------|---------|------------|
| 新叙事 | 新经济 | 国家牛市 | 区块链革命 |
| 群体狂热 | ✅ | ✅ | ✅ |
| 杠杆放大 | ✅ | ✅ | ✅ |
| 大众入场时点 | 顶部 | 顶部 | 顶部 |
| 跌幅 | 78% | 50%+ | 84% |

### 核心规律

> **大多数人在顶部最乐观，在底部最绝望。**
> **因此，大多数人高买低卖。**

3. 什么信号可以帮你在群体狂热时保持清醒？
