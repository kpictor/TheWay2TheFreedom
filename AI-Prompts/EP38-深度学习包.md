## 🔧 第二步：填写辅助

### Prompt A - 回顾Q1-Q3成果

```markdown
帮我整理Q1-Q3学习中产生的所有要素，以便填入投资手册：

Q1产出：
- 财务状况表（EP07）
- 风险偏好档案（EP08）
- 投资目标（EP09）

Q2产出：
- 我的认知偏差清单（EP16）
- 情绪日记模板（EP12）

Q3产出：
- 资产配置方案（EP28）
- 入场规则（EP30）
- 出场规则（EP31）
- 风险预算表（EP32）

请帮我逐项核对，确保手册包含所有这些要素。
```

### Prompt B - 手册评审

```markdown
我完成了投资手册v1.0的初稿：
[粘贴你的手册]

请帮我评审：
1. 完整性：有没有缺失的关键要素？
2. 明确性：规则是否足够具体可执行？
3. 一致性：各规则之间是否矛盾？
4. 可操作：你（另一个人）能按这个手册操作吗？

给出具体的改进建议。
```
