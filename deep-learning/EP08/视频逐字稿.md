# EP08 - 视频逐字稿

> 《风险容忍度自测》教学视频脚本

---

**【画面】** 过山车画面，有人尖叫有人享受

**【画面】** 切换到股市K线图

**【旁白】**
> 今天，我们来做一次自我测试：你的风险容忍度是多少？

---

## 📍 第一部分：什么是风险容忍度（0:30-2:30）

**【画面】** 两个维度的图

**【旁白】**
> 风险容忍度有两个维度。

**【动画】**
```
维度一：客观能力
= 你的财务状况允许你承受多大损失

决定因素：
├── 收入稳定性
├── 储蓄水平
├── 投资期限
└── 负债情况

维度二：主观意愿
= 你的心理能承受多大波动

决定因素：
├── 性格特点
├── 过往经历
├── 投资知识
└── 情绪稳定性
```

**【旁白】**
> 关键来了：你真正的风险容忍度，等于两者中较小的那个。

**【举例】**

```
小王：
  月入5万，存款200万 → 客观能力：高
  但股票跌5%就焦虑失眠 → 主观意愿：低
  
结论：小王的风险容忍度 = 低

这意味着即使他财务上能承受，心理上也会崩溃。
```

---

## 📍 第二部分：为什么测评不准（2:30-5:00）

**【画面】** 问卷调查

**【旁白】**
> 很多人做过风险测评问卷。
>
> 但问题是：测评结果和实际行为经常不一致。

**【动画场景】**

```
牛市时填问卷：
"亏损30%我也能接受！" → 测评结果：激进型

熊市来了：
账户-15% → "受不了了，全卖！" → 实际行为：保守型
```

**【旁白】**
> 为什么会这样？因为"想象中的亏损"和"真实的亏损"感受完全不同。

**【心理学解释】**

```
卡尼曼的发现：

亏损的痛苦 ≈ 2倍 × 同等收益的快乐

例如：
  亏1万的痛苦 = 赚2万的快乐
  
这叫"损失厌恶"
```

---

## 📍 第三部分：如何真正了解自己（5:00-8:00）

**【画面】** 工具箱

**【旁白】**
> 既然问卷不准，怎样才能真正了解自己的风险容忍度？

### 方法1：回忆历史

**【旁白】**
> 最好的测试是过去的真实经历。
>
> 回忆一下：上一次市场大跌，你是什么反应？

**【问题清单】**
```
1. 你是恐慌卖出，还是冷静持有？
2. 你晚上睡得着吗？
3. 你忍不住反复查看账户吗？
4. 这次经历让你变得更保守还是更激进？
```

### 方法2：睡眠测试

**【旁白】**
> 一个简单但有效的问题：
>
> 亏损多少会让你睡不着觉？

**【动画】**
```
如果答案是：
  5% → 你是保守型
  10% → 你是稳健型
  20% → 你是积极型
  不影响睡眠 → 你是激进型（或者没有真金白银投入）
```

### 方法3：小额试验

**【旁白】**
> 如果没有投资经历，可以做一个小额试验。
>
> 用少量资金体验一下波动的感受。

**【建议】**
```
1. 投入可承受损失的少量资金
2. 选择波动较大的资产
3. 观察自己的情绪和行为
4. 据此调整正式投资的配置
```

---

## 📍 第四部分：匹配与实践（8:00-9:30）

**【画面】** 配置图

**【旁白】**
> 知道了风险容忍度，接下来是匹配合适的配置。

**【配置建议表】**

| 风险等级 | 股票 | 债券 |
|----------|------|------|
| 保守型 | 20% | 80% |
| 稳健型 | 40% | 60% |
| 积极型 | 60% | 40% |
| 激进型 | 80% | 20% |

**【旁白】**
> 记住：宁可保守，不可激进。
>
> 因为一次情绪崩溃做出的错误决策，可能毁掉多年的积累。

---

## 📍 总结（9:30-10:00）

**【画面】** 镜子中的自己

**【旁白】**
> 了解自己的风险容忍度，比学任何投资技术都重要。
>
> 因为最大的敌人不是市场，是你自己的情绪。

**【三个要点】**
```
1. 客观能力和主观意愿取较小值
2. 实际行为比测评问卷更真实
3. 宁可保守，这不是认输
```

**【结束语】**
> 下一期，我们来设定投资目标。
>
> 知道自己能承受多少风险后，才能谈该追求多少收益。
