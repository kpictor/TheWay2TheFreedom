# EP09 - 投资目标SMART设定工具箱

> 用科学方法设定真正可执行的投资目标

---

## 🎯 用前须知

**核心理念**：
> "没有目标的投资，就像没有目的地的航行——你永远不知道是否到达。"

**本工具用途**：
- 设定具体、可衡量的投资目标
- 区分短期、中期、长期目标
- 评估目标的可实现性
- 建立目标追踪机制

**建议**：每年至少审视一次目标

---

## 📋 工具1：SMART目标检验表

### 目标质量自检

将你的投资目标填入，用SMART原则逐项检验：

| 原则 | 检验问题 | 你的目标内容 | 通过？ |
|------|----------|-------------|--------|
| **S**pecific 具体 | 目标是否足够具体明确？ | | □是 □否 |
| **M**easurable 可衡量 | 能用数字衡量吗？ | | □是 □否 |
| **A**chievable 可实现 | 基于现实可能达到吗？ | | □是 □否 |
| **R**elevant 相关 | 与人生目标相关吗？ | | □是 □否 |
| **T**ime-bound 有时限 | 有明确的截止日期吗？ | | □是 □否 |

**评估**：5个"是" = 优秀目标 | 4个 = 需微调 | ≤3个 = 需重新设定

## 📋 工具2：目标转化器

### 从模糊到清晰

| 模糊目标 | 问题 | SMART版本 |
|----------|------|-----------|
| "我要赚钱" | 赚多少？什么时候？ | 3年后投资账户达到50万 |
| "我要财务自由" | 什么是财务自由？ | 10年后被动收入达到月支出的150% |
| "我要跑赢大盘" | 什么大盘？多少期限？ | 未来5年年化收益超过沪深300+2% |
| "尽量多赚" | 无法衡量 | 年化收益目标8%，风险控制在最大回撤20%内 |

## 💡 使用建议

1. **先从小目标开始**：第一个目标不求大，求能实现
2. **写下来**：写下来的目标实现概率增加42%
3. **告诉别人**：公开承诺增加执行力
4. **定期检查**：未检查的目标等于没有目标
5. **灵活调整**：目标可以调整，但要有依据

**黄金法则**：
> 宁可设定保守目标并超额完成，也不要设定激进目标后失望放弃。
