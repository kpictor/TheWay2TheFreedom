## 开场（0:00-1:30）

大家好，欢迎来到《通往自由之路》第35期。

今天我们进入一个激动人心的话题：**AI在投资中的应用**。

随着ChatGPT等大语言模型的出现，AI能力有了质的飞跃。但很多人用错了方式：
- "AI，告诉我明天买什么股票？"
- "AI，这只股票会涨吗？"

**错误的期望会导致失望和损失。**

正确的思路是：
> **让AI做信息处理，你做判断决策。**

今天，我们来学习如何正确使用AI辅助投资。

---

## 第一部分：AI能做什么和不能做什么（1:30-5:00）

### AI的能力边界

**AI擅长的事情**：

| 能力 | 投资应用 |
|------|---------|
| 信息整理 | 归纳研报要点、整理财务数据 |
| 概念解释 | 解释投资术语、金融概念 |
| 逻辑分析 | 分析公司商业模式、竞争格局 |
| 文本处理 | 总结新闻、分析公告 |
| 多角度思考 | 列出利多/利空因素 |
| 对话辅助 | 作为思维对手盘挑战你 |

**AI不能做的事情**：

| 不能做 | 原因 |
|--------|------|
| 预测价格 | 市场本质不可预测 |
| 保证赚钱 | 没有任何方法能保证 |
| 替你决策 | 决策责任在你 |
| 提供内幕 | 违法且不可能 |
| 管理情绪 | 这是你自己的功课 |

### 正确心态

```
AI是工具，不是神谕
AI是助手，不是决策者
AI提供信息，你做判断
```

## 第二部分：AI的实用场景（5:00-10:00）

### 场景1：研报分析

**提示词示例**：
```
请帮我分析这份研报：

1. 核心观点是什么？（3句话总结）
2. 支撑观点的关键假设是什么？
3. 如果这些假设不成立，结论会如何变化？
4. 研报没有提到但你认为重要的风险是什么？

[粘贴研报内容]
```

### 场景2：公司分析

**提示词示例**：
```
请帮我分析[公司名]：

1. 用3句话描述它的核心业务
2. 它的竞争优势（护城河）是什么？
3. 最大的风险因素是什么？
4. 这个行业的关键成功因素是什么？

请用简单的语言，假设我是投资新手。
```

### 场景3：新闻解读

**提示词示例**：
```
请帮我解读这条新闻：

1. 这是新信息还是已知信息？
2. 这对市场是利好还是利空？为什么？
3. 哪些公司/行业可能受影响？
4. 我需要立即采取行动吗？还是只是噪音？

[粘贴新闻内容]
```

### 场景4：概念学习

**提示词示例**：
```
请用费曼技巧解释[金融概念]：

1. 用最简单的语言解释核心定义
2. 给一个日常生活的类比
3. 在投资中如何应用这个概念？
4. 常见的误解是什么？
```

### 场景5：思维对手盘

**这是最重要的应用**

```
我正在考虑买入[公司名]：

我的理由是：
1. [理由1]
2. [理由2]
3. [理由3]

请扮演做空者，尽可能挑战我的观点：
1. 我的逻辑哪里可能有问题？
2. 我可能忽略了什么风险？
3. 如果我错了，最可能是因为什么？
4. 有什么证据能推翻我的判断？
```

## 第三部分：AI使用的注意事项（10:00-13:00）

### 注意1：验证信息准确性

```
AI可能会"幻觉"：
- 编造不存在的数据
- 给出过时的信息
- 信息是基于训练数据，可能不是最新的

应对方法：
- 重要数据去官方来源核实
- 投资决策前交叉验证
- 不完全依赖AI的事实性陈述
```

### 注意2：警惕过度自信

```
AI给出的分析可能看起来很专业
可能让你产生"已经分析透彻"的错觉

应对方法：
- 记住AI不承担后果，你承担
- 多问"还有什么我没考虑到的"
- 永远保持怀疑和谦虚
```

### 注意3：不要期待预测

```
错误问法：
"这只股票会涨吗？"
"应该买入还是卖出？"
"什么时候是底部？"

正确问法：
"帮我分析这家公司的风险因素"
"这个估值水平在历史上处于什么位置"
"如果经济衰退，这个行业会如何受影响"
```

### 注意4：保护隐私

```
注意不要向AI透露：
- 完整的账户信息
- 具体的资金规模
- 个人敏感信息

可以说"一笔中等规模的投资"
而不是"我准备投入100万"
```

## 第四部分：建立你的AI工作流（13:00-14:30）

### 投资分析AI工作流

```
发现一个投资机会
    ↓
第一步：用AI了解基本面
"帮我介绍这家公司的核心业务和竞争格局"
    ↓
第二步：用AI分析财务数据
"帮我解读这些财务数据的意义"
    ↓
第三步：用AI找反方观点
"扮演做空者，挑战我的看法"
    ↓
第四步：你做最终判断
AI只提供输入，决策是你的
    ↓
按你的系统规则执行
```

### 每周AI辅助流程

```
每周一次：
1. 让AI总结持仓相关的重要新闻
2. 让AI挑战你当前的持仓逻辑
3. 让AI帮你思考潜在的风险

记住：
这是辅助思考，不是外包决策
```
