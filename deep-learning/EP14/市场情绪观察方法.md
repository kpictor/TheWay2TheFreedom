# EP14 - 市场情绪观察方法

> 实用工具：如何读懂市场的集体心理

---

## 🎯 核心理念

> "价格是投票机，反映的是所有人心理的合力。学会观察情绪，就是学会站在市场外面看市场。"

---

## 📋 工具1：市场情绪温度计

### 日常观察框架

每周记录一次，追踪市场情绪变化：

```

【量化指标】

【定性观察】
媒体氛围：□极度乐观 □乐观 □中性 □悲观 □极度悲观

□ 正常讨论

```

## 📋 工具2：情绪周期定位器

### 你现在在哪个阶段？

```
市场情绪周期（选择当前最接近的阶段）：

□ 怀疑期
  - 特征：市场刚经历大跌，多数人仍悲观
  - 行为：成交量低迷，少量抄底者试探
  - 机会：可能是长期投资的好时机

□ 乐观期
  - 特征：市场温和上涨，信心逐渐恢复
  - 行为：媒体开始正面报道，资金流入
  - 机会：趋势确认，可以参与

□ 兴奋期
  - 特征：市场持续上涨，赚钱效应明显
  - 行为：新投资者入场，讨论热度高
  - 风险：需要开始警惕，控制仓位

□ 狂热期
  - 特征：人人谈股票，"这次不一样"
  - 行为：高杠杆，追热点，忽视估值
  - 风险：高风险区域，考虑减仓

□ 否认期
  - 特征：市场开始下跌，多数人认为只是回调
  - 行为：抄底者涌入，但越跌越多
  - 风险：趋势可能已经转变

□ 恐惧期
  - 特征：下跌加速，开始担心本金安全
  - 行为：减仓、割肉增多
  - 机会：接近底部区域的信号

□ 恐慌期
  - 特征：暴跌，恐惧弥漫，无人敢买
  - 行为：不计成本抛售
  - 机会：极端恐慌往往是底部

□ 绝望期
  - 特征："再也不碰股票"
  - 行为：彻底离场，成交极度萎缩
  - 机会：最佳买入区域
```

## 📋 工具3：反向指标清单

### 顶部信号（当出现多个时警惕）

- [ ] 出租车司机/理发师给你推荐股票
- [ ] 从不投资的亲戚开始问你买什么
- [ ] 媒体标题用"历史新高""疯狂"等词
- [ ] "这次不一样"的声音很多
- [ ] IPO首日涨幅惊人且数量激增
- [ ] 你觉得赚钱太容易了
- [ ] 开始后悔仓位太轻
- [ ] 新手投资者大量涌入

符合项数：____/8
→ ≥5项：高度警惕，可能接近顶部

### 底部信号（当出现多个时关注）

- [ ] 有经验的投资者也开始恐惧
- [ ] "这次熊市会很久"成为共识
- [ ] 媒体标题用"暴跌""崩盘"等词
- [ ] 成交量极度萎缩，无人关注市场
- [ ] 好公司也被无差别抛售
- [ ] 你觉得永远不会解套
- [ ] 恨不得割肉离场
- [ ] 老股民说"再也不玩了"

符合项数：____/8
→ ≥5项：可能接近底部，值得关注

## 💡 使用建议

1. **定期记录** - 每周观察一次，形成习惯
2. **保持客观** - 记录时尽量像观察别人一样观察自己
3. **回顾对照** - 每月回顾情绪记录与市场走势的关系
4. **逆向思考** - 当情绪极端时，考虑反方向的可能

**终极原则**：
> 当你最想行动时，往往应该冷静。
> 当你最不想行动时，往往应该考虑。
