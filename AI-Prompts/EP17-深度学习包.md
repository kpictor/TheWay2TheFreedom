## 📚 模块一：概念深化

### Prompt 1.1 - 羊群效应的进化逻辑

```markdown
帮我从进化心理学角度理解羊群效应：

**史前环境**
- 跟随群体的生存价值是什么？
- 特立独行的代价是什么？
- 这种本能如何编码进基因？

**现代金融市场**
- 从众本能如何被市场利用？
- 为什么"别人都在买"是强大的购买理由？
- 为什么"全民恐慌"时你也想跑？

**本能与理性的冲突**
- 什么时候从众是理性的？
- 什么时候从众是危险的？
- 如何区分两者？
```

### Prompt 1.2 - 信息瀑布理论

```markdown
解释"信息瀑布"如何产生羊群效应：

**理论框架**
1. 每个人都有私人信息
2. 每个人也观察他人的行为
3. 当他人行为成为更强信号时
4. 私人信息被忽略
5. 所有人跟随少数先行者
6. 即使群体方向是错的

**投资中的表现**
- 为什么分析师评级趋同？
- 为什么机构投资者扎堆？
- 这对散户意味着什么？

用具体的市场例子说明。
```
