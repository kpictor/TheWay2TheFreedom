# EP06 - 基金选择评估清单

> 实战工具：评估基金与基金经理

---

## 🎯 用前须知

**核心理念**：
> "理解激励结构，就理解了行为。选择与你利益一致的产品。"

**本工具用途**：
- 评估是否购买某只基金
- 分析现有持仓的风险
- 比较不同基金产品

---

## 📋 工具1：基金激励结构分析

### 购买前必查项目

| 检查项 | 查询方式 | 你的发现 | 风险等级 |
|--------|----------|----------|----------|
| **管理费率** | 基金合同 | ____% | □高(>1.5%) □中(1-1.5%) □低(<1%) |
| **托管费率** | 基金合同 | ____% | □高(>0.2%) □中 □低 |
| **申购费** | 销售渠道 | ____% | □高(>1%) □中 □低/免 |
| **赎回费** | 基金合同 | ____% | □高 □中 □低/无 |
| **基金规模** | 季报 | ____亿 | □过大(>500) □适中 □过小(<1) |
| **换手率** | 年报 | ____% | □高(>300) □中(100-300) □低(<100) |

**费用影响计算**：
```
假设持有10年，年化收益8%

费率对终值的影响：
1.5%费率：100元 → 约165元
0.5%费率：100元 → 约205元
0.1%费率：100元 → 约215元

差距可达30%！
```

| 评估项 | 情况 | 得分(1-5) |

## 📋 工具6：利益一致性检查清单

### "一致还是冲突"速查

买入前快速过一遍：

**基金公司层面**：
- [ ] 是否有主动限购控规模的记录？
- [ ] 是否在牛市顶部发新基金？
- [ ] 费率是否合理（<1%为佳）？

**基金经理层面**：
- [ ] 经理是否自购了这只基金？
- [ ] 任职时间是否超过3年？
- [ ] 过往规模扩大后业绩是否稳定？

**产品设计层面**：
- [ ] 是否有托管机制？
- [ ] 信息披露是否及时完整？
- [ ] 是否有合理的锁定期设计？

**你的层面**：
- [ ] 我是否理解这只基金的策略？
- [ ] 我是否能承受可能的波动？
- [ ] 我是否在追逐热点？

**警示**：
> 如果超过3项无法打勾，请重新考虑是否购买。

---

## 💡 使用建议

1. **新购基金必做**：工具1-4
2. **每季度**：工具5定期体检
3. **考虑赎回时**：工具6快速检查

**核心原则**：
> 费率是确定的损失，激励一致是成功的前提。
> 当有疑虑时，选择低费率指数基金永远不会太错。
