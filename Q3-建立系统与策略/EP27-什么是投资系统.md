---

## 💡 第一性原理拆解

### 系统 vs 随机决策

| 随机决策 | 系统化决策 |
|----------|------------|
| 看心情买卖 | 按规则执行 |
| 每次不一样 | 可重复验证 |
| 无法改进 | 可迭代优化 |
| 依赖运气 | 依赖概率 |

### 投资系统必须包含的5个要素

```
1. 入场规则 - 什么条件下买入？
2. 仓位规则 - 买多少？
3. 出场规则 - 什么条件下卖出？
4. 风险管理 - 最大亏损限制
5. 记录复盘 - 如何评估和改进
```

### Dalio原则

> "把决策过程变成原则，让原则指导决策，而不是每次都重新思考。"
