## 第一部分：为什么需要实盘（1:00-4:00）

### 模拟vs实盘的区别

| 维度 | 模拟/学习 | 真实实盘 |
|------|---------|---------|
| 情绪 | 几乎没有 | 真金白银的压力 |
| 执行 | 完美执行 | 会犹豫、会冲动 |
| 验证 | 只验证策略 | 验证策略+验证自己 |
| 学习 | 理论学习 | 实践出真知 |

### 实盘的真正目的

```
实盘的目的不是赚钱
（至少在学习阶段不是）

真正的目的是：
1. 验证你的系统在实战中是否有效
2. 认识真实的自己（情绪、执行力）
3. 发现系统的漏洞和改进点
4. 建立规则执行的肌肉记忆

赚钱是正确执行的副产品
```

## 第二部分：开始实盘的原则（4:00-7:00）

### 原则1：使用可承受的资金

```
关键问题：
这笔钱亏光了，会影响你的生活吗？

如果会 → 金额太大
如果不会 → 可以开始

建议：总可投资资金的10-20%
```

### 原则2：金额要有意义

```
太少：不痛不痒，学不到东西
太多：压力过大，变形操作

找到平衡：
- 亏损会让你在乎
- 但亏光不影响生活
```

### 原则3：完全按系统执行

```
实盘的第一个任务：
不是赚钱，是按规则执行

即使规则让你亏钱
也要先执行，再复盘调整

用规则验证规则
而不是用直觉推翻规则
```

### 原则4：详细记录

```
每一笔交易都要记录：
- 买卖理由
- 当时的情绪
- 是否按规则执行
- 事后反思

没有记录 = 没有改进
```

### 第一笔交易建议

```

1. 选择你最熟悉的标的
2. 仓位控制在总资金的10-20%以内
3. 在下单前，对照入场规则检查一遍
4. 下单后，记录当时的情绪状态
5. 设好止损
```

## 第四部分：常见问题应对（10:00-11:30）

### 问题1：第一笔就亏了

```
正确心态：
- 亏损是必然会发生的
- 关键是：按规则止损了吗？
- 如果按规则执行，亏损也是"对"的

记录：
- 分析为什么亏损
- 是系统问题还是判断问题
- 调整改进
```

### 问题2：赚了一点想落袋

```
检查：
- 当前是否触发了止盈规则？
- 如果没有，只是想落袋，那是情绪

应对：
- 回顾你的出场规则
- 问AI助手："这符合我的规则吗？"
- 按规则操作
```

### 问题3：规则和直觉冲突

```
原则：
在学习阶段，以规则为准

理由：
- 直觉需要时间验证
- 随意改变规则无法验证规则
- 先执行，再复盘

如果规则确实有问题：
- 在复盘时修改
- 不在交易时修改
```
