## 📌 本期核心问题

**你真正有多少钱可以投资？**

---

## 💡 第一性原理拆解

### 用资产负债表思维看个人财务

```
┌─────────────────────────────────────────────┐
│           个人资产负债表                      │
├─────────────────────────────────────────────┤
│ 资产（你有什么）    │ 负债（你欠什么）         │
├─────────────────────────────────────────────┤
│ 现金及存款          │ 房贷                    │
│ 股票/基金           │ 车贷                    │
│ 房产                │ 信用卡/消费贷           │
│ 其他投资            │ 其他负债                │
├─────────────────────────────────────────────┤
│ 总资产              │ 总负债                  │
├─────────────────────────────────────────────┤
│         净资产 = 总资产 - 总负债              │
└─────────────────────────────────────────────┘
```

### 可投资资金的真实计算

```
总现金资产
- 6个月紧急备用金
- 未来1年确定支出
- 已承诺的支出
─────────────────
= 真正可投资的钱
```

---

## 🔍 Dalio原则应用

### 原则：面对现实

不要用"应该有多少钱"来投资，而要用"实际能承受亏损多少"来规划。

**关键问题**：
- 如果投资亏损50%，对你的生活有什么影响？
- 这笔钱3年内需要用吗？
- 你有其他收入来源吗？
