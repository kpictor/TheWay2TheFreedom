## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 角色扮演：如果我是庄家

```markdown
假设你有1亿资金要控盘一只流通市值5亿的小盘股：

请和我一起思考每个阶段的操作：

1. **吸筹阶段**
   - 如何不惊动市场？
   - 目标筹码比例？
   - 需要多长时间？

2. **洗盘阶段**
   - 如何制造恐慌？
   - 如何判断洗盘成功？

3. **拉升阶段**
   - 需要多大资金推动？
   - 如何吸引跟风？
   - 媒体配合如何安排？

4. **出货阶段**
   - 如何边拉边出？
   - 如何让散户接盘？
   - 出货后如何收尾？

通过这个练习，理解庄家的完整思维链。
```

### Prompt 3.2 - 实盘案例分析
