## 📋 日志使用说明

```
┌────────────────────────────────────────────┐
│                                            │
│     交易日志使用指南                        │
│                                            │
│  📝 何时记录                               │
│  • 交易前：填写决策过程和情绪              │
│  • 交易后立即：确认基本信息                │
│  • 持仓期间：每日/每周更新跟踪              │
│  • 平仓后：完成结束复盘                    │
│                                            │
│  💡 记录要点                               │
│  • 诚实记录情绪，不美化                    │
│  • 明确标注是否符合规则                    │
│  • 区分决策质量和结果                      │
│                                            │
│  🔍 复盘要点                               │
│  • 每周回顾所有日志                        │
│  • 找出模式：什么时候容易犯错？            │
│  • 累积改进：每次学到一点                  │
│                                            │
│  ⚠️ 常见错误                               │
│  • 赚钱才记录，亏钱不记                    │
│  • 只记结果，不记过程                      │
│  • 记了不回顾                              │
│                                            │
└────────────────────────────────────────────┘
```
