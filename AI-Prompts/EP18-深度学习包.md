# EP18 - 叙事经济学 | AI深度学习包

> 🎯 目标：理解故事如何驱动市场，学会分析和利用叙事

---

## 📚 模块一：概念深化

### Prompt 1.1 - 叙事的力量

```markdown
帮我理解"叙事"如何驱动市场：

**人类是叙事动物**
- 为什么故事比数据更有影响力？
- 大脑处理故事和数据有什么不同？
- 为什么"下一个茅台"比财报数据更能驱动股价？

**市场叙事的传播**
1. 起源：谁创造叙事？
2. 传播：通过什么渠道？
3. 变异：如何简化和夸大？
4. 高峰：何时"所有人都知道"？
5. 衰退：新叙事如何取代旧叙事？

**识别叙事的方法**
- 当前市场最流行的叙事是什么？
- 这个叙事已经存在多久了？
- 处于传播周期的哪个阶段？
```

### Prompt 1.2 - 叙事与估值的关系

```markdown
分析叙事如何影响估值：

**案例分析**
- 同样的业绩，"消费升级"叙事下PE 40，"行业衰退"叙事下PE 15
- 为什么叙事能带来如此大的估值差异？

**叙事的量化影响**
- 有没有研究测量叙事对股价的影响？
- 叙事驱动多少？基本面驱动多少？
- 短期和长期有什么不同？

**叙事陷阱**
- 什么时候叙事是真实的增长潜力？
- 什么时候叙事只是泡沫制造器？
- 如何区分两者？
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 罗伯特·席勒的叙事经济学

```markdown
介绍诺贝尔经济学奖得主席勒的叙事经济学理论：

**核心观点**
- 经济事件如何通过叙事传播？
- 病毒式传播的叙事有什么特点？
- 为什么叙事可以自我实现？

**历史案例**
- "咆哮的二十年代"叙事如何催生泡沫
- "新经济"叙事的形成和破灭
- 加密货币叙事的演变

这个理论对我的投资有什么启示？
```

### Prompt 2.2 - 概念股/题材股分析

```markdown
教我如何分析"概念股"和"题材热点"：

**分析框架**
1. **叙事来源**：政策？科技突破？社会趋势？
2. **传播阶段**：早期/中期/晚期？
3. **受益证伪**：公司真的受益吗？多大程度？
4. **估值锚定**：重新定价是否合理？
5. **持续性判断**：叙事能持续多久？

**红旗信号**
- 什么信号说明叙事快到顶了？
- 什么信号说明叙事开始崩塌？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 当前叙事分析

```markdown
帮我分析一个当前热门叙事：

请选择一个（或我指定）：AI、新能源、半导体、元宇宙、医药...

分析框架：
1. 叙事的核心故事是什么？
2. 叙事如何传播？谁在传播？
3. 叙事的真实成分有多少？
4. 泡沫成分有多少？
5. 处于周期的什么阶段？
6. 投资应该怎么应对？

我会给出分析，请给我反馈。
```

### Prompt 3.2 - 反叙事练习

```markdown
练习识别和质疑流行叙事：

**练习1**：给一个流行的牛市叙事，写出5个反驳论点
**练习2**：给一个流行的熊市叙事，写出5个反驳论点

目的：
- 强化逆向思考能力
- 不被当下情绪左右
- 培养独立思考习惯
```
