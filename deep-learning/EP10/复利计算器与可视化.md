## 🔢 复利核心公式

```
终值 = 本金 × (1 + 年化收益率)^年数

FV = PV × (1 + r)^n

其中：
FV = Future Value（未来值）
PV = Present Value（现值/本金）
r  = 年化收益率
n  = 年数
```

## 📊 72法则速算
