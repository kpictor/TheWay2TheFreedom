# EP01 - 视觉呈现建议

> 视频制作的视觉指南

---

## 🎨 整体视觉风格

| 元素 | 建议 |
|------|------|
| **主色调** | 深蓝 + 金色（专业+信任） |
| **字体** | 思源黑体（标题）+ 苹方（正文） |
| **风格** | 简洁现代，避免花哨 |
| **背景** | 浅灰或虚化书房环境 |

---

## 📊 逐段视觉建议

### 开场（0:00 - 0:30）

**画面1：人物特写**
- 主讲人面对镜头，自然光
- 背景虚化，可见书架或电脑

**画面2：B-Roll插入**
- 股票交易大厅画面（2-3秒）
- 投资者看屏幕的表情（2秒）
- 数字跳动动画（2秒）

**字幕设计**
```
[大号标题，渐入效果]
投资的第一性原理
```

---

### 核心内容（0:30 - 2:30）

**动画1：核心定义呈现**

```
背景：深蓝色渐变

第一步（3秒）：
    "投资"文字从左侧滑入

第二步（3秒）：
    "=" 淡入
    "不确定性" 从上方落下
    "资源配置" 从下方升起
    "决策" 从右侧滑入

第三步（2秒）：
    整句话组合，金色边框高亮
```

**动画2：三个关键词**

```
样式：三个圆形图标横排

    🎲              💰              ✋
  不确定性          资源            决策
   
  [依次放大+        [依次放大+      [依次放大+
   文字解释]         文字解释]       文字解释]
```

**动画3：生活场景**

| 场景 | 视觉元素 | 时长 |
|------|----------|------|
| 租房vs买房 | 房子图标 + 钥匙 | 2秒 |
| 跳槽vs留守 | 办公室 + 箭头 | 2秒 |
| 结婚时机 | 戒指 + 日历 | 2秒 |

---

### 收尾（2:30 - 3:00）

**画面：核心定义强调**

```

投资 = 在不确定性中
       做出资源配置决策
       
[配合音效：书写声]
```

## 🎬 动画资源建议

### 适合使用的动画

1. **文字动画**
   - 打字机效果（逐字显示）
   - 滑入效果（方向感）
   - 缩放强调（重点内容）

2. **图表动画**
   - 数据增长曲线
   - 饼图分解
   - 流程图逐步展示

3. **转场**
   - 简洁的淡入淡出
   - 推拉效果
   - 避免花哨的3D转场

### 不建议使用

- ❌ 过多的3D效果
- ❌ 复杂的粒子动画
- ❌ 华丽但分散注意力的转场
- ❌ 与内容无关的装饰元素

## 🎵 音频建议

| 时段 | 音乐类型 | 音量 |
|------|----------|------|
| 开场 | 轻快科技感 | 60% |
| 核心内容 | 低沉背景音 | 30% |
| 强调时刻 | 暂停/静音 | 0% |
| 收尾 | 渐强励志感 | 50% |

**音效使用**：
- 关键词出现：清脆提示音
- 转场：轻微空间音
- 问题提出：悬念音效

## 📐 字幕规范

| 类型 | 字号 | 颜色 | 位置 |
|------|------|------|------|
| 主标题 | 72px | 白色/金色 | 居中 |
| 重点句 | 48px | 白色 | 下方1/3 |
| 普通字幕 | 36px | 白色 | 底部 |
| 注释 | 24px | 灰色 | 底部角落 |

## ✅ 视频自检清单

制作完成后检查：

- [ ] 核心定义至少出现3次
- [ ] 三个关键词都有视觉强调
- [ ] 每个场景停留时间>2秒
- [ ] 音乐不遮盖人声
- [ ] 字幕无错别字
- [ ] 结尾有明确的Call to Action
