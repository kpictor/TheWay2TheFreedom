# EP14 - 市场是人心的映射 | AI深度学习包

> 🎯 目标：理解价格波动背后的集体心理，建立市场心理分析能力

---

## 📚 模块一：概念深化

### Prompt 1.1 - 价格的本质

```markdown
帮我深入理解"价格是集体心理的表达"：

**哲学层面**
1. 价格反映的是"价值"还是"看法"？
2. 市场价格何时接近"真实价值"？何时偏离？
3. 为什么说"美是观看者眼中的美"同样适用于价格？

**心理学层面**
1. 价格变动如何影响人的心理？
2. 心理变化又如何影响价格？（反身性预览）
3. 这个循环如何自我强化？

**实践层面**
1. 如何从价格变化"读"情绪？
2. 成交量能告诉我们什么情绪信息？
```

### Prompt 1.2 - 四大情绪驱动力

```markdown
深入分析驱动市场的四种情绪：

**恐惧**
- 触发因素是什么？
- 在K线和成交量上如何体现？
- 恐惧到极点时通常是什么信号？

**贪婪**
- 贪婪如何自我强化？
- "害怕踏空"是贪婪还是恐惧？
- 贪婪极点的信号是什么？

**希望**
- 为什么亏损时人们抱有希望？
- 希望如何导致延迟止损？
- 什么时候希望是有害的？

**后悔**
- 后悔如何导致追高买入？
- 后悔如何导致恐慌卖出？
- 如何打破后悔的影响？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 情绪历史案例分析

```markdown
用情绪视角分析历史市场事件：

请和我一起分析：
1. 2007年A股6124点：当时的集体情绪是什么？
2. 2008年最低点：情绪又是什么？
3. 2015年牛市到股灾：情绪如何演变？
4. 2020年3月暴跌：恐惧如何传染？

对每个案例：
- 描述当时的主流情绪
- 识别情绪极端的信号
- 事后看来的正确做法
```

```markdown

1. 你经历过的最恐慌时刻是什么时候？
2. 当时你做了什么决策？
3. 这个决策正确吗？
4. 如果再来一次，你会怎么做？
5. 你经历过的最贪婪时刻呢？
```
