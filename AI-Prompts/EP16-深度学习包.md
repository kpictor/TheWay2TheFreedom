## 📚 模块一：概念深化

### Prompt 1.1 - 认知偏差深度解读

```markdown
请帮我深入理解这12个认知偏差，每个都用投资场景举例：

1. **确认偏差** - 买入后只看利好
2. **锚定效应** - 死守成本价
3. **损失厌恶** - 亏损时死扛，小赚就跑
4. **过度自信** - 频繁交易，仓位过重
5. **后见之明** - "我早就知道会涨/跌"
6. **从众效应** - 追涨杀跌
7. **可得性偏差** - 被最近新闻影响过度
8. **心理账户** - 对"赚来的钱"更冒险
9. **沉没成本** - "已经亏这么多了，舍不得卖"
10. **现状偏差** - 该调仓不调仓
11. **禀赋效应** - 高估自己持有的股票
12. **过度交易** - 用行动获得控制感幻觉

对每个偏差，请给出：
- 心理根源
- 典型场景
- 识别信号
- 对抗方法
```

### Prompt 1.2 - 个人偏差诊断

```markdown
帮我诊断我最常犯的认知偏差：

请逐个问我以下问题，根据我的回答判断我的偏差倾向：

1. 买入后你更关注利好还是利空消息？
2. 你是否因为"已经涨了/跌了很多"而决定买卖？
3. 你的持仓中，盈利和亏损的比例是多少？
4. 你每月交易几次？
5. 你是否经常说"早知道就..."？
6. 你的决策是否受周围人影响？
7. 最近一条新闻如何影响了你的决策？
8. 你对"赚来的钱"更敢冒险吗？
9. 你有没有因为"已经亏太多"而继续持有？
10. 你最近调整过仓位吗？

完成后，告诉我我的TOP 3偏差是什么。
```

### Prompt 2.2 - 去偏差技术

```markdown
介绍专业领域使用的"去偏差"技术：

1. **预先承诺**（Pre-commitment）
   - 在情绪冷静时制定规则
   - 用规则约束未来的自己

2. **红队思维**（Red Team Thinking）
   - 强制考虑反面论点
   - 魔鬼代言人技巧

3. **清单法**（Checklist）
   - 飞行员和医生的清单
   - 投资决策清单如何设计？

4. **外部视角**（Outside View）
   - 基准比率的使用
   - "如果是别人的决策，你怎么评价？"

请帮我设计适合投资的去偏差方法。
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 偏差识别练习

```markdown
让我们做偏差识别练习：

我会给你几个投资决策场景，请判断涉及什么偏差：

**场景1**
"这只股票我20元买的，现在15元，但我查了很多分析都说会涨回来，所以继续持有。"
涉及的偏差？正确做法？

**场景2**
"上周这只股票涨了30%，错过了。今天又涨了，不想再错过了，买入！"
涉及的偏差？正确做法？

**场景3**
"这只股票赚了50%，把本金取出来，用利润继续赌大的。"
涉及的偏差？正确做法？

请逐一分析。
```
