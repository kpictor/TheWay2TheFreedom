## 心理与行为核心提示词

### EP14-16 - 情绪与认知偏差

```markdown
扮演一位认知心理学家，帮我分析我的投资行为：

1. 我将描述一次投资决策
2. 请识别其中的情绪因素
3. 指出可能的认知偏差
4. 分析这些偏差如何影响了结果
5. 给我具体的改进建议

让我开始描述：[开始对话]
```

### EP17-18 - 羊群效应与叙事

```markdown
帮我分析一个当前市场热点的叙事结构：

我将提供一个热门话题（如AI、新能源等）

请分析：
1. 核心叙事是什么？
2. 叙事如何传播？
3. 泡沫成分占多少？
4. 如何验证叙事真实性？
5. 作为投资者应该怎么想？
```

### EP19-20 - 央行与庄家思维

```markdown
帮我解读最近的央行政策或市场异动：

我将提供：
- 政策/事件描述
- 市场反应

请分析：
1. 真正的信号是什么？
2. 谁在背后操作？
3. 对我有什么影响？
4. 我应该如何应对？
```

### EP21-22 - 博弈论与反身性

```markdown
用博弈论和反身性理论分析以下市场现象：

[描述一个市场现象]

请分析：
1. 各方参与者是谁？
2. 各自的策略是什么？
3. 有没有反身性循环？
4. 均衡点可能在哪里？
5. 我应该采取什么策略？
```

### EP23-26 - 周期与逆向思维

```markdown
帮我进行市场情绪周期定位：

根据以下信息：
1. 当前市场估值水平
2. 媒体标题倾向
3. 周围人的言论
4. 融资余额/基金发行

判断：
- 当前处于情绪周期的哪个阶段？
- 我应该采取什么策略？
- 有没有我遗漏的反向信号？
```
