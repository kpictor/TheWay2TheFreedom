## 📚 模块一：概念深化

### Prompt 1.1 - 信息茧房的形成机制

```markdown
帮我深入理解信息茧房如何形成：

**算法推荐的机制**
1. 平台如何"了解"你的偏好？
2. 推荐算法如何强化偏好？
3. 为什么你看到的信息越来越同质？

**投资者的信息茧房**
1. 你持有某股票后会搜索什么？
2. 确认偏差如何与算法共振？
3. 社交圈如何强化茧房？

**信息茧房的危害**
1. 看似"充分研究"实则只见一面
2. 错过关键的负面信息
3. 过度自信的来源

如何识别自己已经在茧房中？
```

### Prompt 1.2 - 逆向思维的本质

```markdown
帮我理解逆向思维在投资中的应用：

**逆向思维不是什么**
- 不是为了反对而反对
- 不是简单的做空或做多
- 不是无脑与众人相反

**逆向思维是什么**
- 主动寻找与自己相反的观点
- 诚实评估反方论据的强度
- 在群体极端时保持清醒

**霍华德·马克斯的"第二层思维"**
- 第一层："这是好公司，买入"
- 第二层："人人都知道这是好公司，价格已经反映，卖出"

给出具体例子说明两层思维的区别。
```

### Prompt 2.2 - 打破信息茧房的工具

```markdown
介绍打破信息茧房的具体工具和方法：

**信息源多元化**
- 关注与自己观点相反的人
- 阅读对立立场的分析
- 定期更换信息渠道

**强制反方辩论**
- Red Team方法：为反方寻找证据
- 预设必须找到X条反对理由
- 和立场相反的人对话

**检查清单方法**
- 决策前必须回答的反向问题
- "最看好的人也会是错的，为什么？"

帮我设计个人的"反茧房"系统。
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 信息茧房自查

```markdown
帮我检查自己的信息茧房：

**信息源审计**
列出你获取投资信息的来源：
1. ___
2. ___
3. ___

对每个来源问：
- 这个来源的倾向性是什么？
- 有没有与之对立的来源？
- 你有关注对立来源吗？

**持仓信息偏差检查**
对你目前最大的持仓：
- 你看到的信息是正面多还是负面多？
- 你有主动搜索负面信息吗？
- 你能说出三个做空这个股票的理由吗？
```

### Prompt 3.2 - 逆向思维练习

```markdown
对以下观点进行逆向思维练习：

**流行观点1**："AI是未来最大的投资机会"

- 什么情况下这个观点会错？

**流行观点2**："房价不可能大跌"
- 反方观点可能是什么？
- 有什么历史先例反驳这个观点？
- 什么条件下这个观点会失效？

**你自己的观点**：[填入你最坚定的一个投资观点]
- 我会帮你寻找反方论据
```
