# EP02 - 为什么大多数人亏钱 | AI深度学习包

> 🎯 目标：深刻理解散户亏损的结构性原因，建立对手盘思维

---

## 📚 模块一：概念深化

### Prompt 1.1 - 数学视角解析

```markdown
我正在学习"为什么大多数散户亏钱"。

请从数学和统计学角度帮我理解：

1. **费用侵蚀**：假设年化收益8%，但每年交易成本2%，10年后差距有多大？
2. **负和博弈**：为什么考虑费用后，股市短期是负和游戏？
3. **帕累托分布**：80-20法则在投资收益分配中如何体现？
4. **均值回归**：过去的高收益为什么不能预测未来？

请用数字和简单图表说明每个概念。
```

### Prompt 1.2 - 博弈论视角

```markdown
用博弈论框架分析股票交易：

1. 每笔交易是什么类型的博弈？（零和/正和/负和）
2. 买卖双方的信息是如何分布的？
3. 散户在这个博弈中处于什么位置？
4. 机构的结构性优势是什么？
5. 有没有散户占优的情况？

请具体举例说明。
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 归因分析练习

```markdown
我将描述我的一次亏损经历。请帮我进行归因分析：

[在这里描述你的亏损经历]

分析框架：
1. 这次亏损中，有多少是**运气因素**（市场环境、黑天鹅）？
2. 有多少是**认知因素**（判断错误、信息不足）？
3. 有多少是**行为因素**（情绪化、违反规则）？
4. 有多少是**结构因素**（作为散户的固有劣势）？

请帮我深入分析，不要让我轻易甩锅给"运气不好"。
```

### Prompt 3.2 - 对手盘模拟

```markdown
我要买入一只股票。请扮演我的对手盘（卖给我的人），告诉我：

1. 你是谁？（机构/大户/另一个散户？）
2. 你为什么愿意在这个价格卖给我？
3. 你可能知道什么我不知道的？
4. 你对这只股票的后续判断是什么？
5. 你认为买入的散户（我）犯了什么错误？

请尽可能站在对手的角度思考，帮我看到自己的盲点。
```
