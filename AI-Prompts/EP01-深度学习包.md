# EP01 - 投资的第一性原理 | AI深度学习包

> 🎯 目标：完全掌握"投资的本质是什么"这一核心问题

---

## 📚 模块一：概念深化

### Prompt 1.1 - 多角度概念解释

```markdown
你是一位结合了认知科学和投资经验的导师。

我正在学习"投资的第一性原理"，核心定义是：
> 投资 = 在不确定性中做出资源配置决策

请从以下5个角度帮我深化理解这个定义：

1. **哲学角度**：不确定性的本质是什么？人类如何面对不确定性？
2. **数学角度**：用概率论的语言重新表述这个定义
3. **心理学角度**：人类大脑如何处理不确定性决策？
4. **经济学角度**：资源配置的本质是什么？
5. **历史角度**：历史上成功的投资者如何理解这个定义？

请用简洁、有洞察力的语言回答，每个角度不超过100字。
```

### Prompt 1.2 - 类比迁移

```markdown
请帮我找出5个日常生活中的场景，它们本质上也是"在不确定性中做资源配置决策"：

要求：
1. 场景必须是大多数人都经历过的
2. 说明这个场景中的"不确定性"是什么
3. 说明这个场景中的"资源"是什么
4. 说明好决策和坏决策的区别

最后，问我：这些类比中，你自己做过哪些决策？你当时的决策质量如何？
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 知识图谱构建

```markdown
作为投资教育专家，请帮我构建"投资第一性原理"的知识图谱。

这个图谱应该包含：

**核心节点**：投资的本质

**关联节点**：
- 历史人物：谁提出过类似思想？
- 学术理论：相关的学术研究有哪些？
- 经典著作：哪些书系统讨论过这个问题？
- 实践案例：哪些投资者以此为原则？

**对立观点**：
- 有人认为投资可以预测确定结果吗？
- 这种观点的问题是什么？

请以思维导图的文字形式呈现。
```

### Prompt 2.2 - 历史案例库

```markdown
请给我讲述3个历史上的投资案例，每个案例要展示：

1. **案例背景**：发生了什么？
2. **决策时刻**：投资者面对什么不确定性？
3. **决策过程**：他们如何思考？
4. **结果与教训**：我们能学到什么？

案例要求：
- 1个成功案例
- 1个失败案例
- 1个争议案例（结果好但过程有问题，或相反）
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 逻辑链条构建

```markdown

从"投资的本质"推导出"为什么大多数人亏钱"

2. 根据我的回答决定下一个问题
3. 如果我的思路偏了，用问题引导我回来
4. 最后帮我总结整个逻辑链条

开始第一个问题吧。
```

### Prompt 3.2 - 反驳练习

```markdown

要求：
1. 问题要深刻，不能是表面的
2. 如果我回答不出来，说明我理解不够深
3. 提供参考答案供我学习
4. 最后评估我的理解程度（1-10分）
```

## ✅ 模块五：掌握度验证

### Prompt 5.1 - 综合测试

```markdown
我已经学完EP01"投资的第一性原理"。

请对我进行掌握度测试（请直接开始提问，一次一题）：

**L1理解**：用你自己的话解释什么是投资的第一性原理？:

**L2应用**：给你一个具体场景，判断这是否是一个"投资决策"，为什么？

**L3分析**：为什么说"未来不可知"是投资的第一真相？这与"可以预测"的观点有什么区别？

**L4评价**：有人说"高频量化可以预测市场"，这与第一性原理矛盾吗？评价这个观点。

**L5创造**：如果你要向一个10岁的孩子解释这个概念，你会怎么说？

根据我的回答，评估我的掌握等级（L1-L5），并指出需要补强的地方。
```

