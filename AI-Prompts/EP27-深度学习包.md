## 📚 模块一：概念深化

### Prompt 1.1 - 系统vs随机决策

```markdown
帮我深入理解"系统化决策"与"随机决策"的区别：

**随机决策的特征**
1. 每次决策标准不同
2. 受情绪状态影响
3. 难以复盘和改进
4. 结果好坏靠运气

**系统化决策的特征**
1. 有明确的规则
2. 规则优先于情绪
3. 决策可追溯、可评估
4. 通过数据迭代改进

**核心洞察**
> "用规则约束未来的自己"

用具体例子对比两种决策方式的长期结果。
```

### Prompt 1.2 - 投资系统的五个要素

```markdown
详细解释投资系统必须包含的五个要素：

**1. 入场规则**
- 什么条件下买入？
- 如何让规则明确、可验证？

**2. 仓位规则**
- 买多少？如何确定？
- 风险如何控制？

**3. 出场规则**
- 什么条件下卖出？
- 止损、止盈如何设置？

**4. 风险管理**
- 单笔最大亏损？
- 组合最大回撤？

**5. 记录复盘**
- 如何记录每笔交易？
- 如何评估和改进系统？

对每个要素，请给出具体的示例规则。
```
