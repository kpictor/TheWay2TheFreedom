## 📚 模块一：概念深化

### Prompt 1.1 - 群体智慧的四个条件

```markdown
帮我理解群体何时"聪明"：

**苏罗维基的四个条件**

1. **多样性**：成员背景、观点、信息来源不同
   - 市场何时满足？何时不满足？
   - 同质化如何破坏多样性？

2. **独立性**：个体判断不受他人影响
   - 社交媒体如何破坏独立性？
   - 羊群效应的影响？

3. **分散性**：每个人有自己的局部信息
   - 专业分工如何贡献？
   - 信息聚合的价值？

4. **汇聚机制**：有方法将判断汇总
   - 市场价格作为汇聚机制
   - 价格能有效汇聚吗？

当这四个条件满足时，市场是有效的；当任一条件破坏时，市场可能失灵。
```

### Prompt 1.2 - 何时市场失灵

```markdown
分析市场失灵的情况：

**泡沫形成时**
- 多样性如何消失？（都在追涨）
- 独立性如何消失？（从众）
- 这时价格还可信吗？

**恐慌时期**
- 理性分析如何让位于情绪？
- 信息如何被误读？
- 这时应该相信市场还是相反？

**极端共识时**
- 当90%的人持同一观点
- 这是智慧还是疯狂？
- 历史上的案例？

如何识别"群体疯狂"的时刻？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 判断当前市场状态

```markdown
帮我分析当前市场是"智慧"还是"疯狂"：

**检查四个条件**

1. **多样性检查**
   - 市场参与者的观点分布如何？
   - 有没有被某一叙事主导？

2. **独立性检查**
   - 投资者的决策独立吗？
   - 社交媒体的影响程度？

3. **分散性检查**
   - 信息来源多样吗？
   - 是否存在信息优势方？

4. **汇聚机制检查**
   - 价格能反映信息吗？
   - 有没有扭曲因素？

综合判断后给出你的结论。
```

### Prompt 3.2 - 历史案例分析

```markdown
分析一个"群体疯狂"案例和一个"群体智慧"案例：

**疯狂案例**（如2000年科技泡沫）
- 四个条件中哪些被破坏？
- 疯狂的信号是什么？
- 如何能事先识别？

**智慧案例**（如某次价格发现）
- 四个条件如何满足？
- 市场如何有效定价？
- 有什么值得学习的？
```
