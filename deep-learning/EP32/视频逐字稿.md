大家好，欢迎来到《通往自由之路》第32期。

今天我们学习一个可能会彻底改变你投资方式的概念：**风险预算**。

传统思维是：
> "我想赚多少钱，所以我要买多少。"

风险思维是：
> "我能亏多少钱，所以我只能买多少。"

**用风险倒推仓位，而不是用期望倒推仓位。**

这是专业投资者和业余投资者的核心区别之一。

---

## 第一部分：风险预算的概念（1:00-4:00）

### 什么是风险预算

```
风险预算 = 你愿意为一笔交易承担的最大亏损

例子：
总资产：100万
风险预算（单笔）：1% = 1万

意味着：这笔交易最多亏1万
```

### 为什么要用风险倒推

**收益导向的问题**：
```
"我想赚10万"
→ 买入100万（假设预期涨10%）
→ 结果跌了20%
→ 亏损20万
→ 远超承受能力
```

**风险导向的优势**：
```
"我最多亏1万"
→ 计算合理仓位
→ 无论涨跌，损失可控
→ 始终有资金继续战斗
```

### 风险预算的三个层次

| 层次 | 描述 | 示例 |
|------|------|------|
| 单笔风险 | 单次交易最大亏损 | 账户的1-2% |
| 组合风险 | 所有持仓的总风险 | 账户的10-20% |
| 时间风险 | 一段时间内最大亏损 | 月度最大5% |

## 第二部分：风险预算计算（4:00-7:00）

### 单笔风险计算

**公式**：
```
可承受亏损 = 总资产 × 单笔风险比例
最大仓位 = 可承受亏损 ÷ 止损幅度
```

**例子**：
```
总资产：100万
单笔风险：2%
可承受亏损 = 100万 × 2% = 2万

止损设在10%
最大仓位 = 2万 ÷ 10% = 20万

即：只能买20%仓位
```

### 不同止损下的仓位变化

```
可承受亏损 = 2万

止损5%  → 仓位40万（40%）
止损10% → 仓位20万（20%）
止损20% → 仓位10万（10%）

止损越宽松，仓位越小
这是风险恒定的逻辑
```

### 组合层面风险预算

```
规则：所有持仓的总风险不超过账户的20%

检查方法：
持仓1：风险 = 仓位 × 止损幅度 = 20万 × 10% = 2万
持仓2：风险 = 仓位 × 止损幅度 = 15万 × 12% = 1.8万
持仓3：风险 = 仓位 × 止损幅度 = 10万 × 8% = 0.8万

总风险 = 2万 + 1.8万 + 0.8万 = 4.6万
总风险比例 = 4.6万 / 100万 = 4.6%

在20%以内，组合风险可控
```

## 第四部分：风险预算的应用（10:00-11:30）

### 每次交易前的检查

```
交易前风险检查：

1. 这笔交易的风险是多少？
   风险 = 计划仓位 × 止损幅度

2. 是否超过单笔风险预算？
   风险 ≤ 账户 × 单笔风险比例 ？

3. 加上这笔交易，组合总风险是多少？
   总风险 ≤ 账户 × 组合风险比例 ？

4. 本月已亏损多少？还有多少风险预算？
   剩余预算 = 月度预算 - 已实现亏损
```

### 亏损后的调整

```
连续亏损后的应对：

如果月度亏损接近预算：
→ 暂停交易
→ 复盘检查
→ 下月重新开始

如果单笔亏损超预算：
→ 分析原因（止损未执行？仓位太大？）
→ 修正系统漏洞
→ 下次严格执行
```
