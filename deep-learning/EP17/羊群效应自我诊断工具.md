# EP17 - 羊群效应自我诊断工具

> 识别并量化你的从众倾向

---

## 🎯 使用说明

本工具帮助你：
1. 评估自己的从众倾向程度
2. 识别触发从众行为的情境
3. 建立个人化的对抗策略

## 📊 第一部分：从众倾向基础测评

### 测评说明
对于每个陈述，诚实评分（1-5分）：
- 1 = 完全不符合
- 2 = 较少符合

- 5 = 完全符合

## 📋 第四部分：决策前检查清单

### 买入决策检查

```
当我想买入时，必须回答：

□ 1. 这个决定是我自己分析的结果吗？
     是/否：____
     如果"否"，暂停

□ 2. 我能说出3个独立于他人意见的买入理由吗？
     理由1：_____________
     理由2：_____________
     理由3：_____________
     如果说不出，暂停

□ 3. 如果没有人讨论这个标的，我还会感兴趣吗？
     是/否：____
     如果"否"，这可能是FOMO

□ 4. 我是否感到"不买就来不及了"？
     是/否：____
     如果"是"，强制等待24小时

□ 5. 这个买入符合我的投资计划吗？
     是/否：____
     如果"否"，不要买入
```

### 卖出决策检查

```
当我想卖出时，必须回答：

□ 1. 我卖出是因为分析还是因为恐惧？
     原因：____
     如果是恐惧，暂停

□ 2. 基本面发生了什么变化？
     变化：_____________
     如果没变化，谨慎卖出

□ 3. 所有人都在卖时，我确定要跟随吗？
     是/否：____
     考虑逆向思维

□ 4. 这符合我预设的止损/止盈规则吗？
     是/否：____
     如果"是"，执行规则
     如果"否"，重新评估

□ 5. 我能冷静地等待24小时再决定吗？
     是/否：____
     如果能等，等一下
```
