## ⚡ 第一步：认识盈利的危险

### 为什么盈利比亏损更危险

```markdown
帮我理解为什么连续盈利可能更危险：

| 亏损时 | 盈利时 |
|--------|--------|
| 容易识别问题 | 难以发现问题 |
| 变得谨慎 | 变得大胆 |
| 反思很自然 | 反思被忽略 |
| 不会放松规则 | 容易放松规则 |

**盈利带来的心理陷阱**
1. 过度自信："我已经掌握了市场"
2. 归因错误：把运气当能力
3. 放松纪律："这次可以例外"
4. 生活方式改变：提前消费未来收益

历史上有哪些"盈利后崩溃"的案例？
```

---

## 🔍 第二步：盈利自检清单

### 每次盈利后的强制检查

```markdown
# 盈利自检表

当你连续盈利或账户创新高时，完成以下检查：

**归因分析**
□ 这次盈利是因为运气还是能力？
□ 市场环境帮了多少忙？
□ 如果市场反转，我还能盈利吗？

**纪律检查**
□ 我的止损规则还在执行吗？
□ 我的仓位是否超过了规则上限？
□ 我有没有放松入场标准？

**行为警示**
□ 我有没有想加杠杆的冲动？
□ 我有没有提前消费的念头？
□ 我是否开始"指导"别人投资？

**如果有任何一项警示亮起，你需要冷静了。**
```

### Prompt A - 盈利时的对手盘

```markdown
我最近连续盈利，账户盈利___，请扮演我的对手盘：

1. 这个盈利中有多少是运气？
2. 你有什么证据证明自己有能力而非运气？
3. 如果下个月亏掉一半，你会怎么想？
4. 你的交易频率有变化吗？
5. 你的仓位有变化吗？
6. 你确定没有放松规则吗？

请严格挑战我，不要让我飘。
```

## 🧊 第三步：保持纪律的机制

### 盈利后的强制冷静

```markdown
设计盈利后的强制冷静机制：

**触发条件**
- 月度收益率超过___%
- 账户创新高
- 连续盈利___次

**强制动作**
1. 完成盈利自检表
2. 和___分享并请其监督
3. 暂停___天后才能新开仓
4. 复查所有持仓的止损设置

**禁止动作**
- 加仓
- 加杠杆
- 放松止损
- 告诉别人自己"赚了很多"
```
