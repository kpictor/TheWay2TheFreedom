## 🔧 第二步：复盘辅助

### Prompt A - AI辅助周复盘

```markdown
请引导我完成本周复盘。

逐步问我以下问题：
1. 本周做了几笔交易？结果如何？
2. 哪些是按规则执行的？哪些不是？
3. 本周情绪最波动的时刻是什么？
4. 最值得骄傲的决策是哪个？
5. 最后悔的决策是哪个？
6. 下周最想改进的一点是什么？

每个问题等我回答后再问下一个。
```

### Prompt B - 复盘质量评估

```markdown
这是我的周复盘报告：
[粘贴复盘内容]

请帮我评估复盘质量：
1. 是否坦诚面对错误？
2. 是否有具体的改进计划？
3. 有没有遗漏重要的反思点？
4. 改进计划是否可执行？
5. 给这份复盘打分（1-10）
```
