# EP29 - 仓位计算工具箱

> 科学计算每一笔交易的合理仓位

---

## 🎯 使用说明

本工具帮助你：
1. 基于风险控制计算每笔交易仓位
2. 使用凯利公式评估最优仓位
3. 建立仓位管理规则

---

## 📋 工具一：单笔交易仓位计算器

### 基于风险的仓位计算

```
单笔交易仓位计算器
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤1：基本信息】

账户总资产：____________元
本次交易标的：______________
当前价格：____________元

【步骤2：确定风险参数】

单笔最大风险比例：____%
（建议：保守1%，标准2%，激进3%）

可承受最大亏损 = 账户 × 风险比例
= _________ × ____%
= _________元

【步骤3：设定止损】

计划止损价：____________元
止损幅度 = (当前价 - 止损价) ÷ 当前价
= (_______ - _______) ÷ _______
= ____%

【步骤4：计算仓位】

最大可买入金额 = 可承受亏损 ÷ 止损幅度
= _________ ÷ ____%
= _________元

最大可买股数 = 可买金额 ÷ 当前价
= _________ ÷ _________
= _________股

仓位占比 = 可买金额 ÷ 账户总资产
= _________ ÷ _________
= ____%

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【结论】

建议仓位：____%，约_________元
最大购买：_________股

【检查】
□ 不超过单只股票上限（____%）
□ 不超过行业上限（____%）
□ 该风险可接受
```

## 📋 工具二：凯利公式计算器

### 计算理论最优仓位

```
凯利公式仓位计算
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【输入参数】

交易标的：______________

你估计的胜率(p)：____%
（过去类似交易的成功率，或主观估计）

你估计的盈亏比(b)：____
（平均盈利 ÷ 平均亏损）

失败率(q) = 1 - p = ____%

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【凯利公式计算】

f* = p - q/b
= ___% - ___% ÷ ____
= ___% - ___%
= ____%

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【调整后建议】

全凯利仓位：____%
半凯利仓位：____%（推荐）
1/4凯利仓位：____%（保守）

我选择使用：
○ 全凯利
○ 半凯利
○ 1/4凯利
○ 自定义：____%

━━━━━━━━━━━━━━━━━━━━━━━━━━━

【与风险规则对比】

风险规则计算的仓位：____%
凯利公式建议仓位：____%

最终决策：取较小值 = ____%

原因：
□ 估计可能过于乐观
□ 黑天鹅风险
□ 心理承受能力
```
