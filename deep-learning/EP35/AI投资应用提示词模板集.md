# EP35 - AI投资应用提示词模板集

> 即拿即用的AI投资分析提示词

---

## 🎯 使用说明

本模板集提供各类投资场景的AI提示词，直接复制使用或根据需要修改。

---

## 📋 模板一：公司基本面分析

```
【公司分析提示词】

请帮我分析[公司名/代码]：

一、业务概述（用简单语言）
1. 核心业务是什么？用一句话概括
2. 收入来源有哪些？占比如何？
3. 在产业链中的位置是什么？

二、竞争优势
1. 这家公司的护城河是什么？
2. 与竞争对手相比有什么独特之处？
3. 这种优势是否可持续？

三、风险因素
1. 最大的经营风险是什么？
2. 行业层面的风险有哪些？
3. 管理层有什么值得关注的问题吗？

四、投资要点
1. 简述看多的理由（3条）
2. 简述看空的理由（3条）

请用投资新手也能理解的语言回答。
```

## 📋 模板二：研报解读

```
【研报解读提示词】

请帮我解读以下研报内容：

[粘贴研报内容]

请分析：

1. 核心结论（3句话总结）
   - 分析师的观点是什么？
   - 目标价和评级是什么？

2. 关键假设
   - 支撑这个结论的核心假设有哪些？
   - 这些假设的可靠性如何？

3. 风险检视
   - 研报提到了哪些风险？
   - 有哪些研报没提但你认为重要的风险？

4. 信息质量评估
   - 这份研报的论证逻辑是否严谨？
   - 有没有明显的乐观/悲观偏差？

5. 我应该如何使用这份研报？
```

## 📋 模板三：新闻解读

```
【新闻解读提示词】

请帮我解读以下新闻/信息：

[粘贴新闻内容]

分析要求：

1. 信息价值评估
   - 这是新信息还是已知信息的重复？
   - 信息来源可靠吗？
   - 是否可能是有意释放的利好/利空？

2. 影响分析
   - 这对哪些公司/行业是利好？
   - 这对哪些公司/行业是利空？
   - 影响程度如何（大/中/小/噪音）？

3. 行动建议
   - 我需要立即采取行动吗？
   - 还是这只是需要关注的信息点？
   - 建议如何应对？

4. 后续关注
   - 未来需要关注哪些相关动态？
```

## 📋 模板四：思维对手盘

```
【思维对手盘提示词】

我正在考虑投资[公司名/标的]

我的投资逻辑：
1. [你的理由1]
2. [你的理由2]
3. [你的理由3]

请你扮演一位专业的做空投资者，尽可能挑战我的观点：

1. 逻辑漏洞
   - 我的哪个论点最薄弱？
   - 我的逻辑链有什么可能的断裂点？

2. 被忽视的风险
   - 我可能忽略了什么重大风险？
   - 有什么"房间里的大象"我没看到？

3. 反面证据
   - 有什么证据可以推翻我的判断？
   - 哪些指标或事件的出现会证明我错了？

4. 最坏情景
   - 如果这笔投资失败，最可能是因为什么？
   - 最坏情况下我会损失多少？

5. 魔鬼代言
   - 如果你要做空这只股票，你的核心论点是什么？

请尽量尖锐和直接，不要客套。
```

## 📋 模板五：投资概念学习

```
【概念学习提示词】

请用费曼技巧帮我理解[投资概念]：

1. 简单定义
   用一句最简单的话定义这个概念

2. 日常类比
   给一个日常生活中的类比，让完全不懂投资的人也能理解

3. 投资应用
   这个概念在实际投资中如何应用？
   给出1-2个具体场景

4. 常见误解
   关于这个概念，人们最常犯的错误理解是什么？

5. 核心要点
   记住这个概念，我最需要记住的3个要点是什么？

6. 进一步学习
   如果我想深入学习，应该了解哪些相关概念？
```

## 📋 模板六：财务数据解读

```
【财务数据解读提示词】

请帮我解读以下财务数据：

公司：[公司名]
数据：
- 营收：[数据]
- 净利润：[数据]
- ROE：[数据]
- 负债率：[数据]
- 现金流：[数据]
- PE/PB：[数据]

分析要求：

1. 关键指标解读
   - 这些数据整体反映了什么？
   - 有哪些指标表现好？哪些有问题？

2. 趋势分析
   - 与过去相比，公司是在变好还是变差？
   - 增长是否可持续？

3. 对比分析
   - 这些数据在行业中处于什么水平？
   - 与主要竞争对手相比如何？

4. 红旗警示
   - 有没有需要特别注意的财务风险信号？
   - 利润和现金流是否匹配？

5. 估值判断
   - 当前估值是高估、合理还是低估？
   - 判断依据是什么？
```

## 📋 模板七：投资决策检验

```
【投资决策检验提示词】

我即将执行以下投资决策，请帮我做最后检验：

交易：[买入/卖出][标的]
仓位：[比例]%
理由：[你的理由]

请检验：

1. 逻辑检验
   - 这个决策的逻辑是否清晰完整？
   - 有没有情绪因素在影响这个决策？

2. 规则检验
   - 这个决策是否符合以下我的投资规则：
     [列出你的关键规则]
   - 有没有违反任何规则？

3. 风险检验
   - 如果这笔交易亏损，最大可能亏多少？
   - 这个风险是否在我的承受范围内？

4. 必要性检验
   - 我必须现在执行吗？还是可以等待？
   - 不执行这个决策会有什么后果？

5. 最终问题
   - 如果24小时后再决定，我可能会有不同选择吗？
```

## 📋 模板八：每周复盘辅助

```
【周复盘AI辅助提示词】

这是我本周的交易记录和持仓情况：

[粘贴你的周记录]

请帮我进行复盘分析：

1. 决策质量评估
   - 本周哪些决策做得好？为什么？
   - 本周哪些决策可以改进？如何改进？

2. 规则执行评估
   - 我是否严格按照规则执行？
   - 有没有出现情绪化操作？

3. 认知偏差检查
   - 回顾这周，我可能受到了哪些认知偏差的影响？
   - 有没有确认偏差、锚定效应等问题？

4. 改进建议
   - 基于这周的表现，你建议我下周注意什么？
   - 有没有需要调整的规则？

5. 情绪状态评估
   - 从记录来看，我的情绪状态健康吗？
   - 有没有过度焦虑或过度自信的迹象？
```

│  • 最终判断和责任在你                      │

│  "这只股票会涨吗？"                       │
