# EP08 - 风险容忍度自测工具

> 完整的风险评估问卷与分析

---

## 🎯 用前须知

**核心理念**：
> "知道自己能承受多少风险，比知道应该承受多少更重要。"

**本工具包含**：
1. 客观能力评估（财务维度）
2. 主观意愿评估（心理维度）
3. 综合风险等级判定
4. 配置建议

---

## 📋 工具3：综合风险等级判定

### 取两者较小值

```
我的客观能力等级：________
我的主观意愿等级：________

综合风险容忍度 = min(客观, 主观) = ________
```

### 风险等级对应配置

| 综合等级 | 建议股票占比 | 典型配置 |
|----------|--------------|----------|
| **高** | 70-90% | 股票80% + 债券20% |
| **较高** | 50-70% | 股票60% + 债券40% |
| **中等** | 30-50% | 股票40% + 债券60% |
| **较低** | 10-30% | 股票20% + 债券80% |

## 💡 使用建议

1. **诚实回答**：欺骗问卷就是欺骗自己
2. **参考历史行为**：实际反应比假设性回答更真实
3. **保守原则**：不确定时选择较保守的选项
4. **预留buffer**：实际配置比测评结果再保守一些
5. **定期更新**：人会变，风险偏好也会变
