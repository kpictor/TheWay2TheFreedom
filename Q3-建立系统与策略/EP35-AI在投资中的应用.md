## 📌 本期核心问题

**让AI做信息处理，你做判断。如何正确使用AI辅助投资？**

---

## 💡 第一性原理拆解

### AI能做什么

| 能做 | 不能做 |
|------|--------|
| 信息整理和归纳 | 预测价格 |
| 数据分析 | 替你决策 |
| 研报解读 | 保证赚钱 |
| 概念解释 | 内幕消息 |
| 逻辑检验 | 情绪管理 |

### AI的正确使用方式

```
错误：让AI告诉我买什么
正确：让AI帮我分析这家公司的基本情况

错误：让AI预测明天涨跌
正确：让AI帮我理解市场结构和逻辑

错误：完全依赖AI的结论
正确：用AI作为思维对手盘，挑战自己的观点
```

## ✍️ 费曼作业

### 任务：用AI完成一次研报分析

1. 找一份你关注公司的研报
2. 让AI帮你总结要点
3. 让AI指出可能的风险点
4. 让AI挑战乐观预期
5. 形成你自己的判断
