## 开场（0:00-1:30）

大家好，欢迎来到《通往自由之路》第30期。

今天我们要解决一个关键问题：**什么条件下买入？**

你有没有过这样的经历：
- 别人说好就买了，结果亏钱
- 感觉要涨就买了，结果下跌
- 买的时候没想清楚，卖的时候更纠结

问题在于：**你没有明确的入场规则。**

今天我们就来学习如何设计清晰、可执行的入场规则。

---

## 第一部分：好的入场规则标准（1:30-5:00）

### 三大特征

一个好的入场规则必须具备：

**1. 明确（Specific）**
```
❌ 模糊规则："股价便宜时买入"
✓ 明确规则："PE低于15时买入"

如何判断是否明确：
把规则给别人看，他能准确判断是否满足
```

**2. 可验证（Verifiable）**
```
❌ 不可验证："感觉公司不错"
✓ 可验证："ROE连续3年>15%"

如何判断是否可验证：
能用历史数据回测这个规则的效果
```

**3. 可执行（Actionable）**
```
❌ 难执行："等待最佳时机"
✓ 可执行："满足条件A+B+C时买入"

如何判断是否可执行：
按规则操作，不需要额外判断
```

### 规则设计的框架

```
入场规则 = 筛选条件 + 触发条件 + 排除条件

筛选条件：什么样的标的可以进入候选池
触发条件：什么时候实际买入
排除条件：什么情况下不买
```

## 第二部分：基本面入场规则（5:00-8:30）

### 价值投资维度

**估值指标**：

| 指标 | 规则示例 | 逻辑 |
|------|---------|------|
| PE（市盈率）| PE < 15 或 PE < 行业均值 | 买入相对便宜的股票 |
| PB（市净率）| PB < 1.5 | 资产打折买入 |
| 股息率 | 股息率 > 3% | 稳定收益保障 |
| PEG | PEG < 1 | 成长被低估 |

**质量指标**：

| 指标 | 规则示例 | 逻辑 |
|------|---------|------|
| ROE | ROE连续3年>15% | 资本回报能力 |
| 净利润增长 | 连续3年增长>10% | 成长性 |
| 负债率 | 资产负债率<60% | 财务稳健 |
| 现金流 | 经营现金流>净利润 | 盈利质量 |

### 价值投资入场规则示例

```
巴菲特式入场规则：

筛选条件：
□ 业务我能理解（能力圈内）
□ 具有持久竞争优势（护城河）
□ 管理层诚实能干

触发条件：
□ 估值具有安全边际（PE<历史中位数的80%）
□ 或买入价<内在价值的70%

排除条件：
✗ 高杠杆行业
✗ 快速变化的行业
✗ 无法理解的业务
```

## 第三部分：技术面入场规则（8:30-11:00）

### 趋势跟踪维度

**常用技术指标**：

| 指标 | 规则示例 | 逻辑 |
|------|---------|------|
| 均线 | 价格突破20日均线 | 趋势开始 |
| 新高 | 突破52周新高 | 强势确认 |
| MACD | 金叉买入 | 动量转多 |
| RSI | RSI<30时买入 | 超卖反弹 |

### 趋势跟踪入场规则示例

```
海龟交易法入场规则：

筛选条件：
□ 流动性足够（日成交量>100万）
□ 有期货市场（可做空）

触发条件（二选一）：
□ 系统1：突破20日高点买入
□ 系统2：突破55日高点买入

排除条件：
✗ 上一次突破信号盈利（系统1）
✗ 相关性过高的品种
```

### 技术面规则的注意

```
技术指标的局限：
1. 无法预测，只能跟随
2. 有滞后性
3. 假突破常见

应对方法：
- 结合多个指标
- 等待确认
- 配合仓位管理
```

## 第四部分：综合入场规则设计（11:00-14:00）

### 多维度结合

**层层筛选**：
```
第一层：宏观环境
"市场情绪不在极端狂热阶段"

第二层：行业选择
"行业处于成长或成熟期"

第三层：个股筛选
"满足基本面筛选条件"

第四层：入场时机
"满足估值或技术触发条件"
```

### 实例：综合入场规则

```
我的入场规则示例：

【筛选条件】（必须全部满足）
□ 行业在我的能力圈内
□ 公司市值>100亿（避免小盘股风险）
□ ROE连续3年>12%
□ 负债率<50%
□ 经营现金流为正

【触发条件】（满足任一）
□ PE低于过去5年均值的80%
□ 股价回撤30%以上，基本面无变化
□ 市场情绪处于恐惧期（参考EP23）

【排除条件】（任一项取消资格）
✗ 最近1年有财务造假嫌疑
✗ 大股东大幅减持
✗ 核心业务逻辑被破坏
✗ 我无法理解的变化
```

### 入场决策流程

```
发现一只股票
    ↓
检查筛选条件 → 不满足 → 放弃
    ↓
全部满足
    ↓
检查排除条件 → 触发 → 放弃
    ↓
无排除项
    ↓
等待触发条件 → 未触发 → 加入观察清单
    ↓
条件触发
    ↓
计算仓位（EP29）
    ↓
执行买入
```

### 常见错误

**错误1：规则太多**
```
症状：条件多到几乎没有股票满足
问题：永远无法入场
建议：必须条件3-5个即可
```

**错误2：规则太少**
```
症状：只有一两个条件
问题：筛选不够，风险高
建议：至少覆盖估值、质量、时机
```

**错误3：规则相互矛盾**
```
症状：PE<10且PEG<1（低估值+高成长少见）
问题：几乎不可能同时满足
建议：检查规则逻辑一致性
```

## 结尾（14:00-15:00）

今天我们学习了入场规则设计：

**好规则特征**：
- 明确、可验证、可执行

**规则框架**：
- 筛选条件 + 触发条件 + 排除条件

**设计方法**：
- 基本面：估值 + 质量
- 技术面：趋势 + 动量
- 综合：多维度层层筛选

**费曼作业**：
写出你的3条入场规则，确保它们是明确、可验证、可执行的。
