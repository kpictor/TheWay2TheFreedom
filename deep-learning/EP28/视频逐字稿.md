> "资产配置是金融领域唯一的免费午餐。"

什么意思？意味着通过合理的资产配置，你可以**在不降低预期收益的情况下降低风险**，或者**在不增加风险的情况下提高收益**。

## 第一部分：马科维茨的发现（1:30-5:00）

### 现代投资组合理论

1952年，哈里·马科维茨发表论文《Portfolio Selection》，开创了现代投资组合理论。

**核心发现**：
> 把不完全相关的资产组合在一起，总体风险可以低于各资产单独风险的加权平均。

### 相关性的概念

**相关系数**：衡量两个资产价格走势的关联程度

| 相关系数 | 含义 | 举例 |
|---------|------|------|
| +1 | 完全正相关，同涨同跌 | 同行业两只股票 |
| 0 | 完全不相关 | 股票和天气 |
| -1 | 完全负相关，一涨一跌 | 做多和做空 |

### 分散的魔力

**例子**：两只股票，各50%仓位

```
情况1：相关性 = +1（完全相关）
股票A涨10% → 股票B也涨10% → 组合涨10%
股票A跌10% → 股票B也跌10% → 组合跌10%
分散效果：零

情况2：相关性 = 0（不相关）
股票A涨10%，股票B可能涨、跌或不变
平均来看，波动被平滑
分散效果：好

情况3：相关性 = -1（完全负相关）
股票A涨10% → 股票B跌10% → 组合不变
理论上可以完全消除波动
分散效果：完美（但现实中很难找到）
```

### 有效前沿

```
        收益率 ↑
              │           ★最优组合
              │        ●
              │      ●●●
              │    ●●●●
              │   ●●●●  ← 有效前沿
              │  ●●●
              │ ●●
              │●
              └──────────→ 风险

有效前沿：给定风险水平下收益最高的组合
或给定收益水平下风险最低的组合
```

## 第二部分：主要资产类别（5:00-9:00）

### 大类资产分类

| 大类 | 子类 | 特点 |
|------|------|------|
| 股票 | 国内股票、发达市场股票、新兴市场股票 | 高收益、高风险 |
| 债券 | 国债、企业债、高收益债 | 低风险、稳定收入 |
| 现金 | 货币基金、存款 | 最低风险、低收益 |
| 另类 | 黄金、房产、大宗商品 | 与传统资产相关性低 |

### 主要资产的历史相关性

```
相关性矩阵（示意）：
           国内股票  发达市场  新兴市场  国债    黄金
国内股票      1.0      0.6      0.7     0.1    0.0
发达市场      0.6      1.0      0.8     0.0   -0.1
新兴市场      0.7      0.8      1.0     0.1    0.2
国债          0.1      0.0      0.1     1.0    0.2
黄金          0.0     -0.1      0.2     0.2    1.0
```

**关键观察**：
- 股票之间相关性较高（分散效果有限）
- 股票和债券相关性低（好的配对）
- 黄金与其他资产相关性低（分散工具）

### 股债搭配的逻辑

**为什么股票和债券是经典搭配？**

```
经济扩张期：
- 企业盈利好 → 股票涨
- 通胀预期升 → 债券表现一般

经济衰退期：
- 企业盈利差 → 股票跌
- 降息预期 → 债券涨

两者互补，平滑整体波动
```

## 第三部分：经典配置策略（9:00-12:00）

### 60/40组合

**最经典的资产配置**：
- 60%股票
- 40%债券

```
历史表现（美国市场）：
- 年化收益：约8-9%
- 最大回撤：约30-35%
- 夏普比率：约0.5-0.6

对比100%股票：
- 年化收益：约10%
- 最大回撤：约50%+
- 夏普比率：约0.4
```

**60/40的优势**：用较小的收益牺牲换取大幅的风险降低。

### 年龄法则

**简单规则**：100 - 年龄 = 股票比例

| 年龄 | 股票 | 债券 | 理由 |
|------|------|------|------|
| 25岁 | 75% | 25% | 时间长，能承受波动 |
| 40岁 | 60% | 40% | 平衡期 |
| 55岁 | 45% | 55% | 接近退休，保守 |
| 70岁 | 30% | 70% | 需要稳定 |

### 全天候策略

**达利欧的设计**：

```
目标：在任何经济环境下都有稳定表现

配置：
- 股票：30%
- 长期国债：40%
- 中期国债：15%
- 大宗商品：7.5%
- 黄金：7.5%

特点：
- 波动较低
- 收益中等
- 任何单一环境都不会太差
```

### 核心+卫星策略

```
核心（70-80%）：
- 低成本指数基金
- 提供市场平均回报
- 被动管理

卫星（20-30%）：
- 主动选择的个股或基金
- 尝试超额回报
- 可承受更高风险
```

## 第四部分：实践应用（12:00-14:30）

### 设计你的配置

**步骤1：确定风险承受能力**
```
回顾EP08的风险测评：
保守型 → 股票20-40%
稳健型 → 股票40-60%
积极型 → 股票60-80%
激进型 → 股票80-100%
```

**步骤2：选择资产类别**
```
初学者建议从简单开始：
- 国内股票指数
- 国债/货币基金
- 可选：一点黄金

进阶可加入：
- 海外市场
- 不同行业/因子
```

**步骤3：确定比例**
```
示例配置（稳健型）：
- 沪深300 ETF：30%
- 标普500 ETF：20%
- 债券基金：30%
- 货币基金：15%
- 黄金ETF：5%
```

**步骤4：定期再平衡**
```
再平衡的意义：
- 买低卖高的自动机制
- 保持预设的风险水平
- 纪律化操作

频率建议：
- 每年1-2次
- 或偏离目标5-10%时
```

### 再平衡示例

```
初始配置：股票60%，债券40%
一年后：股票涨了，变成70%，债券30%

再平衡操作：
卖出部分股票，买入债券
回到股票60%，债券40%

效果：
自动"卖高买低"
```

### 常见误区

**误区1：分散就是买很多只股票**
```
错误：买10只银行股
正确：买不同行业、不同国家的资产
关键是相关性低，不是数量多
```

**误区2：追求完美配置**
```
不存在"最佳"配置
适合你的风险偏好和生活阶段才是好配置
简单配置执行>复杂配置放弃
```

**误区3：忽视再平衡**
```
配置完就不管了
久而久之偏离目标
需要定期维护
```
