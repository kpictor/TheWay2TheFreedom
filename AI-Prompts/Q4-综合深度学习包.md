## 实战阶段核心提示词

### EP40-42 - 实盘与日志

```markdown
你是我的交易日志助手。

每次我描述一笔交易，请帮我：
1. 结构化记录（时间、标的、价格、理由）
2. 评估是否符合系统规则
3. 识别情绪因素（1-10打分）
4. 追问可能遗漏的细节
5. 周末时帮我总结周复盘

开始记录今天的交易：[等待对话]
```

### EP43-44 - 处理亏损与盈利

```markdown
我正在经历连续亏损/盈利，请帮我保持理性：

当前情况：[描述]

请帮我：
1. 分析情绪状态是否正常
2. 评估是否需要暂停交易
3. 检查是否有违反规则的迹象
4. 给出调整建议
5. 必要时扮演"冷静的声音"

直接开始评估。
```

### EP45-46 - 异常事件与错误学习

```markdown
帮我进行Dalio式错误分析：

错误描述：[我会提供]

请按以下框架分析：
1. 发生了什么（事实）
2. 5次为什么（根因分析）
3. 这个错误教会我什么
4. 需要修改哪条规则
5. 如何防止复发
```

### EP47-48 - 系统升级与原则

```markdown
帮我迭代投资系统：

最近的发现/问题：[我会描述]

请帮我：
1. 评估是否需要升级系统
2. 确保变更有充分理由
3. 设计变更内容
4. 记录变更日志
5. 提炼新的投资原则
```

### EP49-50 - AI协作与社区

```markdown
扮演我的投资决策对手盘：

我正在考虑：[描述投资决策]

请：
1. 用5个尖锐问题挑战我
2. 指出我可能的盲点
3. 提供反方观点
4. 用我的原则评估这个决策
5. 给出综合建议（不是买卖建议）
```

### EP51-52 - 年度复盘与定义

```markdown
帮我进行年度投资学习总结：

请引导我回答：
1. 一年前的自己vs现在的自己
2. 最大的认知改变是什么
3. 最有价值的教训是什么
4. 投资系统经历了几次迭代
5. 明年的目标是什么
6. 如何定义"合格投资者"

最后，帮我写一封给一年前自己的信。
```
