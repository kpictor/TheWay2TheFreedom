## 📚 模块一：概念深化

### Prompt 1.1 - 现代投资组合理论

```markdown
帮我理解马科维茨现代投资组合理论的核心：

**核心发现**
> "把不完全相关的资产组合在一起，可以在不降低预期收益的情况下降低风险"

**关键概念**
1. **预期收益**：加权平均收益
2. **组合风险**：不是简单加权，取决于相关性
3. **相关系数**：-1到+1之间

**相关性的力量**
| 相关性 | 组合效果 | 示例 |
|--------|----------|------|
| +1 | 无分散效果 | 两只同行业股票 |
| 0 | 良好分散 | 股票和债券 |
| -1 | 风险可消除 | 理论情况 |

用数字例子说明相关性如何影响组合风险。
```

### Prompt 1.2 - 有效前沿

```markdown
解释"有效前沿"概念：

**定义**
- 在给定风险下，收益最高的组合
- 在给定收益下，风险最低的组合

**投资意义**
1. 不在有效前沿上的组合是"低效"的
2. 应该寻找在有效前沿上的组合
3. 在前沿上选择哪个点取决于风险偏好

**实践困难**
- 预期收益难以估计
- 相关性会变化
- 模型假设与现实差异

这些困难如何影响实际配置？
```
