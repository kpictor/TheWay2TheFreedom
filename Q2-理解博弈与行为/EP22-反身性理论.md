## 💡 第一性原理拆解

### 反身性的核心

> 市场参与者的预期不仅反映现实，还会**改变**现实。

```
传统金融学：现实 → 预期 → 价格
反身性理论：现实 ⟷ 预期 ⟷ 价格（双向影响）
```

### 正反馈循环

**股价上涨→更乐观→更多买入→股价继续涨→...**

**银行案例**：
```
银行健康 → 大家信任 → 不挤兑 → 银行真的健康
银行健康 → 谣言不信任 → 挤兑 → 银行真的倒闭
```

### 反身性的三个阶段

| 阶段 | 特征 | 操作 |
|------|------|------|
| 早期 | 少数人发现偏差 | 顺势而为 |
| 加速期 | 反馈强化，偏离加大 | 谨慎，设止损 |
| 反转期 | 偏离极端，回归开始 | 逆向思考 |
