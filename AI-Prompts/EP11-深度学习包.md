## 📚 模块一：概念深化

### Prompt 1.1 - 成本全景图

```markdown
帮我建立投资成本的完整认知：

**显性成本**
1. 交易佣金：每次交易___%
2. 印花税：卖出时___%
3. 基金管理费：每年___%
4. 托管费：每年___%
5. 认购/赎回费：___%

**隐性成本**
1. 价差成本（买一卖一的差价）
2. 冲击成本（大额交易对价格的影响）  
3. 机会成本（资金闲置的损失）
4. 时间成本（研究和操作的时间）

请帮我估算我过去一年的总成本。
```

### Prompt 1.2 - 成本对复利的侵蚀

```markdown
用复利思维分析成本的长期影响：

**计算模拟**
假设本金10万，投资20年：
- 情景A：年化8%，0成本
- 情景B：年化8%，每年成本1%（实际7%）
- 情景C：年化8%，每年成本2%（实际6%）

请计算并展示：
1. 20年后三者差距多大？
2. 2%成本看起来不多，但累计损失多少？
3. 如果每月还有交易成本呢？

用数字震撼我。
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 个人成本审计

```markdown
请引导我进行个人投资成本审计：

需要我提供的信息：
1. 过去一年交易次数
2. 平均每次交易金额
3. 券商佣金率
4. 持有的基金及其管理费
5. 账户平均资金规模

我会提供数据，请帮我计算：
- 显性成本总额
- 占账户的百分比
- 与"持有指数不动"的差距
- 降低成本的具体建议

开始审计：[等待数据]
```

### Prompt 3.2 - 交易频率优化

```markdown
帮我分析交易频率与成本的关系：

**思维实验**
假设我一年交易50次，平均成本0.2%/次：
- 年化交易成本 = 50 × 0.2% = 10%！

问我：
1. 你过去一年交易了多少次？
2. 每笔交易真的必要吗？
3. 有多少交易是情绪驱动的？
4. 如果减少一半交易次数，省多少钱？
5. 你愿意设定"最大月度交易次数"吗？
```
