## 📋 原则提炼指南

### 如何从经验中提炼原则

```
原则提炼四步法
━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一步：回顾经历
回想一次重要的投资经历（成功或失败）
_________________________________

第二步：提取教训
这次经历告诉我什么？
_________________________________

第三步：形成原则
把教训转化为可执行的原则
"当______时，我应该______"
_________________________________

第四步：验证
这个原则在其他情况下也适用吗？
□ 是，可以作为原则
□ 否，只是特例
```

## 📋 原则参考库

### 常见投资原则参考

```
【关于市场】
• 市场短期是投票机，长期是称重机
• 没有人能持续准确预测市场
• 极端情况终将回归
• 市场可以保持非理性比你保持偿付能力更久

【关于自己】
• 我最大的敌人是自己的情绪
• 我在[具体情况]时容易冲动
• 我倾向于高估自己的判断
• 当我感到极度自信时，往往是危险时刻

【关于决策】
• 在冷静时制定规则，在任何时候执行
• 不确定时，选择不行动
• 简单的策略比复杂的更容易坚持
• 每次决策前问：这符合我的规则吗？

【禁止事项】
• 绝不在情绪激动时做决策
• 绝不借钱投资
• 绝不把所有资金押在一个标的上
• 绝不忽视止损
```

## 📋 原则自检

```
原则质量检查
━━━━━━━━━━━━━━━━━━━━━━━━━━━

对每条原则检查：

□ 来自真实经验？（不是抄来的）
□ 足够清晰？（不模糊）
□ 可以执行？（能直接指导行动）
□ 经过检验？（用过且有效）

如果有一项不满足，考虑：
• 修改使其更清晰可执行
• 或暂时标记为"待验证"
```
