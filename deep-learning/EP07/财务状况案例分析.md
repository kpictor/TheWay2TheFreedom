## 📜 案例1：职场新人小王（25岁）

### 基本情况

```
职业：互联网公司产品经理
月收入：15,000元（税后）
工作年限：2年
家庭状态：单身，租房
```

### 财务状况

**资产负债表**

| 项目 | 金额 |
|------|------|
| 活期存款 | 30,000元 |
| 货币基金 | 20,000元 |
| 股票基金 | 50,000元 |
| 定期存款 | 0 |
| **总资产** | 100,000元 |
| 负债 | 0 |
| **净资产** | 100,000元 |

**月度现金流**

| 项目 | 金额 |
|------|------|
| 月收入 | 15,000元 |
| 房租 | 4,000元 |
| 生活费 | 4,000元 |
| 娱乐社交 | 2,000元 |
| 其他 | 1,000元 |
| **月结余** | 4,000元 |
| **储蓄率** | 26.7% |

### 财务诊断

| 指标 | 数值 | 评级 |
|------|------|------|
| 储蓄率 | 26.7% | ✅ 健康 |
| 负债率 | 0% | ✅ 健康 |
| 紧急储备 | 5个月 | ⚠️ 需加强 |
| 投资占比 | 50% | ✅ 合理 |

### 建议

1. **优先**：将紧急备用金提升到6个月（约6万元）
2. **保险**：配置医疗险+意外险（年费约500元）
3. **投资**：年轻可承受风险，但不要all in股票
4. **潜力**：人力资本是最大资产，投资学习技能

## 📜 案例2：中年白领老张（38岁）

### 基本情况

```
职业：外企中层经理
月收入：35,000元（税后）
家庭状态：已婚，一孩，有房贷
配偶月收入：15,000元
```

### 财务状况

**资产负债表**

| 项目 | 金额 |
|------|------|
| 流动资产 | 200,000元 |
| 股票基金 | 500,000元 |
| 定期理财 | 300,000元 |
| 自住房（估值）| 4,000,000元 |
| 车辆 | 150,000元 |
| **总资产** | 5,150,000元 |
| 房贷余额 | 2,000,000元 |
| 车贷余额 | 80,000元 |
| **总负债** | 2,080,000元 |
| **净资产** | 3,070,000元 |

**月度现金流**

| 项目 | 金额 |
|------|------|
| 家庭月收入 | 50,000元 |
| 房贷月供 | 12,000元 |
| 车贷月供 | 3,000元 |
| 子女教育 | 5,000元 |
| 生活费 | 10,000元 |
| 保险费 | 2,000元 |
| 其他支出 | 5,000元 |
| **月结余** | 13,000元 |
| **储蓄率** | 26% |

### 财务诊断

| 指标 | 数值 | 评级 |
|------|------|------|
| 储蓄率 | 26% | ✅ 健康 |
| 负债率 | 40.4% | ⚠️ 偏高 |
| 偿债比 | 30% | ✅ 健康 |
| 紧急储备 | 5.4个月 | ⚠️ 勉强 |
| 投资占比 | 16%（不含房）| ⚠️ 偏低 |

### 建议

1. **风险**：过于集中在房产，需增加金融资产
2. **紧急备用**：提升到40万（覆盖失业+房贷）
3. **保险**：作为家庭支柱，寿险和重疾险必备
4. **投资**：减少定期理财，增加权益类资产
5. **提前还贷**：车贷利率高，优先还清

## 📜 案例3：创业者李姐（45岁）

### 基本情况

```
职业：小企业主
月收入：波动大，平均80,000元
家庭状态：已婚，二孩，有房无贷
```

### 财务状况

**资产负债表**

| 项目 | 金额 |
|------|------|
| 流动资产 | 500,000元 |
| 股票基金 | 2,000,000元 |
| 企业股权（估值）| 3,000,000元 |
| 自住房 | 6,000,000元 |
| 商铺（投资）| 2,000,000元 |
| **总资产** | 13,500,000元 |
| 负债 | 0 |
| **净资产** | 13,500,000元 |

### 财务诊断

| 指标 | 数值 | 评级 |
|------|------|------|
| 储蓄率 | 50%+ | ✅ 优秀 |
| 负债率 | 0% | ✅ 优秀 |
| 紧急储备 | 10个月+ | ✅ 优秀 |
| **但是...** | | |
| 资产集中度 | 企业占22% | ⚠️ 风险 |
| 流动性 | 商铺+企业难变现 | ⚠️ 风险 |

### 问题与建议

**核心问题**：
> 表面看资产雄厚，但企业风险可能威胁全部家庭财务

1. **企业与家庭分离**：设立有限责任，保护个人资产
2. **增加流动性**：减少商铺，增加金融资产
3. **分散风险**：不要把全部身家押在企业上
4. **保险**：企业主更需要保障，包括关键人险
5. **规划退出**：考虑企业何时、如何变现

## 📜 案例4：临退休夫妇（55岁）

### 基本情况

```
职业：国企退休在即
家庭月收入：30,000元
预期退休金：家庭月8,000元
子女已独立
```

### 财务状况

**资产负债表**

| 项目 | 金额 |
|------|------|
| 流动资产 | 800,000元 |
| 股票基金 | 1,500,000元 |
| 债券/理财 | 700,000元 |
| 自住房 | 3,000,000元 |
| **总资产** | 6,000,000元 |
| 负债 | 0 |
| **净资产** | 6,000,000元 |

### 退休规划挑战

```
退休后的现金流平衡：

预期月支出：15,000元
- 退休金：8,000元
─────────────────
= 缺口：7,000元/月 = 84,000元/年

假设退休生活25年：
需要金融资产产生：84,000 × 25 = 2,100,000元+
考虑通胀和医疗，实际需求更高
```

### 建议

1. **降低风险**：逐步减少股票占比，增加债券
2. **现金流规划**：用年金或定期分红产品创造稳定现金流
3. **保留应急**：老年医疗支出不可预测，保留更多流动性
4. **房产考虑**：如需要，可考虑以房养老方案
5. **遗产规划**：提前安排，避免未来纠纷

## 💡 案例总结

### 生命周期规律

| 阶段 | 主要特点 | 财务重点 |
|------|----------|----------|
| 25岁 | 收入低、风险承受高 | 储蓄习惯、人力资本 |
| 38岁 | 收入高、支出也高 | 平衡增长与保障 |
| 45岁 | 积累增多、风险意识 | 分散、保护已有 |
| 55岁 | 退休在即 | 创造现金流、降风险 |

### 通用教训

1. **每个阶段需求不同**：投资策略应随人生阶段调整
2. **风险意识逐渐增强**：年龄越大，越不能承受大损失
3. **现金流管理永远重要**：收入-支出=决定一切的基础
4. **提前规划**：不要等到需要时才开始准备
