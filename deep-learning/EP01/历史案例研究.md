# EP01 - 历史案例研究

> 3个历史投资案例：成功、失败、争议

---

## 📈 案例1：成功案例 - 巴菲特的华盛顿邮报投资

### 背景
**时间**：1973年

**市场环境**：
- 石油危机导致股市暴跌
- 华盛顿邮报股价从高点下跌近50%
- 市场极度恐慌，大量投资者抛售

**华盛顿邮报情况**：
- 刚经历水门事件报道，声誉卓著
- 内在价值被严重低估
- 股价约为内在价值的25%

### 决策时刻

**巴菲特面临的不确定性**：
- 股市会继续下跌多久？
- 报纸行业会不会衰落？
- 水门事件会不会反噬公司？
- 何时能够回本？

**巴菲特的思考过程**：
> "我用四个标准评估：(1)业务我理解吗？(2)有持久竞争优势吗？(3)管理层诚实能干吗？(4)价格合理吗？"

答案全是肯定的，而且价格远低于价值。

### 结果

| 指标 | 数值 |
|------|------|
| 初始投资 | 1060万美元 |
| 持有时间 | 数十年 |
| 最终价值 | 超过10亿美元 |
| 回报倍数 | 100倍+ |

### 教训

1. **在恐慌中保持理性**：当别人恐惧时贪婪
2. **关注内在价值**：股价下跌不等于价值下降
3. **长期持有**：复利需要时间发挥作用
4. **能力圈**：只投资自己理解的业务

## 📉 案例2：失败案例 - 长期资本管理公司(LTCM)崩盘

### 背景
**时间**：1994-1998年

**团队**：
- 两位诺贝尔经济学奖得主
- 前美联储副主席
- 华尔街最聪明的头脑

**策略**：
- 利用数学模型发现定价偏差
- 大规模杠杆放大收益
- "科学化"消除不确定性

**前期业绩**：
- 1995年回报 43%
- 1996年回报 41%
- 1997年回报 17%

### 决策时刻

**他们面临的不确定性**：
- 模型假设会不会失效？
- 极端事件会不会发生？
- 杠杆会不会反噬？

**他们的思考错误**：
> "我们的模型基于历史数据，显示这种亏损的概率是10的几十次方分之一，不可能发生。"

### 1998年俄罗斯债务危机

**实际发生了什么**：
- 俄罗斯宣布债务违约（模型认为概率极低）
- 全球市场恐慌，流动性枯竭
- 所有资产同时下跌（相关性飙升到1）
- LTCM持仓无法平仓

**结果**：
- 4个月亏损46亿美元
- 杠杆率一度高达25:1
- 濒临破产，美联储被迫组织救援
- 几乎引发系统性金融危机

### 教训

1. **模型不是现实**：历史数据不能预测未来
2. **极端事件必然发生**：只是不知道何时
3. **杠杆是双刃剑**：放大收益也放大亏损
4. **相关性会变**：危机时资产往往同步下跌
5. **聪明人也会犯愚蠢的错误**

## ⚖️ 案例3：争议案例 - 索罗斯狙击英镑

### 背景
**时间**：1992年

**市场环境**：
- 欧洲汇率机制(ERM)要求成员国货币锁定汇率
- 英国经济疲软，但被迫维持高汇率
- 英镑明显高估

### 决策时刻

**索罗斯面临的不确定性**：
- 英国政府会不会死撑到底？
- 有多少外汇储备可以动用？
- 其他投机者会不会配合？
- 时机何时成熟？

**索罗斯的思考**：
> "这是不对称的赌注。如果我错了，损失有限；如果我对了，收益巨大。"

### 黑色星期三（1992年9月16日）

**事件经过**：
1. 索罗斯量子基金做空100亿英镑
2. 英国央行疯狂买入英镑护盘
3. 一天内加息两次（10%→12%→15%）
4. 最终放弃，英镑贬值15%

**结果**：
- 索罗斯一天获利10亿美元
- 英国纳税人损失超过30亿英镑

### 争议

| 视角 | 评价 |
|------|------|
| **投资者视角** | 天才的宏观判断，风险回报极佳 |
| **英国民众视角** | 投机者掠夺国家财富 |
| **经济学家视角** | 揭示了固定汇率的内在矛盾 |
| **道德层面** | 合法但有争议的财富转移 |

### 复杂的教训

1. **结果好不代表过程对**：高风险策略可能恰好成功
2. **过程对不代表结果好**：正确的分析也可能失败
3. **不对称风险**：寻找有限下行、巨大上行的机会
4. **系统脆弱性**：寻找不可持续的市场结构
5. **伦理灰色地带**：合法≠道德

---

## 💡 三个案例的对比总结

| 维度 | 巴菲特（成功） | LTCM（失败） | 索罗斯（争议） |
|------|----------------|--------------|----------------|
| **时间框架** | 长期无限 | 短期依赖 | 事件驱动 |
| **杠杆使用** | 极低 | 极高 | 高 |
| **对不确定性态度** | 承认并利用 | 试图消除 | 承认并博弈 |
| **安全边际** | 充足 | 几乎没有 | 用仓位控制 |
| **可复制性** | 高 | 低 | 中 |
