# EP43 - 处理连续亏损

> 📅 第43期 | 第四季度：实战与进化

---

## 📌 本期核心问题

**最大回撤时如何不崩溃？如何心理准备和应对？**

---

## 💡 第一性原理拆解

### 连续亏损是必然的

> 即使胜率60%的策略，也有20%的概率连续亏损3次以上。

### 亏损时的心理阶段

```
1. 否认 - "只是运气不好，会反弹"
2. 愤怒 - "市场不公平，有人操控"
3. 讨价还价 - "再给我一次机会"
4. 沮丧 - "我不适合投资"
5. 接受 - "这是正常的，继续执行系统"
```

### 应对策略

| 策略 | 内容 |
|------|------|
| 预设底线 | 回撤到X%强制暂停交易 |
| 减少仓位 | 亏损时降低仓位 |
| 冷静期 | 连亏3次暂停3天 |
| 求助外部 | 找人讨论，打破情绪循环 |
