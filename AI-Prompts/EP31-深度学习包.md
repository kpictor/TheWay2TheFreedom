## 📚 模块一：概念深化

### Prompt 1.1 - 三类出场规则

```markdown
帮我理解三类出场规则的设计：

**止损规则（防守）**
- 目的：限制单笔亏损，保护本金
- 必须在买入前就设定
- 执行是铁律

**止盈规则（进攻）**
- 目的：锁定收益，避免坐电梯
- 可以分批止盈
- 需要平衡"不贪"和"不早"

**时间止损（效率）**
- 目的：释放被套资金
- 持有X时间没变化，考虑退出
- 机会成本的考量

为什么三者都需要？给出具体示例。
```

### Prompt 1.2 - 止损方法详解

```markdown
介绍几种止损方法：

**固定比例止损**
- 定义：亏损X%止损
- 优点：简单明确
- 缺点：不考虑波动性
- 示例：亏损8%止损

**波动率止损（ATR止损）**
- 定义：根据股票波动性设置
- 优点：适应不同股票
- 缺点：计算复杂
- 示例：止损设在买入价下方2倍ATR

**技术止损**
- 定义：跌破支撑位止损
- 优点：有技术依据
- 缺点：支撑位主观
- 示例：跌破前低止损

**心理障碍**
> "止损不是认输，是保护继续战斗的能力"

为什么很多人做不到止损？如何克服？
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 止盈的艺术

```markdown
介绍止盈的不同策略：

**目标价止盈**
- 设定目标价，到达即卖
- 优点：明确
- 缺点：可能错过更大涨幅

**移动止盈（Trailing Stop）**
- 随着价格上涨，上移止损位
- 优点：让利润奔跑
- 缺点：可能在回调时被止出

**分批止盈**
- 分几次卖出锁定收益
- 优点：平衡风险和收益
- 缺点：需要多次决策

**估值止盈**
- 估值过高时卖出
- 优点：有基本面依据
- 缺点：高估值可能更高

每种策略适合什么情况？
```

### Prompt 2.2 - 专业交易员的出场纪律

```markdown
介绍专业交易员的出场纪律：

**规则优先**
- 设定好就执行，不重新评估
- 避免"再等等"的陷阱

**预设不可变**
- 买入前设定的出场点
- 持有期间不随意修改

**自动化执行**
- 使用止损单
- 减少人为干预

**事后复盘**
- 记录每次出场的原因和结果
- 评估规则有效性

这些纪律如何应用到我的交易中？
```
