# EP06 - 基金经理的激励结构 | AI深度学习包

> 🎯 目标：理解委托代理问题，认识为什么基金经理可能和你利益冲突

---

## 📚 模块一：概念深化

### Prompt 1.1 - 委托代理问题

```markdown
帮我深入理解"委托代理问题"（Principal-Agent Problem）：

1. 什么是委托代理问题？用生活例子说明
2. 在投资管理中如何体现？
3. 有哪些解决方案？（激励设计、监督、声誉等）
4. 这些方案的局限是什么？
5. 作为投资者，我应该如何保护自己？
```

### Prompt 1.2 - 基金业激励分析

```markdown
请详细分析基金行业的激励结构：

**公募基金**：
- 收入来源是什么？
- 规模增长和收益的关系？
- 排名压力如何影响行为？

**私募基金**：
- 业绩提成如何计算？
- 高水位法则是什么？
- 和公募的区别？

**对冲基金**：
- 2/20模式是什么？
- 激励是否与投资者利益一致？

每种模式的问题和相对优势是什么？
```

## 📖 模块二：专业背景补充

### Prompt 2.1 - 基金经理行为研究

```markdown
介绍学术研究中关于基金经理行为的发现：

1. **规模诅咒**：为什么规模扩张会损害收益？
2. **末日效应**：为什么年底表现往往失常？
3. **跟风行为**：机构投资者为什么扎堆？
4. **门面装饰**：为什么季末会调仓？
5. **主动管理悖论**：为什么主动管理难以跑赢指数？

这些发现对我选择基金有什么启示？
```

## 🧠 模块三：逻辑思维训练

### Prompt 3.1 - 设计理想基金

```markdown
和我一起设计一个"利益完全一致"的理想基金结构：

约束条件：
- 基金经理要有足够动力工作
- 投资者的钱要尽量安全
- 要能吸引到好的基金经理
- 监管和运营要可行

讨论以下问题：
1. 管理费应该怎么收？
2. 业绩提成应该怎么设计？
3. 是否需要跟投要求？
4. 赎回限制如何设置？
5. 透明度要求如何？

帮我思考这个问题的各个维度。
```

## ✅ 模块五：掌握度验证

### Prompt 5.1 - 案例分析

```markdown
分析以下场景是否存在利益冲突：

1. 某基金经理持有某股票，推荐客户买入
2. 某基金公司代销某新基金，获取销售佣金
3. 某分析师给买入评级，公司是投行客户
4. 某私募规模100亿，收1.5%管理费

对每个场景：
- 利益冲突在哪里？
- 会如何影响投资者？
- 如何识别和防范？
```
