## 💡 第一性原理拆解

### 主动vs被动的数据

> 长期来看，约80%的主动基金跑不赢指数。

### ETF的优势

| 优势 | 说明 |
|------|------|
| 低费用 | 管理费约0.1-0.5%，远低于主动基金 |
| 分散风险 | 一篮子股票，不依赖个股 |
| 透明 | 知道买了什么 |
| 流动性好 | 可随时买卖 |

### 常见的ETF类型

| 类型 | 示例 |
|------|------|
| 宽基指数 | 沪深300、标普500 |
| 行业指数 | 科技、医药、消费 |
| 跨境 | 纳斯达克、恒生 |
| 商品 | 黄金、原油 |
| 债券 | 国债、企业债 |
